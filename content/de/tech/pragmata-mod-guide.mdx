### Optimierung der VR-Performance
Beim Ausführen der VR-Mod berichten viele Benutzer von einem "Flimmereffekt" oder Doppelbildern im HUD. Um diese Probleme zu beheben, folgen Sie diesen Optimierungsschritten:

1.  **Raytracing deaktivieren:** Stellen Sie im Optionsmenü Raytracing auf "Aus". Dies reduziert das Flimmern im VR-Headset erheblich.
2.  **Subsurface Scattering ausschalten:** Diese Einstellung kann Ghosting-Effekte bei Charaktermodellen im 3D-Raum verursachen. Das Deaktivieren führt zu einem wesentlich saubereren Bild.
3.  **HUD verwalten:** Da die aktuelle VR-Mod ein Doppelbild für das Heads-up-Display (HUD) erzeugt, bevorzugen es viele Spieler, das HUD in den Spieleinstellungen komplett auszuschalten oder eine spezielle "Toggle HUD"-Mod zu verwenden.

## Schritt-für-Schritt-Installationsanleitung

Die meisten Modifikationen für diesen Titel folgen einem Standard-Installationspfad. Egal, ob Sie einen neuen Anzug oder ein Gameplay-Skript hinzufügen, der Prozess beinhaltet im Allgemeinen das Platzieren von Dateien im Stammverzeichnis des Spiels.

### Installation von Mods auf Engine-Ebene (VR, Trainer)
1.  Laden Sie die Mod-Datei von einer vertrauenswürdigen Quelle wie Nexus Mods herunter.
2.  Suchen Sie Ihren Installationsordner des Spiels (normalerweise unter `SteamLibrary\steamapps\common\PRAGMATA` zu finden).
3.  Entpacken Sie die Mod-Dateien direkt in den Ordner, in dem sich die ausführbare Datei (.exe) des Spiels befindet.
4.  Starten Sie das Spiel. Wenn Sie eine VR-Mod verwenden, stellen Sie sicher, dass Ihr Headset über Virtual Desktop oder Link verbunden ist, bevor Sie die Anwendung starten.

### Installation von Reshade-Presets
1.  Laden Sie die neueste Version des ReShade-Frameworks herunter und installieren Sie diese.
2.  Wählen Sie während des ReShade-Setups die ausführbare Datei von *Pragmata* aus.
3.  Platzieren Sie die spezifische heruntergeladene `.ini`-Preset-Datei (z. B. "Giorma's Realistic Reshade") in den Spielordner.
4.  Drücken Sie im Spiel die `Pos1`-Taste (Home), um das Preset auszuwählen und zu aktivieren.

> **💡 Tipp:** Wenn Sie beim Verwenden von visuellen Mods einen "doppelten Cursor" oder UI-Fehler bemerken, prüfen Sie, ob die Mod eine bestimmte Version des Spiels erfordert. Die meisten Mods von 2026 sind für den neuesten "Sketchbook"-Build optimiert.

## Fehlerbehebung bei häufigen Problemen

Modding ist selten eine perfekte Wissenschaft. Aufgrund der High-Fidelity-Natur von Capcoms RE Engine können selbst kleine Konflikte zu Abstürzen oder visuellen Artefakten führen.

### Fehlerbehebungstabelle
| Problem | Wahrscheinliche Ursache | Lösung |
| :--- | :--- | :--- |
| **Spiel stürzt beim Start ab** | Inkompatible Skriptversion. | Kürzlich hinzugefügte `.dll`-Dateien entfernen und Spieldateien auf Steam verifizieren. |
| **HUD-Doppelbild** | VR-Mod-Inkompatibilität. | Deaktivieren Sie das HUD im Hauptmenü des Spiels. |
| **Niedrige Bildrate** | Reshade/Textur-Overhead. | Reduzieren Sie "Schattenqualität" und "Volumetrische Beleuchtung" in den Einstellungen. |
| **T-Pose bei Charakteren** | Skriptfehler in VR. | Starten Sie den Checkpoint neu; dies ist eine bekannte Eigenheit früher VR-Injektionen. |

## FAQ

**F: Funktioniert dieser Pragmata Mod-Guide für die Konsolenversionen?**

A: Nein. Derzeit ist Modding nur auf der PC-Version des Spiels möglich. Konsolen wie die PS5 und Xbox Series X/S unterstützen den für diese Modifikationen erforderlichen Zugriff auf Dateiebene nicht.

**F: Ist es sicher, den Ultimate Trainer in Online-Modi zu verwenden?**

A: *Pragmata* ist primär eine Einzelspieler-Erfahrung. Sollten jedoch zukünftig Multiplayer-Komponenten hinzugefügt werden, könnte die Verwendung von Trainern oder Cheats zu einer Kontosperrung führen. Verwenden Sie Gameplay-verändernde Mods immer im Offline-Modus.

**F: Wie entferne ich eine Mod, wenn sie mein Spiel beschädigt?**

A: Der einfachste Weg ist das Löschen der Dateien, die Sie dem Spielverzeichnis hinzugefügt haben. Wenn Sie unsicher sind, welche Dateien hinzugefügt wurden, nutzen Sie die Funktion "Dateien auf Fehler überprüfen" in Steam, um den Originalzustand des Spiels wiederherzustellen.

**F: Können Mods die Performance des Spiels auf älterer Hardware verbessern?**

A: Ja, bestimmte Mods wie "No-HUD-No-Effects" oder spezielle Performance-Konfigurationsdateien können anspruchsvolle Post-Processing-Effekte deaktivieren, was potenziell dazu beitragen kann, dass das Spiel auf Mittelklasse-PCs aus früheren Jahren flüssiger läuft.
