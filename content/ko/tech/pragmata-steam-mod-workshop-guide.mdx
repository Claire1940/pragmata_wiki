## 프라그마타 모딩을 위한 필수 도구

창작마당에 뛰어들기 전에 파일을 관리할 도구 모음이 필요합니다. 이 게임은 정교한 버전의 RE 엔진을 사용하기 때문에 단순히 파일을 드래그하는 방식은 거의 작동하지 않습니다. 게임의 .pak 아카이브를 "읽고" 원하는 변경 사항을 안전하게 주입할 수 있는 모드 매니저가 필요합니다.

### 1. Fluffy Mod Manager (2026 에디션)
캡콤 타이틀 모딩의 표준으로 남아 있는 도구입니다. 게임 디렉토리를 영구적으로 변경하지 않고도 모드를 켜고 끌 수 있습니다. 특정 모드로 인해 달 전이 과정에서 게임이 충돌할 경우 문제 해결을 위해 필수적입니다.

### 2. REFramework
고급 사용자를 위해 REFramework는 강력한 스크립팅 API를 제공합니다. 이는 무한 제트팩 연료나 커스텀 카메라 각도와 같이 게임 로직을 변경하는 모드에 필요합니다. 또한 캐릭터 모델 교체를 방해할 수 있는 엔진 수준의 제한을 우회하는 역할도 합니다.

### 3. Optiscaler 및 프레임 생성
최근의 성능 분석에서 다루었듯이, 많은 플레이어가 Optiscaler와 같은 도구를 사용하여 FSR4 또는 XSS2를 게임에 적용하고 있습니다. 이는 달 도시의 고밀도 환경을 탐험하면서 안정적인 60 FPS를 유지하려는 스팀 덱 사용자에게 특히 유용합니다.

> **💡 팁:** 수동 모드 주입을 시도하기 전에 항상 `re_chunk_000.pak` 파일을 백업하세요. 모드 매니저는 안전하지만, 아카이브가 손상되면 60GB를 다시 다운로드해야 할 수도 있습니다.

## 단계별 설치 가이드

게임의 안정성을 유지하려면 2026년 현재 사용 가능한 두 가지 주요 방법을 통해 모드를 설치하는 다음 단계를 따르세요.

### 방법 A: 스팀 창작마당 (공식)
1. 스팀 라이브러리에서 **Pragmata** 페이지로 이동합니다.
2. **창작마당** 탭을 클릭합니다.
3. 카테고리(도전 과제, 캐릭터, 맵 등)를 탐색합니다.
4. 원하는 모드에서 **구독**을 클릭합니다.
5. 게임을 실행하면 스팀 클라이언트가 자동으로 파일 배치를 처리합니다.

### 방법 B: 수동 설치 (고급)
수동 설치는 "백팩 버디"를 다양한 대중문화 아이콘으로 바꾸거나 커뮤니티에서 언급되는 "디아나" 코스메틱 교체와 같은 인기 모드에 주로 필요합니다.

| 단계 | 작업 | 설명 |
| :--- | :--- | :--- |
| **1** | **모드 다운로드** | 신뢰할 수 있는 저장소에서 .rar 또는 .7z 파일을 받습니다. |
| **2** | **모드 매니저 실행** | Fluffy Mod Manager를 실행하고 게임 목록에서 Pragmata를 선택합니다. |
| **3** | **파일 드롭** | 다운로드한 압축 파일을 모드 매니저 창으로 드래그합니다. |
| **4** | **목록 새로고침** | 처음이라면 "Define Game Path"를 클릭한 후 "Refresh Mod List"를 누릅니다. |
| **5** | **활성화** | 모드를 "On"으로 토글하고 매니저를 통해 게임을 실행합니다. |

## 최고의 코스메틱 모드 찾기

커뮤니티 내에서 자주 묻는 질문 중 하나는 디아나 전용 캐릭터 모드의 위치입니다. 스팀 창작마당은 환경 수정에 뛰어나지만, 캐릭터 중심의 "코스메틱" 모드는 외부 사이트에서 더 자주 발견됩니다.

커뮤니티 멤버들은 유명한 바이오하자드 2 모드와 유사한 고품질 모델 교체 모드들이 넥서스모드나 비공개 커뮤니티 포럼에 게시되어 있다고 언급합니다. 이러한 모드들은 단순한 의상 색상 변경부터 최근 소셜 미디어에서 화제가 된 "대니 드비토 백팩 버디"나 "호머 심슨" 교체와 같은 완전한 모델 변경까지 다양합니다.

> **⚠️ 경고:** 비공식 사이트에서 모드를 다운로드할 때는 주의하세요. 항상 사용자 댓글과 "추천(endorsement)" 수를 확인하여 해당 파일이 안전하고 현재 2026년 게임 버전과 호환되는지 확인해야 합니다.

**Q: 포럼에서 언급된 디아나 코스메틱 모드는 어디서 찾을 수 있나요?**

**Q: 스팀 덱에서 플레이 중인데 모드를 사용할 수 있나요?**

**Q: 프라그마타 모딩은 합법인가요?**

A: 모딩은 일반적으로 팬 아트와 유사한 "회색 영역"으로 간주됩니다. 저작권이 있는 게임 파일을 배포하거나 경쟁적인 멀티플레이어 환경에서 부정행위를 위해 모드를 사용하지 않는 한, 캡콤은 일반적으로 커뮤니티의 활동을 허용하는 편입니다.
