### VR 성능 최적화
VR 모드를 실행할 때 많은 사용자가 HUD에서 "반짝임(shimmering)" 현상이나 이중 이미지를 보고합니다. 이러한 문제를 해결하려면 다음 최적화 단계를 따르세요.

1.  **레이 트레이싱 비활성화:** 옵션 메뉴에서 레이 트레이싱을 "끄기(Off)"로 설정합니다. 이는 VR 헤드셋의 반짝임 현상을 크게 줄여줍니다.
2.  **하표면 산란(Subsurface Scattering) 끄기:** 이 설정은 3D 공간에서 캐릭터 모델에 잔상을 일으킬 수 있습니다. 이를 비활성화하면 훨씬 깨끗한 이미지를 얻을 수 있습니다.
3.  **HUD 관리:** 현재 VR 모드는 헤드업 디스플레이(HUD)에 이중 이미지를 생성하므로, 많은 플레이어는 게임 설정에서 HUD를 완전히 끄거나 전용 HUD 토글 모드를 사용하는 것을 선호합니다.

## 단계별 설치 가이드

이 타이틀의 대부분의 수정 사항은 표준 설치 경로를 따릅니다. 새로운 슈트를 추가하든 게임플레이 스크립트를 추가하든, 프로세스는 일반적으로 파일을 게임의 루트 디렉토리에 배치하는 것과 관련이 있습니다.

### 엔진 레벨 모드 설치 (VR, 트레이너)
1.  Nexus Mods와 같은 신뢰할 수 있는 소스에서 모드 파일을 다운로드합니다.
2.  게임 설치 폴더를 찾습니다 (일반적으로 `SteamLibrary\steamapps\common\PRAGMATA`에 위치).
3.  게임의 실행 파일(.exe)이 있는 폴더에 모드 파일의 압축을 직접 풉니다.
4.  게임을 실행합니다. VR 모드를 사용하는 경우 응용 프로그램을 시작하기 전에 버추얼 데스크탑(Virtual Desktop) 또는 링크(Link)를 통해 헤드셋이 연결되어 있는지 확인하십시오.

### 리셰이드(Reshade) 프리셋 설치
1.  최신 버전의 ReShade 프레임워크를 다운로드하여 설치합니다.
2.  ReShade 설정 중에 *Pragmata* 실행 파일을 선택합니다.
3.  다운로드한 특정 `.ini` 프리셋 파일(예: "Giorma's Realistic Reshade")을 게임 폴더에 넣습니다.
4.  게임 내에서 `Home` 키를 눌러 프리셋을 선택하고 활성화합니다.

> **💡 팁:** 비주얼 모드를 사용할 때 "이중 커서"나 UI 글리치가 발생하면 해당 모드가 특정 버전의 게임을 요구하는지 확인하세요. 대부분의 2026년 모드는 최신 "스케치북" 빌드를 위해 제작되었습니다.

## 일반적인 문제 해결

모딩이 항상 완벽한 것은 아닙니다. 캡콤 RE 엔진의 고정밀 특성상, 작은 충돌도 충돌(Crash)이나 시각적 아티팩트로 이어질 수 있습니다.

### 문제 해결 표
| 문제 | 가능성 높은 원인 | 해결 방법 |
| :--- | :--- | :--- |
| **실행 시 게임 충돌** | 호환되지 않는 스크립트 버전. | 최근에 추가한 `.dll` 파일을 제거하고 Steam에서 게임 파일 무결성을 검사합니다. |
| **HUD 이중 이미지** | VR 모드 호환성 문제. | 게임 메인 옵션 메뉴에서 HUD를 비활성화합니다. |
| **낮은 프레임 속도** | 리셰이드/텍스처 과부하. | 설정에서 "그림자 품질"과 "볼류메트릭 라이팅"을 낮춥니다. |
| **T-포즈 캐릭터** | VR 스크립팅 오류. | 체크포인트를 재시작하세요. 이는 초기 VR 인젝션의 알려진 특이 사항입니다. |

## FAQ

**Q: 이 프라그마타 모드 가이드가 콘솔 버전에서도 작동하나요?**

A: 아니요. 현재 모딩은 PC 버전의 게임에서만 가능합니다. PS5 및 Xbox Series X/S와 같은 콘솔은 이러한 수정에 필요한 파일 레벨 접근을 지원하지 않습니다.

**Q: 온라인 모드에서 얼티밋 트레이너를 사용해도 안전한가요?**

A: *프라그마타*는 기본적으로 싱글 플레이어 경험 중심입니다. 하지만 향후 멀티플레이어 구성 요소가 추가될 경우, 트레이너나 치트를 사용하면 계정 정지 처분을 받을 수 있습니다. 게임플레이를 변경하는 모드는 항상 오프라인 모드에서 사용하세요.

**Q: 모드 때문에 게임이 망가졌을 때 어떻게 제거하나요?**

A: 가장 쉬운 방법은 게임 디렉토리에 추가한 파일을 삭제하는 것입니다. 어떤 파일이 추가되었는지 확실하지 않은 경우, Steam의 "게임 파일 무결성 검사" 기능을 사용하여 원래 게임 상태로 복구하세요.

**Q: 모드가 저사양 하드웨어에서 게임 성능을 향상시킬 수 있나요?**

A: 네, "No-HUD-No-Effects"나 특수한 성능 구성 파일과 같은 특정 모드는 사양을 많이 타는 포스트 프로세싱 효과를 비활성화하여 구형 PC에서도 게임이 더 원활하게 실행되도록 도울 수 있습니다.
