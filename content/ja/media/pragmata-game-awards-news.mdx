## The Game Awards 2025：トレーラーの分析と物語の手がかり

## PRAGMATA Game Awards ニュース：発売時期とレーティング機関のリーク情報

カプコンは正確な発売日の特定には慎重ですが、業界関係者の間で流れている**PRAGMATA Game Awards ニュース**は、2026年4月または5月のリリースを指し示しています。このタイムラインは、韓国のゲーム物管理委員会（GRAC）による最近のレーティング審査によっても裏付けられています。歴史的に、GRACのレーティングは製品版の発売の約4〜6ヶ月前に公開されることが多く、2026年第2四半期という窓口に完璧に合致しています。

カプコンの2026年の戦略は、同社の歴史の中でも最も野心的なものになりそうです。この新規IPを他の主要タイトルの間に配置することで、会計年度を通じて「トリプルA」級のコンテンツを継続的に提供する狙いがあります。

## ゲームプレイメカニクス：戦闘、トラバーサル、そして「クレイドル」

2025年末に公開されたゲームプレイ映像では、三人称視点のアクションとタクティカルな管理要素のユニークな融合が披露されました。主人公のスーツは単なる防具ではなく、ドローンの展開、一時的な重力圏の生成、月面環境とのインターフェースが可能なモジュール式プラットフォームです。ダイアナもまた、単なる護衛対象以上の重要な役割を担います。彼女は電子システムを操作したり、敵の弱点を強調する「ビジョン」を提供したりすることができます。

### 主なゲームプレイの特徴：
1. **モジュール式スーツのアップグレード**：月面コロニー内でリソースを収集し、アーマーのスラスターや防御シールドを強化。
2. **環境パズル**：重力を無視するツールを使用して、崩壊した「クレイドル」の構造内をナビゲート。
3. **ダイナミックな戦闘**：遠距離エネルギー弾と、接近する「未登録者」に対する強力な近接攻撃を切り替えて戦う。
4. **ダイアナの支援**：ダイアナに指示を出してターミナルをハックさせたり、プレイヤーが位置を変える間に敵の注意を引かせたりする。

> **警告：** スーツの酸素とエネルギーレベルの管理は極めて重要です。真空環境でパワーが尽きると、急速に体力が減少します。

## なぜ延期が必要だったのか

2020年の初公開以来、このプロジェクトは数回の公式な延期に直面してきました。最新の**PRAGMATA Game Awards ニュース**によると、これらは「開発の難航」によるものではなく、むしろ「規模の拡大」によるものだったようです。カプコンは当初の線形なレベル構成から、「クレイドル」の環境を完全にシームレスなオープンワールド体験へと拡張したと報じられています。この拡張により、特に低重力下での戦闘メカニクスに関するゲーム物理エンジンの大幅な再構築が必要となりました。

開発チームは、新しいフランチャイズを確立するために、磨き上げられた体験を提供することにコミットしていると述べています。追加の時間をかけることで、カプコンは初期の内部プレイテストからのフィードバックを統合し、特に主人公とダイアナの相互作用を改善して、物語的な重みがより感じられるように調整することができました。

## よくある質問（FAQ）

**Q: PRAGMATAは既存のカプコンゲームの続編ですか？**

A: いいえ、完全に新しい知的財産（新規IP）です。『デビル メイ クライ』や『バイオハザード』のような高品質なアクションのDNAを引き継いでいますが、設定やキャラクターはすべてオリジナルです。

**Q: Game Awardsのニュースで言及された「クレイドル」とは何ですか？**

A: 「クレイドル」はゲームの主な舞台となる、巨大なハイテク月面施設です。2026年のトレーラーに基づくと、地球を逃れた人類の聖域のようですが、現在は朽ち果て、厳重な自動防衛システムが稼働している状態にあります。

**Q: マルチプレイヤーモードはありますか？**

A: 現在のところ、すべての公式情報はシングルプレイヤーの物語体験に焦点を当てています。2026年初頭の時点では、協力プレイや対戦マルチプレイヤーモードについての言及はありません。

**Q: PS4やXbox Oneなどの旧世代機でプレイできますか？**

A: いいえ。カプコンは、SSDの速度や高度なレンダリング技術を最大限に活用するため、本作をPlayStation 5、Xbox Series X|S、およびハイエンドPC専用に構築していることを認めています。
