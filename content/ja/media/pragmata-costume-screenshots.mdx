## キャビン・スタンプ・システムの解説

プラグマタで新しい装備を入手する主な方法は、キャビン・スタンプ・システムです。拠点となる「シェルター」内にあるこのシステムでは、キャビン・コインを消費して様々な報酬を獲得できます。コレクションを最大化し、自分だけのギャラリーで全ての**プラグマタ コスチューム スクリーンショット**を確認するには、トレーニング・シミュレーションのミッションを通じてコインを集めることを優先する必要があります。

ゲームの5つのセクターを進めるにつれて、シェルターのレベルが上がります。レベルごとに新しいスタンプカードが解放され、合計で5枚のカードが登場します。一部の衣装はカード内の個別のスロットに配置されていますが、特定のカードの全スロットを埋めた時のみに与えられる限定アイテム「ブラックアウト」報酬となっているものもあります。

| シェルターレベル | 解放条件 | スタンプカード報酬タイプ |
| :--- | :--- | :--- |
| **レベル 1** | セクター1クリア | 基本のエクゾレイヤー装備 |
| **レベル 2** | セクター2クリア | Type B バリアント |
| **レベル 3** | セクター3クリア | レトロ＆ストライキングブラック・セット |

> **💡 ヒント:** 各セクターの早い段階で、常にトレーニング・シミュレーションのミッションを優先してください。これらのミッションではキャビン・コインが大量に手に入るため、カードが解放された直後に「ブラックアウト」させることが可能になります。

## ヒューのアンロック衣装

主人公であり保護者であるヒューは、多様な重装備のエクゾレイヤーや大気圏用スーツを着用できます。彼のワードローブは、標準的な軍用ギアから、月面基地の最深部でのみ見つかる実験的なシャドウ・テックまで多岐にわたります。

### ヒューのスタンプカード報酬

ヒューの進行は、最初の3枚のスタンプカードに大きく依存しています。初期装備の**デリスーツ**も機能的ですが、多くのプレイヤーはすぐに、より専門的なギアを求め始めます。

1.  **エクゾレイヤー・アウトフィット:** 1枚目のスタンプカードにあります。初期スーツを直接アップグレードしたもので、より装甲化された外観になります。
2.  **エクゾレイヤー Type B:** 2枚目のスタンプカードにあります。異なるアクセントカラーと細かな構造変更が特徴です。
3.  **レギオン・ゼロ:** 2枚目のスタンプカードのブラックアウト報酬です。より攻撃的でタクティカルな外観を求めるファンに人気です。
4.  **アストロクラシック:** 3枚目のスタンプカードにあります。20世紀半ばのSFコンセプトを彷彿とさせる「レトロフューチャー」なデザインで、非常に人気が高いスーツです。
5.  **レギオン・ゼロ Type B:** 3枚目のスタンプカードのブラックアウト報酬で、より暗く、隠密性を重視したパレットになっています。

### ヒューのスペシャルミッション・スーツ

スタンプ以外にも、ストーリーの主要な節目や、難易度の高いクリア後のチャレンジを完了することで、特別なスーツを解放できます。

| 衣装名 | 解放方法 | ビジュアルスタイル |
| :--- | :--- | :--- |
| **シャドウ・ウォーカー** | カード5（シェルターレベル5） | 不気味でダーク、有機的なデザイン |
| **アストロクラシック Type B** | ラスボスを撃破 | 派手な赤と金のアクセント |
| **オブシディア** | 「アンノウン・シグナル」完了 | 洗練された黒曜石のような仕上げ |

## ダイアナのアンロック衣装

ダイアナの衣装は、ヒューの重厚なアーマーとは対照的に、より俊敏でスタイリッシュなデザインが多くなっています。彼女のギアはストーリーにおけるユニークな役割を反映しており、月面サバイバル技術とハイファッションの要素が融合しています。

### ダイアナのスタンプカード報酬

ダイアナの進行には少し努力が必要です。彼女の最高のルックスの多くは、単純なスロット購入ではなく、ブラックアウト報酬としてロックされているからです。

*   **ブルー・クレイドル・ジャケット:** ダイアナのデフォルトの初期衣装です。
*   **ルナ・アクア:** 1枚目のスタンプカードのブラックアウト報酬です。コミュニティの間で評価が分かれる特徴的なヘッドギアが付属しています。
*   **ルナ・ブラック:** 3枚目のスタンプカードにあります。月面の塵の中で際立つ、印象的なハイコントラストな外観です。
*   **ルナ・ホワイト:** 4枚目のスタンプカードのブラックアウト報酬です。「スパイ・ムーン」のような雰囲気と評され、彼女のベストデザインの一つとして広く認識されています。
*   **ハートビート:** 5枚目のスタンプカードで最初の「ビンゴ」（1列または1行を揃える）を達成した際の報酬です。
*   **スター・リリー:** 5枚目のスタンプカードのブラックアウト報酬です。この衣装はダイアナの髪型をショートカットに変更することに注意してください。

### ダイアナのスペシャルミッション・スーツ

## ビジュアルギャラリー：プラグマタ コスチューム スクリーンショットとプレビュー

**プラグマタ コスチューム スクリーンショット**を探す際、プレイヤーは標準のレビュー版には含まれていなかった予約特典やDLCバンドルに注目することがよくあります。これらには、ヒュー用の侍風衣装やダイアナ用の忍者風衣装が含まれます。これらのセットはあらゆる意味で「特別」であり、本編のSFリアリズムとは一線を画すデザインを提供します。

「アンノウン・シグナル」ミッションは、2026年におけるプラグマタの真のエンドゲームを象徴しています。これらのミッションをクリアすると、「オブシディア」や「ブラック・ネオン」スーツが手に入るだけでなく、シェルターの最終ティアが解放され、コレクションを完遂させることができます。

> **⚠️ 警告:** 「シャドウ・ウォーカー」と「シャドウ・エクセキューター」の衣装は、その「不気味な」外観で知られています。一部のプレイヤーは、これらのスーツの有機的でピクピクと動くテクスチャがカットシーン中に気になると感じています。

## キャビン・コインを効率よく稼ぐ方法

最新の**プラグマタ コスチューム スクリーンショット**に写っているものをすべてアンロックするには、数百枚のキャビン・コインが必要です。自然な進行でも入手できますが、稼ぎ作業を行うほうがはるかに効率的です。

1.  **トレーニング・シミュレーションの反復:** ヒューのメイン武器をアップグレードしていれば、特定の戦闘シミュレーションは2分以内でクリア可能です。
2.  **セクターの隅々を探索:** 各セクターには、近づくまでマップに表示されない隠されたコインのキャッシュが存在します。
3.  **デイリーログイン:** オンライン統合版をプレイしている場合、毎日のシェルターチェックインで少量ながら着実な通貨収入が得られます。

## FAQ

**Q: 難易度「Lunatic（ルナティック）」限定の衣装はありますか？**

A: 現在、ほとんどの衣装はどの難易度でもスタンプカードやクリア後ミッションを通じて入手可能です。ただし、特定のミッションをLunaticでクリアした後にのみ表示されるユニークなエンブレムや、細かなアーマーデカールについての報告が一部のプレイヤーから寄せられています。

**Q: 衣装とは別にダイアナの髪型を変更できますか？**

A: いいえ、現在ヘアスタイルは特定の衣装に紐付けられています。例えば、「スター・リリー」衣装は強制的にショートヘアになりますが、「ルナ・ホワイト」は標準の長さを維持します。この違いは、コミュニティで共有されている様々な**プラグマタ コスチューム スクリーンショット**で確認できます。

**Q: 「ブラックアウト」報酬とは何ですか？**

A: ブラックアウト報酬は、キャビン・スタンプカードのすべてのマスを埋めたときに与えられる特別なボーナスです。通常、1枚のカードにつき25個のスタンプが必要で、十分なコインを稼ぐためにいくつかのサイドクティビティに参加する必要があります。

**Q: 「アンノウン・シグナル」ミッションにはどうやってアクセスしますか？**

A: これらのミッションは、ラスボスを倒してクレジットを見た後にのみ、シェルターのミッション端末に表示されます。メインストーリーのミッションよりも大幅に難易度が高く、装備をフルアップグレードしたプレイヤー向けに設計されています。
