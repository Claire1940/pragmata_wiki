## PRAGMATAにおけるセクターの構造

PRAGMATAは、レベルデザインに「メトロイドヴァニア風（ライト）」のアプローチを採用しています。ストーリーはナラティブに沿って進行しますが、物理的な世界は広大で半開放的な「セクター」に分かれています。これらは単なるレベルではなく、垂直性と隠されたショートカットが重要な役割を果たす多層的な環境です。

プレイヤーが最初に遭遇する主要なゾーンは、**太陽光発電所（セクター1）**です。このエリアは、ハッキング、ステイシスネットを用いた戦闘、基本的なプラットフォームアクションなど、ゲームのコアメカニクスへの導入として機能します。しかし、このゲームのデザインの特徴として、セクター1は最初の訪問では完全には攻略できません。「フィラメント」で作られた障壁や、**セクター2**以降で付与される能力を必要とする高度なセキュリティロックに頻繁に遭遇することになります。

### 進行とバックトラッキング

自分のプレイ時間の多くをどのセクターで過ごすことになるのかを真に理解するには、アップグレードのために必要なバックトラッキング（戻り探索）を考慮する必要があります。各セクターは、マップスコアに寄与する「追跡対象の収集品」で満たされています。

| セクター名 | 主要な環境 | 解放される主要アビリティ | 主要ボス |
| :--- | :--- | :--- | :--- |
| **セクター1** | 太陽光発電所 | ステイシスネット / ハッキング | センチネル・ドローン |
| **セクター2** | リサーチハブ | **リム・イレイザー** | アーカイブ・コア |
| **セクター3** | 月面 | 酸素スタビライザー | ザ・ワンダラー |
| **セクター4** | シェルター | 高度合成 | ファイナル・プロトコル |

> **💡 ヒント:** セクター1の最初の探索で100%のスコアに執着しないでください。次のセクターで**リム・イレイザー**を入手するまで、技術的に到達できないエリアが存在します。

## セクター1：太陽光発電所の深掘り

太陽光発電所は、月面コロニーに電力を供給する巨大な産業施設です。配送センター、コンコース、発電機入口など、いくつかのサブゾーンに分かれています。

### セクター1の重要ロケーション
太陽光発電所を探索する際は、最も価値のある戦利品が隠されている以下の特定の場所に目を光らせてください。

1.  **配送センター:** ここは垂直方向の遊び場であり、最初のセーフボックスが見つかる場所です。プラットフォームを動かすために、リフトやハッキング端末を駆使する必要があります。
2.  **コンコース:** シェルターへのショートカットがあるハブエリアです。ここでは、重要なスーツMODやヘルスアップグレードが見つかります。
3.  **発電機入口:** 次のゾーンへの移行につながるセクターの最終区間です。このエリアはウォーカーや飛行ドローンによって厳重に警備されています。

### 追跡対象の収集品リスト
マップで100%のスコアを達成するには、特定のアイテムを見つける必要があります。ファイルや特定のホログラムは達成率には含まれませんが、以下のアイテムは必須です。

| 収集品タイプ | 目的 | 発見方法 |
| :--- | :--- | :--- |
| **セーフボックス** | **リム**（通貨）が入っている | 隠れた角やエリート敵が守っている場所にある |
| **RAMユニット** | ロアと世界観の構築 | プラットフォームパズルの背後に隠されていることが多い |
| **MOD** | スーツのアップグレード（例：硬化スーツ） | 青い宝箱に入っている |
| **ミニキャビン** | シェルター用の収集品 | 近くにあると独特の音が鳴る。撃つ必要がある |
| **カートリッジホルダー** | 回復容量の増加 | メドベイ（医療室）やオレンジ色の装置にある |

## 戦闘とハッキングのマスター

セクターを進むにつれて、戦闘の複雑さは大幅に増していきます。セクター1では、**ステイシスネット**が導入されます。この武器はクラウドコントロールに不可欠で、敵をその場に凍結させている間に、「クラッシャー」のような優先度の高いターゲットに集中することができます。

### ハッキングノードとダイアナ
パートナーであるダイアナは、さまざまなセクターをナビゲートするために不可欠です。ミニゲームや戦闘チャレンジをトリガーするハッキングノードに遭遇することがあります。これらを完了すると、コンコースのシェルター用ハシゴなどのショートカットが解放されることが多く、その後の探索で危険な通路を回避できるようになります。

> **警告:** 一部のハッキングノードはアリーナのロックダウンを引き起こします。広い部屋でノードを操作する前に、ステイシスネットがフル充電されていることを確認してください。

## リム・イレイザーの重要性

**PRAGMATA（プラグマタ）のセクター数**に関する最も頻繁な質問の一つは、「リム」の障壁に関わるものです。セクター1のいたるところで、光る有機的なフィラメントによってブロックされたドアやチェストを目にするでしょう。これらは「リム」の増殖物です。

ゲームの最初の1時間は、これらと対話することはできません。**リム・イレイザー**を入手するには、第2セクターまで進む必要があります。このツールを手に入れたら、太陽光発電所に戻って以下のものをアンロックできます。

*   **ストレージ・エキスパンダー:** ダイアナのハッキングゲージを増加させ、より強力なマシンを乗っ取ることができるようになります。
*   **ピュア・ルン:** 配送センターのアリーナで見つかる希少なクラフト素材です。
*   **高価値なセーフボックス:** 序盤の一部のボックスには最大800リムが入っていますが、それらはフィラメントの壁の奥に隠されています。

## 100%コンプリートのためのヒント

コンプリートを目指すプレイヤーにとって、PRAGMATAのセクターシステムは少し手ごわいかもしれません。見落としがないように、以下の手順に従ってください。

1.  **音のヒントを聞く:** ミニキャビンのフィギュアは高い歌声のような音を発します。音が聞こえたら立ち止まり、天井や二段ベッドの下を確認してください。
2.  **「隠れた」ルートをチェックする:** レベルデザインには、メインパスの上や下に通路があることがよくあります。崖からリム通貨が続いているのが見えたら、それを辿ってみてください。
3.  **早めにハッキングをアップグレードする:** ストレージ・エキスパンダーを見つけたらすぐに使用しましょう。ハッキングレベルが高ければ、敵のドローンを味方にすることができ、後半のより困難なセクターをはるかに管理しやすくなります。
4.  **ジップラインを活用する:** 「クレヨン」RAMのような多くのRAMユニットは、高いパイプの上に隠されており、ジップラインの上昇のピークでジャンプすることでのみ到達できます。

ゲームのロアやメカニクスの詳細については、[カプコンのPRAGMATA公式サイト](https://www.capcom.com/pragmata/)で最新のアップデートやパッチノートを確認してください。

## 探索報酬のまとめ

探索は単にトロフィーのためだけではなく、生存に直結します。各セクターには、プレイ体験を変える特定のアップグレードが配置されています。

| アイテム | 場所の例 | メリット |
| :--- | :--- | :--- |
| **硬化スーツMOD** | コンコースの青い宝箱 | ダメージ耐性の向上 |
| **カートリッジホルダー** | セクター1のメドベイ | 回復アイテムの所持数増加 |
| **ストレージ・エキスパンダー** | フィラメントの奥（セクター1） | ダイアナのハッキング容量の増加 |
| **セーフボックス (100-800 リム)** | 配送センター | 新しい装備を購入するための通貨 |

**PRAGMATA（プラグマタ）にあといくつのセクター**が残っているかを知るには、ワールドマップを確認するのが一番です。マップには、各ゾーンで見つかった収集品の割合が明確に表示されます。もし90%で止まっているなら、おそらく後のセクターのアビリティが必要な道の先にあるミニキャビンやMODを見逃している可能性が高いでしょう。

## FAQ

**Q: PRAGMATAのメインストーリーをクリアするには、いくつのセクターが必要ですか？**

A: メインストーリーは、太陽光発電所、リサーチハブ、月面、シェルターの4つの主要セクターにわたって展開されます。ただし、エンドゲームコンテンツや隠しボスが存在するいくつかの移行エリアや「隠し」サブセクターも存在します。

**Q: セクター間をいつでも移動できますか？**

A: はい、各セクター内のシェルターのショートカットとトラムシステムをアンロックすれば、以前のエリアにファストトラベルできます。第2セクターでリム・イレイザーを入手した後に戻ることを強くお勧めします。

**Q: マップスコア100%のために、すべてのホログラムを見つける必要がありますか？**

A: いいえ。ホログラムは月面コロニーの崩壊に関する深いロアや背景情報を提供しますが、「追跡対象の収集品」のスコアにはカウントされません。セーフボックス、RAM、MOD、および特定のアップグレードに集中するだけで十分です。

**Q: セクター1で見つけるべき最も重要なアイテムは何ですか？**

A: メドベイで見つかる「カートリッジホルダー」が間違いなく最も重要です。これは回復容量を増加させるもので、太陽光発電所の最後にあるボス戦や、リサーチハブでの難易度急上昇を乗り切るために不可欠です。
