カプコンが贈る月面SFアドベンチャー『PRAGMATA（プラグマタ）』の「Sketchbook」体験版は、製品版の発売を前に、その複雑なシステムをマスターしようとするプレイヤーに濃密なバーティカル・スライス（垂直切り出し）を提供しています。多くのプレイヤーは単にクレジットに到達することを目指しますが、この試練の真の深みは、ダイアナが描いた8つのユニークなエンディングイラストをアンロックするために必要な**プラグマタ 体験版 エンディング 条件**を解き明かすことにあります。これらの報酬は単なる飾りではなく、戦闘、探索、そして主人公ヒューと相棒ダイアナのシナジーの習熟度を証明するものです。

月面の荒廃した施設を進むには、単なる火力以上のものが必要です。体験版で100%の達成率を記録するには、平和主義的なアプローチから高速効率化まで、複数の周回にわたってプレイスタイルを調整する必要があります。このガイドでは、2026年版体験版で利用可能なすべての秘密の報酬を獲得できるよう、すべての**プラグマタ 体験版 エンディング 条件**を詳しく解説します。

## 8つの隠しエンディングイラストとその達成条件

8つすべてのイラストをアンロックすることは、コンプリートを目指すプレイヤーにとっての究極の目標です。「平和主義」ランと「クラウドコントロール」条件のように、一部の条件は互いに排他的であるため、すべてを見るには少なくとも3〜4回は体験版をプレイする必要があります。ゲームエンジンは、受けたダメージ、収集したリソース、さらにはとどめを刺す際に使用した特定の武器などの指標を記録しています。

以下は、各イラストの**プラグマタ 体験版 エンディング 条件**の包括的な内訳です。

| イラストNo. | 条件名 | 達成のための具体的な基準 |
| :--- | :--- | :--- |
| **No. 1** | **パシフィスト・サポート** | ダイアナのアビリティのみ使用。最初の強制射撃以外は銃を撃たない。 |
| **No. 2** | **エリート・エンカウンター** | エレベーター内で高レベルの**オレンジ/レッド・ロボット**を出現させ、撃破する。 |
| **No. 3** | **コンプリーショニスト** | すべてのイエローキューブを回収し、体験版内のすべての大きな宝箱を開ける。 |
| **No. 4** | **パーフェクトガード** | ダメージを一切受けずに体験版をクリアする。 |
| **No. 5** | **スピードの悪魔** | 合計プレイ時間**10分未満**で最終リザルト画面に到達する。 |
| **No. 6** | **クラウドコントロール** | グリーンガンを使用して、1発のショットで2体の敵を拘束・連結する。 |
| **No. 7** | **スカベンジャー** | マップ全体に散らばる一般リソースの80%以上を回収する。 |
| **No. 8** | **サバイバー** | 特殊な条件を満たさずに通常通り体験版をクリアする。 |

> **💡 ヒント:** イラストNo. 1では、最初のカットシーンを発生させるために、冒頭で1発だけ撃つ必要があります。その後すぐに武器を外し、ダイアナのハッキングとサポートスキルのみに頼って進行してください。

## 秘密のエリートとの遭遇（No. 2）を発生させる方法

イラストNo. 2は、最後のエレベーター上昇中に特定の「バラエティ・キル」シーケンスを必要とするため、最も入手困難な報酬とされています。通常、エレベーター内には標準的なドローンと白いロボットが出現しますが、正確な実行順序を守ることで、レアな高レベルのオレンジ・ロボットを強制的に出現させることができます。

この特定の**プラグマタ 体験版 エンディング 条件**を満たすには、エレベーター内で以下の手順に従ってください。

1.  **ドローンを排除:** 標準のピストルまたはライフルを使用して、飛行ユニットを即座に撃墜します。
2.  **ダイアナのターン:** ダイアナにコマンドを出し、サポートアビリティを使用して1体目の地上ロボットを無力化します。
3.  **グリーンガンでフィニッシュ:** グリーンガン（スタティック/コントロールガン）を使用して、2体目の標準ロボットにとどめを刺します。
4.  **出現:** 正しく実行されると、2体の高レベル・オレンジ・ロボットがワープしてきます。これらを撃破すればイラストNo. 2が確定します。

このシーケンスは、プレッシャーのかかる中で武器タイプを切り替えながら、ダイアナのクールダウンを管理する能力をテストします。この遭遇をマスターすることは、RE ENGINEによる深い戦闘メカニクスを理解するための前提条件と言えます。

## 「スピードの悪魔」の称号を目指すスピードラン（No. 5）

10分を切るクリアタイムを達成するには、月面施設のレイアウトを熟知している必要があります。不要な戦闘を回避し、「緊急ロックダウン」端末だけに集中しなければなりません。体験版には手動のパワーセル再起動が必要なゲートエリアがいくつかあり、それらの正確な場所を知ることで数分を短縮できます。

| 目的 | 必要なアクション | 時短戦略 |
| :--- | :--- | :--- |
| **最初の突破** | 侵入者の第一波を排除 | 即座に**オーバードライブ**を使用して数秒で部屋を殲滅する。 |
| **パワーセル 1** | 上部3つのプロトコルを迂回 | 小さな箱は無視し、端末へ直行する。 |
| **パワーセル 2** | 手動セル再起動 | ウサギ型ロボットは無視する（索敵から攻撃までが遅いため）。 |
| **エレベーター** | 侵入者を排除 | ロックダウンタイマーを止めるため、ドローンを最優先で倒す。 |

> **⚠️ 警告:** メンテナンスシャフトエリアでの落下や環境トラップへの接触は、ダメージを受ける（No. 4失敗）だけでなく、タイムロスにも繋がります。スピードよりも正確な動きが重要です。

## 探索とリソース収集（No. 3 & No. 7）

月面基地には、イエローキューブや隠しキャッシュを通じた環境ストーリーテリングが散りばめられています。イラストNo. 3を確実にするには、細心の注意が必要です。多くの宝箱は破壊可能な壁の裏や、ダイアナの「リフト」能力が必要な高い梁の上に隠されています。

イラストNo. 7の「スカベンジャー」エンディングは、希少性よりも量に焦点を当てています。HUDにはライブカウンターが表示されませんが、通路や脇道にある素材の少なくとも80%を拾う必要があります。下のメンテナンスシャフトに早く進みすぎると、前のエリアに戻れなくなる可能性があるため、進む前に上層階を完全に探索したか確認してください。

## 戦闘習熟：グリーンガンのメカニクス（No. 6）

グリーンガンは単なるスタン用ツールと誤解されがちですが、第6のイラストを解放する鍵となります。この武器は敵同士を繋ぐテザーを発射し、衝突させたりダメージを共有させたりします。この**プラグマタ 体験版 エンディング 条件**を満たすには、少なくとも2体の敵が近接するまで待つ必要があります。

1発のショットで2体の敵を同時に「コントロール（連結）」すると、アンロックがトリガーされます。これを達成するのに最適な場所は、最初の緊急ロックダウンを解除した直後の第2遭遇エリアで、狭い通路に複数のロボットが出現します。

## 体験版から製品版へ：真のエンディングの文脈

体験版はこれら8つのイラストに焦点を当てていますが、製品版の「シナジー」システムの入門編としての役割も果たしています。『PRAGMATA』の製品版では、隠された「シナジーフラグ」を通じて追跡されるダイアナとの関係によって、デフォルトの「見捨てられた（Abandonment）」エンディングになるか、「真エンディング（True Ending）」になるかが決まります。

ダイアナのハッキング中に彼女を守ったり、シナジーコンボを繰り出したりといった、**プラグマタ 体験版 エンディング 条件**を追い求める中で学ぶスキルは、まさに製品版で求められる要素です。2026年の製品版では、バッドエンディングを避けるために20個のメモリフラグメントを収集し、100%のシナジースコアを維持する必要があります。「Sketchbook」体験版でこれらの行動を練習しておくことが、月面コロニーでの挑戦に備える最善の方法です。

## FAQ

**Q: 1回の体験版プレイで複数のエンディングイラストをアンロックできますか？**

A: はい、特定の条件を重ねることは可能です。例えば、「パーフェクトガード」（No. 4）を達成しながら「スピードラン」（No. 5）を完了することができます。ただし、「パシフィスト」（No. 1）と「クラウドコントロール」（No. 6）は、一方が射撃を禁じ、もう一方が特定の武器使用を要求するため、1回のランで両立させることはできません。

**Q: ラスボス戦でダメージを受けるとNo. 4の条件は無効になりますか？**

A: はい、イラストNo. 4の**プラグマタ 体験版 エンディング 条件**は、ヒューを操作可能になった瞬間から最終リザルト画面まで「ノーヒット」であることを要求します。ボス戦を含め、どの時点でヒットを受けても、最初からやり直す必要があります。

**Q: すでにアンロックしたイラストはどこで見られますか？**

A: 現在、体験版ではクリア後のサマリー画面でのみイラストが表示されます。再度確認するには、同じ条件でクリアするか、手動でキャプチャした場合はプラットフォームのスクリーンショットギャラリーを確認してください。

**Q: エリートのオレンジ・ロボットの出現はランダムですか？**

A: 一見ランダムに見えますが、エレベーター内の最初の3体の敵に対して3つの異なる殺害方法（標準銃、ダイアナのアビリティ、グリーンガン）を使用することで発生する確定的なスポーンであることがコミュニティによって確認されています。この特定のシーケンスに従うのが最も確実な方法です。
