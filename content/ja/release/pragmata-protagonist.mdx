## ダブル主人公システム：ヒューとダイアナ

『PRAGMATA』の核心は、その「エイム、ハック、デストロイ」というリズムにあります。多くのゲームに伴走キャラクターが登場しますが、『PRAGMATA』はその関係を共生レベルまで引き上げています。ヒューが物理的な存在感と伝統的な火力を提供する一方で、ダイアナは戦術的な頭脳として機能し、デジタル環境を操作してヒューの武器を実際に効果的なものにします。

| 特徴 | ヒュー（人間） | ダイアナ（アンドロイド） |
| :--- | :--- | :--- |
| **主な役割** | ダメージディーラー / 破城槌 | 戦術支援 / ヘビーヒッター |
| **動作** | 重厚で慎重、600ポンドのスーツの重量感 | 流動的で機敏、しばしば背中に乗る |
| **戦闘入力** | 従来の銃器とガジェット | グリッドベースのハッキングインターフェース |
| **主な強み** | クラウドコントロールとシールド除去 | 甚大なダメージと環境制御 |

このゲームは、プレイヤーにリアルタイムでのマルチタスクを強います。単にキャラクターを切り替えるだけでなく、両方の入力を同時に管理することが求められる場面も少なくありません。これにより、反射神経だけでなく知的な判断力も試される「知的なリズム」が生まれます。

## ヒュー：装甲の先遣隊

ヒューは**プラグマタの主人公**チームにおける「力（マッスル）」ですが、この世界において「力」は相対的な言葉です。巨大なハイテクスーツに身を包んだヒューは、過酷な月面環境を反映した重量感のある動きを見せます。彼の顔は隠されたままであり、より脆弱に見えるダイアナを守る「人間の殻」としての役割を強調しています。

その外見とは裏腹に、ヒューの標準的な弾丸は、高度なAIシールドに対してしばしば「ゴム弾」のようであると表現されます。彼の主な仕事は、ダイアナが致命的な書き換え（リワイアリング）を行えるよう、自身の兵装を使って外側の防御を剥ぎ取る破城槌として行動することです。

### ヒューの武器とガジェット

ヒューは、戦闘の流れを制御するために4つの異なる武器タイプを利用します。ダイアナの高ダメージハックを準備するためには、これらのツールの適切なシーケンス（順序）が極めて重要です。

| 武器タイプ | 主な機能 | 主な用途... |
| :--- | :--- | :--- |
| **グレネードランチャー** | 範囲攻撃 (AoE) | 小型の敵集団の管理。 |
| **ショックウェーブ・ライフル** | 精密な弱点露出 | ダイアナが標的にするための回路の露出。 |
| **ステイシス・ネット** | クラウドコントロール / 遅延 | 複雑なハッキングパズルのための時間稼ぎ。 |
| **デコイ・ジェネレーター** | 脅威の誘導 | AIを撹乱するために「ヒューの身代わり」を配置。 |

> **💡 ヒント:** ダイアナが高度なノードハックを実行している間は、デコイ・ジェネレーターを使用しましょう。これにより、敵が彼女のフローを妨害し、パズルのやり直しを余儀なくされるのを防ぐことができます。

## ダイアナ：戦術の中核

ヒューが肉体であるなら、ダイアナは**プラグマタの主人公**コンビにおける魂であり、主なダメージソースです。視覚的に、ダイアナは脆弱さを感じさせるようにデザインされています。長い髪の少女の姿をしており、しばしば裸足で、自分には大きすぎるように見えるスーツを着ています。しかし、このデザインは巧妙な覆し（サブバージョン）です。

ダイアナは月面基地のセキュリティシステムとインターフェースをとることができる高度なアンドロイドです。ヒューがライフルで傷一つつけるのに苦労する一方で、ダイアナはミサイルの軌道を変え、セキュリティグリッド全体をシャットダウンし、大規模なデジタル攻撃で敵を粉砕することができます。

### ハッキングインターフェース

ダイアナがハッキングを開始すると、プレイヤーはグリッドベースのインターフェースに入ります。コントローラーの前面ボタンを使用して、ノードを介してルートを作成する必要があります。
*   **長いルート:** 敵の脆弱な状態が続く時間が長くなります。
*   **ノードの消費:** ハッキングノードは消耗品ですが、ゲーム内では頻繁な使用が推奨されています。
*   **アップグレード可能な能力:** 進行に合わせて、ダイアナはマルチターゲットハックや、より迅速なシールド破壊能力を獲得していきます。

## 「エイム、ハック、デストロイ」の戦闘ループ

**プラグマタの主人公**としての成功は、3段階の戦闘ループをマスターできるかどうかにかかっています。このゲームで慎重さが罰せられることは稀ですが、計画なしに「撃ちまくる（ラン・アンド・ガン）」行為には厳しいペナルティが課せられます。

1.  **エイム（狙う）:** ヒューがターゲットを特定し、ショックウェーブやグレネードを使用して敵の外装を剥ぎ取ります。
2.  **ハック（乗っ取る）:** ダイアナが露出した回路にロックオンします。プレイヤーはAIの「脳」をバイパスするためにリアルタイムパズルを解きます。
3.  **デストロイ（破壊する）:** ハッキングが成功すると、敵はヒューの高ダメージバースト攻撃やダイアナの環境オーバーライドに対して脆弱になります。

このループは、デジタルで再現されたタイムズスクエアのような「広大なアリーナ」において、ますます複雑になります。これらの広い空間では、空間認識能力が正確さと同じくらい重要な「ソフトスキル」となります。ダイアナが作業している間、複数の角度から飛んでくる飛び道具を避けつつ、彼女を守るためにヒューを配置しなければなりません。

## 物語のテーマ：サピエンスとAI

メカニクスを超えて、**プラグマタの主人公**の関係性は深い哲学的な領域を探求しています。開発者は、物語がアンドロイドの自我の妥当性に焦点を当てていることを示唆しています。クレヨンで絵を描いたり、独り言のように歌を歌ったり、人間との繋がりを求めたりするダイアナの行動は、彼女が本当に自我を持っているのか、それとも人間の保護を確実にするために「可愛く」プログラムされているだけなのかという疑問を投げかけます。

2人のコントラストは鮮明です。
*   **ヒュー:** ロボットのような、顔のない殻の中にいる人間。
*   **ダイアナ:** 人間のような、表情豊かな殻の中にいるロボット。

この逆転は、真の人間性がどこにあるのかを見つけるために、「殻よりも深く」見ることをプレイヤーに求めていることを示唆しています。暴走したAIが月面コロニーを占拠した敵対者である中で、ヒューとダイアナの絆は、冷徹で論理的な脅威に対する感情的な拠り所として機能します。

## リソース管理と最適化

『PRAGMATA』は伝統的なサバイバルホラーではありませんが、カプコンの他の伝説的なフランチャイズからリソース管理の要素を借りています。ダイアナのハッキングノードは豊富に手に入りますが、無限ではありません。

> **警告:** 最も強力なノードを「後で」使うために温存しないでください。ゲームの難易度は敵の数が増えることでスケールするため、最終ボスまでリソースを貯めることよりも、リアルタイムの最適化と迅速な問題解決の方が重要です。

### カスタマイズとロードアウト
プレイヤーは、自分のプレイスタイルに合わせて**プラグマタの主人公**のビルドをカスタマイズできます。
*   **ハッカービルド:** ダイアナのノード効率と、より速いシールド破壊に焦点を当てます。
*   **タクティカルビルド:** ヒューのクラウドコントロールとデコイの持続時間に焦点を当てます。
*   **バランスビルド:** 初めてプレイする人にお勧めのパスで、どちらのキャラクターもバランスよく成長させます。

## FAQ

**Q: 単独の主人公はいますか？それとも常に2人を操作するのですか？**

A: ヒューとダイアナを共生ユニットとして両方操作します。世界を移動する際のメインキャラクターはヒューですが、ダイアナは戦闘からパズル解きまで、ほぼすべての行動にメカニカルに統合されています。

**Q: 戦闘中にダイアナが死ぬことはありますか？**

A: ダイアナは「ヘビーヒッター」であり驚くほど耐久性がありますが、ハッキング中に中断されたり、直接ダメージを受けすぎたりすると、ヒューが脆弱な状態になります。ゲームはダイアナの伝統的なヘルスバーよりも、彼らのパートナーシップの失敗に焦点を当てています。

**Q: ハッキングノードがなくなったらどうなりますか？**

A: ハッキングノードは月面基地のいたるところで見つかります。もし使い果たしてしまったら、ヒューの基本武器を頼りにノードをスカベンジ（収集）しなければなりません。彼の標準的な射撃は装甲化されたAIに対して効果が低いため、難易度は大幅に上がります。

**Q: 二人の主人公による協力プレイ（Co-op）はありますか？**

A: 2026年のリリース時点では、『PRAGMATA』は1人のプレイヤーが両方のキャラクターの「知的な判断」を管理するシングルプレイヤー体験として設計されています。重なり合う入力は、このゲーム独自の挑戦（チャレンジ）の中核をなす部分です。
