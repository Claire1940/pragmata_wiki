**プラチナトロフィー（platine pragmata）**の獲得は、カプコン愛好家にとって名誉の証です。これには、テクニカルな戦闘スキル、細部まで行き届いた探索、そしてゲームの拠点におけるソーシャル管理の融合が求められます。この至難の100%コンプリートには、少なくとも2回のフルプレイが必要です。1回目はすべてのアップグレード・コンポーネントと収集品の回収に集中し、続く2回目で過酷な難易度「Lunatic（ルナティック）」に挑みます。効率的に攻略を進めるには、まずカジュアル（Casual）難易度から始めて、垂直方向のレベルデザインや複雑な「ジャンプ・ダッシュ・グライド」による移動システムに慣れることをお勧めします。本ガイドでは、隠された「ミニ・キャビン」の狙撃から、特定のギフトや対話によるダイアナの好感度の最大化まで、あらゆるトロフィーを確保するための必須ステップを解説します。このロードマップに従えば、月面探索中にホログラム・ファイルや脱出用ハッチを見逃すことはありません。

## トロフィーハンター向け必須設定

最初のプレイを開始する前に、設定を調整することで戦闘やプラットフォーム・アクションのストレスを大幅に軽減できます。『PRAGMATA』も多くのカプコン作品の伝統に漏れず、デフォルトのカメラ速度は、特に激しいボス戦において鈍く感じられる場合があります。

| 設定 | 推奨値 | 理由 |
| :--- | :--- | :--- |
| **カメラ速度** | 最大 (Full) | 高速で移動するウォッチャーを追うために不可欠。 |
| **エイム時のカメラ速度** | 60 | 弱点を狙う際のエイムの行き過ぎを防止。 |
| **減速** | 0 | 常に一定のエイム速度を維持し、筋肉の記憶を定着させる。 |
| **アダプティブトリガー** | オフ | 長時間のプレイによる指の疲労を軽減。 |
| **モーションブラー** | オフ | ミニ・キャビンのような小さな収集品の視認性を向上。 |

> **💡 ヒント:** 常に「センターレティクル」を有効にしておきましょう。これにより、クイック・ハッキング・ミニゲームで青いノードをより正確にヒットできるようになります。

## 移動とハッキングのメカニクス

プラチナトロフィーを目指すには、ジェットパックのスタミナ・システムをマスターする必要があります。標準的なアクションゲームとは異なり、『PRAGMATA』のジャンプには燃料ベースの「ウエストポーチ型」ジェットパックを使用します。特にレアなアップグレード・コンポーネントへ続くような大きな溝を飛び越えるには、特定の入力シーケンスに従う必要があります：**ジャンプ、次にダッシュ、そしてグライド。** ダッシュの前にグライドしてしまうと、動作を完了するためのスタミナが足りなくなり、垂直な環境を長い時間をかけて登り直すことになります。

ハッキングは戦闘ループの核心です。ワーカーやウォッチャーといった敵に遭遇した際、ハッキングを開始すると彼らの青い内部コアが露出します。
1. **コアを露出させる:** 照準を合わせ、青いノードを通じてハッキングを開始します。
2. **ダメージを増加させる:** コアが青くなると、敵は受けるダメージが大幅に増加します。
3. **武器の切り替え:** ショットガンのような重武器のリロード中には、グリップガン（弾薬無限）を使用しましょう。

## シェルター：好感度とビンゴシステム

シェルターはダイアナと交流するための拠点となります。彼女の好感度を上げることは、プラチナ獲得において避けて通れません。ミッションから戻るたびに、彼女の会話をすべて出し切りましょう。「音符」のアイコンが表示されたら、それは絆レベルが上がった合図です。

### ダイアナへのギフト
*   **REMグローブ:** 序盤の研究所のカットシーン後に発見可能。
*   **クレヨン:** メインコントロールロビーの「隠し壁（中空の壁）」の背後に隠されています。
*   **ギフト報酬:** これらのアイテムを贈ると、キャビン・コインや「私に？（For Me?）」などのユニークなトロフィーが手に入ります。

### ビンゴゲーム
シェルターのキャビン・ガイは、キャビン・コインを使用したビンゴ・ミニゲームを提供しています。効率的な100%プレイのためには、まず**中央の垂直ライン**に集中しましょう。このルートを進むと「レッドゲート・キー」が手に入ります。これは高レベルのアップグレード・コンポーネントがある終盤の隠しエリアにアクセスするために不可欠です。

| ビンゴスタンプ | 報酬 | 戦略 |
| :--- | :--- | :--- |
| **上段中央** | クラッシャー・ボットのデータ | 敵図鑑トロフィーに不可欠。 |
| **中央** | ワーカー・ボットのデータ | 標準的な敵のデータ。 |
| **下段中央** | **レッドゲート・キー** | 探索における最優先事項。 |

## ボス戦略：太陽光発電所の守護者

プラチナ獲得への道のりで最初の大きな障害となるのが、発電所のボスです。この遭遇戦では「オーバードライブ」メカニクスと「クリティカルアタック」が導入されます。

*   **突進パターン:** ボスは常に2回連続でダッシュします。反撃を試みる前に、2回目のダッシュが終わるのを待ちましょう。
*   **ステイシス・ネット:** ステイシス・ネットを発射して、ボスをその場に拘束します。
*   **弱点:** ボスの背中にあるオレンジ色のタンクが主要なターゲットです。ショットガンを至近距離で使用し、大ダメージを与えましょう。
*   **オーバードライブ:** ゲージが溜まったら、**L3 + R3**を押してダイアナのサポート攻撃を発動させます。これによりボスが怯み、トドメを刺すチャンスが生まれます。

> **警告:** 難易度Lunaticでは、ボスのダッシュ後の回復が非常に早くなります。ステイシス・ネットのタイミングを完璧に合わせないと、すぐに圧倒されてしまいます。

## 武器のアップグレードと装備構成

難易度Lunaticの攻略を管理しやすくするためには、カジュアルでのプレイ中に装備を完全にアップグレードしておく必要があります。プライマリ・ユニット（銃）は、反動の軽減と装弾数の増加のためにレベル6を優先的に目指すべきです。

1. **プライマリ・ユニット:** 安定したDPS（秒間ダメージ）のために最優先。
2. **ショットガン:** 「クラッシャー」タイプの敵に対して不可欠。
3. **ステイシス・ネット:** 敵のクラウドコントロール（足止め）に最適なユーティリティ・ツール。
4. **硬化スーツMod:** 常に装備しておきましょう。HPが10%アップし、高難易度での一撃死を防ぐことができます。

攻略を進める際は、シェルターでの手動セーブを忘れないでください。ミッション中はセーブができず、脱出用ハッチを最近起動していない場合、一度の死亡で収集品の進行状況が大幅にリセットされる可能性があります。

## FAQ（よくある質問）

**Q: コレクション品を見つけるためのマップはありますか？**

A: 意外なことに、『PRAGMATA』には伝統的なミニマップが存在しません。視覚的なランドマークとミニ・キャビンが発する音を頼りに移動する必要があります。ゲーム後半で手に入る「スキャナー」アップグレードを使用すれば、壁越しのアイテムをハイライトできるようになります。

**Q: プラチナトロフィー獲得には何回の周回が必要ですか？**

A: 最低2回の周回が必要です。1周目は難易度カジュアルでアイテムを100%回収し、ダイアナの好感度を最大にします。2周目の難易度Lunaticでは、純粋に生存とクリアに集中します。

**Q: 以前のエリアでミニ・キャビンを取り逃した場合はどうなりますか？**

A: トラム（路面電車）を使えば、太陽光発電所などの以前のエリアへファストトラベルで戻ることができます。ただし、ストーリーの進行状況によってロックされるエリアもあるため、チェックリストを使いながら進む際に見つけておくのが最も安全です。

**Q: 難易度はビンゴの報酬に影響しますか？**

A: いいえ、ビンゴボードの報酬は固定です。ただし、ビンゴのラインを揃えて追加のヘルス・カートリッジを入手しておけば、難易度Lunaticでの攻略がはるかに楽になります。カプコンの最新タイトルに関する詳細な情報は、[Capcom Dev 1 Twitter](https://twitter.com/dev1_official)公式アカウントでアップデートやパッチノートを確認してください。
