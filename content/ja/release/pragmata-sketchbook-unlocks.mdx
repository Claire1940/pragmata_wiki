## 秘密のエレベーター・エンカウント

現在の謎の核心は、3体のロボット歩哨が進行を阻止しようとするエレベーターのシークエンスにあります。ほとんどのプレイヤーは最も強力な武器を使ってこの部屋をクリアしてしまいますが、実はそうすることで隠しコンテンツから締め出されてしまいます。最も重要な**PRAGMATA スケッチブックのアンロック**の前提条件である隠しボスを出現させるには、全兵装を使いこなす実力を示す必要があります。

ゲームは、最初の3体のボットそれぞれに与えられた「決定打（フィニッシングブロー）」を追跡しています。3体すべてを同じ武器で倒すと、シークエンスは通常通り終了します。しかし、とどめを刺す武器を分散させると、ゲームはプレイヤーの戦術的習熟度を検知し、エレベーターが目的地に到着した際に隠された高ティアの敵をスポーンさせます。

### 武器によるフィニッシュ要件

隠し敵のスポーンを成功させるには、「3のルール」に従う必要があります。各ボットは異なるダメージソースで破壊されなければなりません。敵を弱らせる際はいかなる武器を使っても構いませんが、ゲームがチェックするのは、敵の体力をゼロにする最後の一撃のみであることに注意してください。

| ターゲット | 推奨フィニッシュ武器 | 備考 |
| :--- | :--- | :--- |
| **ボットA（1体目）** | **ピストル / 標準射撃** | 最初のキルとして最も制御しやすい。 |
| **ボットB（2体目）** | **ショックウェーブガン / 重火器** | 爆風が誤ってボットCに当たらないよう注意。 |
| **ボットC（3体目）** | **ハック / マルチハック・フィニッシュ** | フィニッシャーとしてハッキング機能を使うのは非常に確実。 |

> **⚠️ 警告:** 効果範囲（AoE）武器を使用する際は細心の注意を払ってください。1つの爆発や衝撃波で2体のボットを同時に倒すと、両方とも同じ武器ソースとしてカウントされ、隠しアンロックは失敗します。

## ダイアナの2枚目の絵をアンロックするためのステップバイステップガイド

隠しボスのアンロックは、ダイアナの2枚目の絵と直接結びついています。PRAGMATAの世界において、ダイアナのスケッチは、共に直面する脅威に対する彼女の認識を表しています。隠しエレベーターの敵に遭遇しなければ、彼女はそれを描くための「インスピレーション」を得ることができず、スケッチブックに永久に空白が残ることになります。

成功を確実にするために、エレベーターのシークエンス中に以下の手順に従ってください：

1.  **ターゲットを隔離する:** 3体のボットがエレベーターに降りてきたら、隅に移動して敵のヘイト（注意）を引きます。
2.  **体力を削る:** プライマリライフルを使用して、3体すべてのボットの体力を約10%まで減らします。コンボを出しすぎないように注意してください。
3.  **1体目のキル:** サイドパーム（ピストル）に切り替え、1体目のボットに正確な一撃を与えます。
4.  **2体目のキル:** ショックウェーブガンに切り替えます。2体目のボットが3体目から離れるのを待ち、ブラストを放って仕留めます。
5.  **最後のキル:** 最後のボットに近づき、ハックを開始します。ハックのシークエンスを完了させてボットを破壊すれば、3つ目のユニークな武器要件が満たされます。
6.  **出現:** エレベーターが停止すると、通常の出口の代わりに「侵入者検知（Intruders Detected）」という警告が赤く点滅し、隠しボスがアリーナにフェーズイン（出現）します。

### アンロックのための詳細な武器メカニクス

| ダメージソース | アンロックカテゴリー | フィニッシュの有効性 |
| :--- | :--- | :--- |
| **ピストル** | 物理（Kinetic） | **高** - 単発ダメージの制御が非常に容易。 |
| **ショックウェーブガン** | エネルギー/スタン | **中** - 周囲のボットへの巻き込みリスクあり。 |
| **マニュアルハック** | テクニカル | **高** - 体力が低ければ単体フィニッシュが確定。 |
| **マルチハック** | テクニカル | **低** - 誤って複数の敵を同時に倒す可能性がある。 |

マルチハックの使用についてはプレイヤーの間で議論があります。マルチハックによるダメージ自体が要件を「壊す」わけではありませんが、1回のマルチハック発動による最後のダメージ判定で2体の異なるボットが死亡した場合、ゲームはそれらを同じ死亡ソースとして登録します。最も確実に**PRAGMATA スケッチブックのアンロック**を達成するには、単体ターゲットのフィニッシャーに徹してください。

## 完璧主義者のための戦闘戦略

**PRAGMATA スケッチブックのアンロック**を試みる際、最大の敵は往々にして自分自身の攻撃力です。デモで装備をアップグレードしていくと、エレベーターのボットを誤って「ワンパン（一撃）」で倒してしまいやすくなり、武器を切り替える時間が取れなくなります。

### ダメージ閾値の管理

誤爆を防ぐために、エレベーター戦の前に高ダメージのMODを外すことを検討してください。ボットにある程度の「硬さ」を持たせることで、慎重に体力を削ることができます。タイミングに苦労している場合は、以下を試してみてください：

*   **ピストルの指切り:** 戦闘の大部分をピストルで行います。DPSは最低ですが、精度は最高です。
*   **視覚的な合図:** ボットが火花を散らしたり煙を出し始めたりするのを見逃さないでください。これはPRAGMATAにおける、敵の体力が15%以下であるという共通のサインです。
*   **ハック・インジケーター:** ボットの体力が十分に低くなると、「フィニッシャー・ハック」のプロンプトが表示されます。これがアンロックのためにボットを仕留める合図です。

> **💡 ヒント:** もしローテーションを間違えて2体のボットを同じ武器で倒してしまった場合は、直前のチェックポイントから素早くやり直すことができます。チェックポイントはエレベーターに乗る直前で発生するため、高速でリトライが可能です。

## FAQ

**Q: 「マルチハック」は、PRAGMATA スケッチブックのアンロックにおいて、1つの武器としてカウントされますか、それとも複数としてカウントされますか？**

A: マルチハックは単一のダメージソースとしてみなされます。マルチハックを使用して2体のボットを同時に倒した場合、ユニーク武器フィニッシュの要件は失敗となります。しかし、1体のボットのみを倒すために使用した場合は、ユニークな「ハック」フィニッシュとしてカウントされます。

**Q: ボットを倒さない限り、ショックウェーブガンでダメージを与えても大丈夫ですか？**

A: はい。ゲームが追跡しているのは「決定打（Killing Blow）」のみです。敵の体力を減らすために、どのような武器の組み合わせを使用しても構いません。ボットを破壊する最後の一撃が、他の2体のキルとは異なる武器であれば、秘密はアンロックされます。

**Q: 隠しボスをアンロックした後に戦闘で死んでしまった場合はどうなりますか？**

A: 隠しボスが出現した後に死んでしまった場合、通常は出現させるためにエレベーターのシークエンスをやり直す必要があります。ボスのスケッチブック項目は、敵を倒し、ダイアナが絵を描くアニメーションを完了した時点で初めて保存されます。

**Q: 武器を使用する特定の順番はありますか？**

A: いいえ、順番は関係ありません。ハックを1番目、ピストルを2番目、ショックウェーブガンを3番目にしても、あるいはその逆でも構いません。唯一の厳格なルールは、3つの決定打がすべて異なる武器タイプから繰り出される必要があるということです。
