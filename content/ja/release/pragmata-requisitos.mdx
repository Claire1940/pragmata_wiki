## 公式Pragmata Requisitos：PC最小スペック

古い構成のPCを使用している場合、月面の環境を処理できるか心配かもしれません。最小要件は、1080p解像度でプレイ可能な体験を提供することを目的として設計されています。カプコンによると、これらのスペックでの目標は「パフォーマンス」プリセット使用時に約45フレーム/秒（fps）です。

本作は従来のハードディスクドライブ（HDD）もサポートしていますが、ヒューの戦闘シークエンスとダイアナのハッキングセグメント間のシームレスな移行を処理するために、SSDの使用が強く推奨されます。以下は、エントリーレベルのハードウェアの詳細な内訳です。

| コンポーネント | 最小要件 |
|---|---|
| **OS** | Windows 11 (64-bit) |
| **プロセッサー (CPU)** | Intel Core i5-8500 / AMD Ryzen 5 3500 |
| **メモリ (RAM)** | **16 GB** |
| **グラフィックス (GPU)** | NVIDIA GeForce GTX 1660 6GB / Radeon RX 5500 XT 8GB |

> **警告:** これらのスペックを下回るハードウェアで実行すると、暴走したAIドローンとの激しい戦闘シーンで大幅なスタッタリング（カクつき）が発生する可能性があります。

## 高度なパフォーマンスとレイトレーシングの詳細

『プラグマタ』はカプコンの技術の見本市です。本作は、物理的な障害を乗り越えるヒューを導きながら、同時にデジタルな脅威を管理するダイアナを操作する、独自の2キャラクター操作システムを採用しています。この同時処理はCPUに独特の負荷をかけます。「ハック・アンド・スラッシュ」のシークエンス中にフレームレートが低下する場合は、オプションメニューで「AIロジック」や「物理演算」の設定を下げることを検討してください。

**pragmata requisitos**を確認する際、多くのユーザーがVRAMの重要性を見落としがちです。本作は高解像度テクスチャを使用して、月面にある「人類文明の残骸」を描写します。高設定では8GBのVRAMが基準となりますが、1440p解像度では12GBが理想的です。

### パフォーマンスモードの解説
1. **パフォーマンスモード:** 内部解像度をスケーリングすることでフレームレートを優先します。最小スペックレベルのユーザーに最適です。
2. **バランスモード:** 高品質なシャドウと安定した60fpsのパフォーマンスを両立させます。
3. **クオリティモード:** 描画距離とテクスチャフィルタリングを最大化します。RTX 30シリーズまたはRX 6000シリーズ以上を使用している場合にのみ推奨されます。

## ストレージとインストールのヒント

本作の必要容量は40GBで、2026年のAAAタイトルとしては比較的控えめです。しかし、データストリーミングの要求は強烈です。本作は、月面から研究施設内部への移動において「ゼロ・ローディング」の哲学を採用しています。

> **💡 ヒント:** 可能であれば、必ずNVMe SSDにPragmataをインストールしてください。従来のSATA SSDでも十分ですが、NVMeドライブを使用すると、ファストトラベルやキャラクター交代のアニメーション中に発生する微細なスタッタリングを大幅に軽減できます。

[Steam Deck](https://store.steampowered.com/app/3357650/PRAGMATA/)でプレイする場合、本作が**「確認済み（Verified）」**であることを知れば喜ばれるでしょう。ハンドヘルドデバイス上でも「低〜中」のハイブリッドプリセットを使用することで、安定した30〜40fpsで驚くほど快適に動作し、外出先での月面探索に最適です。

## Pragmata デラックスエディション vs スタンダードエディション

どのエディションを選ぶかも、最初の体験に影響を与えます。推奨される**pragmata requisitos**を満たしている方にとって、デラックスエディションはハイエンドな視聴覚エンジンを活かしたいくつかの外見・音声強化特典を提供します。

| 機能 | スタンダードエディション | デラックスエディション |
|---|---|---|
| **ゲーム本編** | 含む | 含む |
| **シェルター・バラエティパック** | なし | 含む |
| **ヒューのコスチューム** | 標準 | グラビテーター、ムーンキャット |
| **ダイアナのコスチューム** | 標準 | ロボメカニック、キティ |
| **デジタルアートブック** | なし | 含む |
| **限定ジェスチャー** | なし | 含む（おねむ、ワクワク） |

デラックスエディションに含まれる「シェルター・バラエティパック」には、ゲーム内拠点「シェルター」で再生できるLo-fiやジャズアレンジのサウンドトラックも収録されています。これは、暴走AIとの命がけのミッションの合間にリラックスするのに最適な方法です。

## ゲームプレイメカニクス：ヒューとダイアナを使いこなす

**pragmata requisitos**をクリアすることは、旅の始まりに過ぎません。ゲームが起動したら、2人の主人公のシナジーをマスターする必要があります。ヒューは主要な戦闘員であり、キネティックな火器と身体能力を駆使して月面の廃墟を進みます。謎のアンドロイドであるダイアナは、環境オブジェクトのハッキングやシールドの生成によってユーティリティを提供します。

- **同時操作:** 単に切り替えるだけでなく、両者を同時に管理することがよくあります。ヒューが援護射撃を行っている間に、ダイアナがリアルタイムでセキュリティ端末をバイパスする必要があるかもしれません。
- **3Dプリント鉱物:** 物語は、即座に3Dプリントを可能にする新鉱物を中心に展開します。ゲームプレイでは、ヒューがその場で一時的な遮蔽物や特殊な弾薬を「プリント」できる能力として反映されています。
- **月の重力:** 物理エンジンは低重力環境に基づいた動きを計算します。ジャンプや回避は浮遊感がありますが正確で、標準的な地上のアクションゲームとは異なるタイミングが要求されます。

## FAQ

**Q: Pragmataは常時オンライン接続が必要ですか？**

A: 本作は主にシングルプレイヤー体験ですが、初回のアクティベーション（Denuvo経由）や、コミュニティ主導の「データライブラリ」機能へのアクセスにはブロードバンドインターネット接続が必要です。

**Q: Windows 10でPragmataを実行できますか？**

A: 公式の**pragmata requisitos**では、明確にWindows 11 (64-bit)が指定されています。Windows 10でも起動する可能性はありますが、カプコンは公式にサポートしておらず、互換性の問題やDirectX 12 Ultimate機能によるパフォーマンスの低下が発生する可能性があります。

**Q: PCのパフォーマンスをテストするための体験版はありますか？**

A: はい、Steamで「PRAGMATA SKETCHBOOK」デモをダウンロードできます。月面研究施設の短いセグメントをプレイして、製品版を購入する前に戦闘やハッキングシステムが自分のハードウェアでどのように動作するかを確認できます。

**Q: ミドルレンジPCに最適な設定は？**

A: RTX 3060のようなミドルレンジのリグでバランスの取れた体験を得るには、グローバルイルミネーションを「中」に設定し、DLSSまたはFSRを「クオリティ」モードで有効にし、テクスチャ品質を「高」に保ってください。これにより、1080pで安定した60fpsを維持できるはずです。

**Q: ウルトラワイドモニターに対応していますか？**

A: はい、Pragmataは21:9および32:9のアスペクト比にネイティブ対応しており、月面の風景をより映画のように楽しむことができます。ピクセル数の増加に対応できるよう、GPUが推奨**pragmata requisitos**を満たしていることを確認してください。
