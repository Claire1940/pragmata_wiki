## 『PRAGMATA』で強くてニューゲームを解放する方法

「強くてニューゲーム（NG+）」モードを解放する手順はシンプルですが、まずは物語を最後まで完遂する必要があります。難易度を問わずメインストーリーのキャンペーンをクリアしてください。エンディングが流れ、最終セーブが記録されると、メインメニューからNG+が選択可能になった旨の通知が表示されます。

NG+を開始することは、いわば世界を「ループ」させるようなものです。ストーリー展開は同じですが、プレイヤーの強化されたレベルに合わせてゲーム環境が変化します。これは装備を最大レベルまで強化したり、1周目の混乱の中で見逃した隠し収集品を回収したりするための主な手段となります。

| 条件 | 内容 |
| :--- | :--- |
| **キャンペーンのクリア** | 最終シネマティックに到達し、「クリアデータ」を保存する。 |
| **セーブファイルの選択** | クリア済みのセーブファイルをロードしてNG+サイクルを開始する。 |
| **難易度の選択** | 標準的なスケーリング、または新たに解放された「ルナティックモード」から選択する。 |

## 2周目に引き継がれる要素

NG+の醍醐味の一つは、苦労して手に入れた力を保持できることです。新規セーブで始めるのとは異なり、NG+ではほとんどのメカニカルなアドバンテージを維持できます。これにより、敵の強さが自分のステータスに追いつくまでの序盤戦を「ビクトリーラン（凱旋）」のような感覚で楽しめます。

### 装備とアップグレード
主要なスーツのアップグレード、特殊兵装、ガジェット類はすべて次のサイクルに引き継がれます。推進システムや戦闘モジュールの最大強化のために素材集めに費やした時間は無駄になりません。これは、NG+の後半戦に待ち受ける高難易度の脅威に立ち向かうために不可欠です。

### 収集品と進行状況
探索の進行状況もセーブデータ間で追跡されます。月面で見つけたデータログやコスメティックアイテムなどはインベントリに残ります。さらに、各種「シミュレーション（Sims）」やサイドミッションの進行状況も引き継がれるため、全体の達成率を100%にするための作業を中断した場所から再開できます。

| カテゴリ | 引き継ぎ可否 | 備考 |
| :--- | :--- | :--- |
| **戦闘装備** | **はい** | すべての武器とスーツモジュールを保持。 |
| **アップグレード** | **はい** | すべてのステータス上昇とスキルツリーの解放状況を保持。 |
| **収集品** | **はい** | データログやロアアイテムはライブラリに残る。 |
| **ミッション進行度** | **一部のみ** | 完了済みのSimsは完了状態を維持。メインストーリーはリセット。 |
| **隠し部屋** | **いいえ** | 新しいサイクルを開始するとアクセス権を失う。 |

## 難易度の上昇と敵のスケーリング

NG+ではプレイヤーが大幅に強化されていますが、敵もまた進化を遂げています。戦闘の緊張感を維持するためにスケーリングシステムが採用されており、敵の体力と攻撃力が大幅に上昇していることに気づくでしょう。

さらに重要なのは、NG+では「敵の行動パターン」が変化する点です。1周目では終盤に登場したエリート個体が、序盤から出現することがあります。これらの変異体はAIの挙動が調整されており、より攻撃的で予測しにくい動きを見せます。このメカニズムの変化こそ、「**PRAGMATAにNG+はあるのか**」という問いに対し、単なる数値調整以上の答えを開発者が用意した証と言えます。

> **⚠️ 警告:** NG+の序盤のボスを侮ってはいけません。エンドゲーム装備を持っていても、新しい攻撃パターンによって、以前の慣れに頼った戦い方では足元をすくわれる可能性があります。

## ルナティックモードの解放と攻略

2026年において究極の挑戦を求めるプレイヤーのために、ゲームを1回クリアすると「ルナティックモード」が解放されます。これは最高難易度のティアであり、ゲームの基本メカニクスをマスターしたプレイヤー向けに設計されています。

ただし、ルナティックモードには重要な注意点があります。**この難易度ではすべての要素を引き継ぐことはできません。** 挑戦の純粋さを保つため、特定の高レベルアップグレードやエンドゲームアイテムが制限されたり、リセットされたりする場合があります。これにより、プレイヤーは装備の力だけでなく、純粋なスキルで攻略することを強いられます。

| 特徴 | 通常のNG+ | ルナティックモード |
| :--- | :--- | :--- |
| **敵のダメージ** | 高い | 極めて高い |
| **アイテム引き継ぎ** | 全て | 制限あり |
| **敵のAI** | 高度なパターン | 最大限の攻撃性 |
| **チェックポイント** | 通常通り | 制限あり |

## 隠し部屋とクリア後のコンテンツ

コンプリートを目指すプレイヤーにとって重要なのが「隠し部屋（Hidden Chamber）」の詳細です。この場所は、ゲームの最も深いクリア後コンテンツや秘密のチャレンジの拠点となります。NG+のサイクルを開始すると、**物語の特定のポイントまで到達するまで隠し部屋へのアクセス権を失う**ということを理解しておく必要があります。

すべてのシークレットミッションを完了させたい、あるいは特定のエンドゲーム素材を集めたい場合は、「強くてニューゲームを開始する」ボタンを押す前に済ませておくべきです。

> **💡 エキスパートのヒント:** 隠し部屋にアクセス可能な状態の、独立した手動セーブファイルを常に残しておきましょう。これにより、特定のアップグレードや収集品を逃したことに気づいた際、キャンペーンを最初からやり直すことなく、すぐにクリア後のコンテンツに戻ることができます。

## NG+を成功させるための戦略

2度目の旅を最大限に楽しむために、以下の戦略的な推奨事項に従ってください。

1.  **未回収の収集品を優先する:** 強化された機動力を駆使して、以前は到達が困難だったエリアを探索しましょう。
2.  **ビルドを試す:** ほとんどの装備が解放されているため、1周目では無視していた武器の組み合わせを試してみましょう。
3.  **パリィを極める:** 敵のダメージが増加しているため、生のDPS（火力）よりも防御メカニクスが格段に重要になります。
4.  **リソース管理:** アイテムは引き継がれますが、上位のクラフト素材は依然として希少です。お気に入りの装備セットの最終強化のために温存しておきましょう。

最新のパッチやコンテンツのドロップに関する情報は、[カプコンのPRAGMATA公式サイト](https://www.capcom.co.jp/pragmata/)をチェックして、コミュニティの議論や公式アップデートを常に把握しておきましょう。

## NG+機能のまとめ

『PRAGMATA』の強くてニューゲームは、単なるリプレイではありません。ゲームシステムへの習熟度が試される、より洗練された体験です。装備やアップグレードを引き継ぐことで「無双」の楽しさを味わえる一方で、より賢く強力になった敵や、過酷なルナティックモードによって絶妙なバランスが保たれています。

最後の数個のトロフィーを狙う場合でも、フル強化されたスーツで再び物語を体験したい場合でも、NG+は最高のプラットフォームを提供してくれます。隠し部屋用のバックアップセーブを忘れずに用意し、変化した月面の脅威に備えましょう。

## FAQ

**Q: PRAGMATAに限定スーツなどのNG+報酬はありますか？**

A: ほとんどの装備は1周目から引き継がれますが、NG+やルナティックモードで特定の節目を達成することで、通常のプレイでは入手できない限定のコスメティックスキンやスーツデカールが解放されます。

**Q: 強くてニューゲームを開始する際にキャラクターの外見を変更できますか？**

A: はい。新しいサイクルを開始する際、カスタマイズメニューにアクセスでき、外見の変更や1周目で獲得した新しいコスメティックアイテムの適用が可能です。

**Q: NG+での通貨はどうなりますか？**

A: 蓄積した通貨やクラフト素材はすべて引き継がれます。これにより、新しいサイクルで最初の商人に会った直後に、高価なエンドゲームアイテムやアップグレードを購入することが可能です。

**Q: 強くてニューゲームを開始できる回数に制限はありますか？**

A: 現在のところ、セーブファイルを無限にループさせることができます。各周回（NG++、NG+++など）でも最高難易度の敵スケーリングが維持されますが、2ループ目以降の難易度の上昇幅は緩やかになります。

**Q: NG+を開始した後、通常のニューゲームのセーブに戻ることはできますか？**

A: 別途セーブスロットを作成していた場合に限り可能です。新しいサイクルを開始する際に唯一のセーブファイルを上書きしてしまうと、ゲームは「NG+開始前」の状態を自動的に保持しません。必ず複数のスロットを使用してください！
