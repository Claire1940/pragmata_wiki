## セクターガード・プロトコル：最初の試練

**pragmata all bosses**リストにおける最初の大きな壁はセクターガードです。この遭遇戦は、主電源の再接続中にエディスが不正アクセスを検知したときに発生します。ガードは、連射エネルギー弾と範囲（AoE）スキャンの組み合わせを使用します。

セクターガードを倒すには、「無力化スキャン」を回避しながらエネルギーレベルを維持することに集中する必要があります。ガードのシールドが赤く光ったときがオーバードライブ・プロトコルのチャンスです。これを発動することで、ヒューは短時間で大ダメージを与えることができます。

1.  **電力を再接続する:** 3つのノード間を移動し、エネルギーを60%以上に保つ。
2.  **スキャンを避ける:** 地面が光ったら、すぐに円形の範囲外へダッシュする。
3.  **ダイアナのアシスト:** 「これを使って！」のコマンドでダイアナにボットの注意を引かせ、背面の冷却ベントを露出させてクリティカルヒットを狙う。

## 空の支配者：プテロドン

プテロドン戦では戦場が空へと移ります。この機械の獣は輸送機を守っており、急降下爆撃と音響の叫びを駆使します。これは**pragmata all bosses**の進行において最も機動力が必要な戦いの一つであり、正確なエイムが求められます。

プテロドンが飛行している間は、ライフルで翼の関節を削りましょう。急降下爆撃のモーションに入ったら、動き続けてください。構造物の柱に激突させるように誘導できれば、数秒間地上に拘束できます。これが重火器やオーバードライブ・プロトコルでとどめを刺すチャンスです。

## クレイドル・セキュリティ四脚機との対峙

シェルターのライセンスキー付近に登場するこの四脚ボットは、エリア拒絶の達人です。床にオレンジ色の光る軌跡を残し、短時間で爆発する「ヒートライン」を使用します。

### 四脚機攻略テーブル

| フェーズ | ボスの行動 | プレイヤーの戦略 |
| :--- | :--- | :--- |
| **フェーズ 1** | 尻尾レーザーの追尾 | ビームを避けるために狭い円を描くように走る。 |
| **フェーズ 2** | ヒートラインの格子 | ラインの間の「冷たい」青いゾーンに留まる。 |
| **フェーズ 3** | グランドスラム | 衝撃波を避けるために、着弾の直前にジャンプする。 |

> **警告:** 尻尾が光っている間は、四脚機に近接攻撃を仕掛けないでください。カウンター掴みが発動し、ヒューの体力の50%が即座に削られる可能性があります。

## 最終決戦：エディスとデッド・フィラメント

PRAGMATAのクライマックスは、汚染されたAI「エディス」と、増殖する「デッド・フィラメント」のバイオマスとの多段階のバトルです。この戦いでは、ゲーム全体で学んだすべてのメカニクスが試されます。エディスはエリアを封鎖しようとするため、脱出路を確保するためにダイアナのハッキング・マトリクスを使用する必要があります。

最終段階では、デッド・フィラメントが巨大な不定形の脅威となり、輸送レールを飲み込もうとします。ヒューが重キャノンをチャージする間、シャトルを守らなければなりません。

*   **ハッキング・マトリクス:** ダイアナがハッキング状態に入ったら、ヒューは彼女を小型の「デッド」のスポーンから守る必要があります。
*   **重キャノン:** キャノンはチャージ時間が長いです。フィラメントが「目」またはコアを開くのを待ってから発射してください。タイミングを外すと長いクールダウンが発生し、シャトルが脆弱になります。
*   **犠牲:** エンディングシーケンスでは、ヒューが手動でシャトルレールを押し出す必要があります。この間、ダイアナがシャトルから援護射撃を行う中、インタラクトボタンを連打しなければなりません。

## ボス戦におけるダイアナのサポート役割

**pragmata all bosses**との遭遇戦を通じて、ダイアナの役割は受動的な観察者から主要な戦闘員へと進化します。彼女の特定の能力を理解することで、困難な戦いも管理可能なものになります。

*   **エネルギー回復:** ダイアナは倒した小型の敵からエネルギーセルを回収し、ヒューのスーツの電力を維持できます。
*   **囮（デコイ）:** 彼女の小さな体格を活かして大型ボットの懐に潜り込み、ヘイトを引くことができます。
*   **ハッキング:** エディス・メインフレームのシールドを無効化するために不可欠です。
*   **属性バフ:** ゲーム後半では、ダイアナはヒューの弾薬に「氷」や「炎」の属性を付与でき、ボスの弱点を突くために重要になります。

## FAQ

**Q: pragmata all bossesの中で最も倒すのが難しいのはどれですか？**

A: 多くのプレイヤーは、移動が制限されるヒートラインのメカニクスを持つ「クレイドル・セキュリティ四脚機」が最も難しいと感じています。成功の鍵は、ボス自身だけでなく床のパターンを注視することにあります。

**Q: ダイアナの能力を使わずにボスを倒せますか？**

A: 基本的な敵はソロで対処可能ですが、主要なボスは通常、ダイアナのオーバードライブやハッキング・プロトコルで無効化する必要がある「ハードライト・シールド」を持っています。協力プレイはゲームデザインの核心的な要件です。

**Q: ボス戦に「ノーダメージ」報酬はありますか？**

A: はい、**pragmata all bosses**をダメージを受けずにクリアすると、ヒュー用のコスメティックセット「ムーン・ゴースト」と、ミッション結果画面で特別な「パーフェクト・シンク」メダルがアンロックされます。

**Q: ボス戦でヒューが死ぬとどうなりますか？**

A: ヒューの体力がゼロになると、ダイアナは遭遇戦ごとに一度だけ「メモリー・フィラメント」を使用して彼を蘇生させようとします。彼女も戦闘不能になるか、すでに蘇生を使用している場合は、最後のチェックポイントからやり直しになります。
