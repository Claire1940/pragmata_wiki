## パークシステム：核となるゲームプレイの改造

Pragmataにおいて、パークはキャラクター性能の主要な「Mod」として機能します。これらはダメージ、防御、ユーティリティにパッシブボーナスを与える装備スロットです。従来のRPGのようにステータスが永続的なものとは異なり、Pragmataでは各ランの前にこれらのパークを入れ替えることができ、特定のミッション要件に適応する柔軟性を備えています。

現在、最も効果的なビルドは「スローダウン」と「空中戦闘」を中心としています。ジャスト回避を利用することで、クリティカルヒットを与えるための大きな隙を生み出すタイムダイレーション（時間遅延）効果が発動します。この状態と相乗効果のあるModを装備することで、DPSを50%以上向上させることができます。

| パーク名 | 効果説明 | ダメージボーナス |
| :--- | :--- | :--- |
| **イーグルアイ** | 空中にいる間に与えるすべてのダメージが増加する。 | **+15%** |
| **クローズクォーター** | 短距離で敵を攻撃した際にダメージボーナスを付与する。 | **+10%** |
| **秘められた可能性** | ハッキング成功直後に一時的なダメージブーストが発動する。 | **+10% (15秒)** |
| **精密射撃** | 回避によるスローダウン効果中のダメージを特化して増加させる。 | **+10%** |
| **ラストリゾート** | メイン武器以外の武器の最後の弾丸に大幅なダメージブーストを与える。 | **+50%** |

> **💡 エキスパートのヒント:** **イーグルアイ**パークは、間違いなく序盤で最も価値のあるModです。ジャンプしながらの射撃は基本的な防御メカニズムであるため、この15%のブーストの恩恵をほぼ常に受けることができます。

## ハッキングノード：戦術的戦闘Mod

ハッキングはPragmataの戦闘ループの根幹です。相棒のダイアナを通じて、戦場にある様々なノードに干渉し、リアルタイムの戦闘改造として機能させることができます。これらのノードは色によって分類され、それぞれ異なる戦術的役割を担っています。敵の密度が大幅に増す終盤の生存には、「マルチハック」コンボのマスターが不可欠です。

### ノードの種類と機能

1.  **黄色ノード（タクティカル）:** これらはクラウドコントロールを提供する限定的なリソースです。敵をその場に凍結させたり、防御ステータスを下げたり、あるいは敵を混乱させて味方として戦わせたりすることができます。
2.  **青色ノード（ダイアナアップグレード）:** これらはダイアナのハッキング速度とダメージを強化します。黄色ノードとは異なり、より頻繁に使用することが可能です。
3.  **紫色ノード（クリティカルショット）:** これらは敵の体力が30%以下になると出現します。敵をよろけさせ、「ヒートゲージ」による処刑攻撃の隙を作ります。

| ノードコンボ | 必要なノード | 戦闘結果 |
| :--- | :--- | :--- |
| **フリーザーバーン** | 凍結 + 防御ダウン | 敵はダメージ耐性ゼロの状態でその場に固定される。 |
| **クラウドキラー** | マルチハック + 凍結 + 防御 | 周囲のすべての敵にステータス異常を適用する。 |
| **インフィニットループ** | 青色ノード + ダイアナ連打 | 弾薬を消費せずに持続的な削りダメージを与える。 |

## 武器の改造とビルドの相乗効果

Pragmataの武器は、サブ武器については「使い捨て」のメカニズムを採用しています。メインライフルは常に所持していますが、**チャージピアサー**やショットガンのような上位武器は、フィールドで拾うかシェルターで購入します。これらを真に最適化するには、**mods pragmata**のパークを武器独自の特性に合わせる必要があります。

例えば、**ラストリゾート**パークはチャージピアサーと組み合わせると驚異的です。チャージピアサーのデフォルトダメージは1677ですが、ラストリゾートModを使用すると、最後の1発のダメージは2377に跳ね上がります。これにジャスト回避（精密射撃）と空中ジャンプ（イーグルアイ）を重ねれば、一撃で2500を超えるダメージを叩き出すことも可能です。

### 推奨ロードアウト

*   **スナイパービルド:** イーグルアイと精密射撃に集中。スローダウン中にチャージピアサーを使用して、遠距離から優先順位の高いターゲットを排除します。
*   **ハッキングスペシャリスト:** 秘められた可能性とマルチハックノードを優先。ダイアナを使って雑魚敵を殲滅しつつ、ボス用に弾薬を温存します。
*   **タンク/ブローラー:** クローズクォーターとステイシスネットを使用。敵を拘束し、ヒートゲージ処刑でトドメを刺します。

> **警告:** 黄色ノードを溜め込みすぎないようにしましょう。リソースは限られていますが、中級の「エリート」敵に対して使用することで、最終的に到達できないかもしれないボスのために温存するよりも、体力と弾薬を節約できます。

## 高度な戦闘メカニズム：ヒートと機動力

標準的な**mods pragmata**の枠を超えて、プレイヤーは「ヒートゲージ」をマスターしなければなりません。これは敵の体力バーの直下にあるスタガーメーターです。ゲージが満タンになると、R2ボタンを押してテレポート近接処刑を実行できます。これは莫大なダメージを与え、標準的な敵なら一撃で倒せることが多いです。

### 機動力とエネルギー管理
動き続けることは絶対条件です。しかし、ジャンプやブーストはエネルギーを消費します。
*   **ブーストダッシュ:** 段差から飛び出す際は、フルジャンプするよりもブーストダッシュで前方に飛び出す方がエネルギー効率が良いことが多いです。
*   **モーションキャンセル:** Pragmataの回復モーションは非常に遅いことで知られています。UI上で体力が回復した瞬間にダッシュやジャンプをすることで、モーションの後半をキャンセルできます。
*   **フリーファイア:** 狭い通路では、照準（L2）を避けてください。射撃ボタンをタップするだけの「フリーファイア」なら、移動速度を高く保ったまま素早い回避が可能です。

## テクニカルな最適化：RE Engineの設定

PragmataはRE Engineで動作しているため、他のカプコン作品と共通のグラフィック特性があります。隠しアイテムやハッキングノードを見つけやすくするために、ハードウェアに合わせて明るさとディスプレイ設定を調整することをお勧めします。

| ディスプレイタイプ | カラースペース | 推奨輝度設定 |
| :--- | :--- | :--- |
| **小型モニター** | sRGB | 最初のスライダーを最小にして黒を深くする。 |
| **大型TV/ディスプレイ** | REC 709 | 最後のスライダーを+1または+2にしてHDRの輝きを強調する。 |
| **HDR有効** | ゲームネイティブ | 中央のスライダーでUIのコントラストを調整する。 |

これらの設定を適切に行うことは一種の「視覚的Mod」として機能し、壁越しの「ダイアナピン」の識別や、ステージ100%達成に必要な隠し通路の発見が格段に容易になります。

## FAQ

**Q: 序盤で手に入る最強のPragmata Modはどこにありますか？**

A: 序盤の最強改造は、シェルターにて100通貨で購入できます。まずは戦闘効率を最も高めてくれるイーグルアイ（Eagle Eye）パークとマルチハック（Multihack）ノードに集中しましょう。

**Q: 「ラストリゾート」パークはメインライフルにも効果がありますか？**

A: いいえ、ラストリゾートはショットガンやチャージピアサーのように、弾薬を使い切ると破棄されるメイン以外の武器専用に設計されています。

**Q: ダイアナのハッキングダメージを増やすにはどうすればいいですか？**

A: ハッキングボード上の青色ノードを経由することで、ハッキングダメージを増加させることができます。さらに、シェルターでダイアナをアップグレードすることで、ハッキング速度とダメージ出力を永続的に強化できます。

**Q: PragmataでコミュニティModを使用することはできますか？**

A: ゲーム内の「Mod（改造）」はコア機能ですが、外部ソフトウェアによる改造はプラットフォームに依存します。PC版プレイヤーはコミュニティサイトを通じてビジュアルやパフォーマンスのModにアクセスできることが多いですが、これらはカプコンによって公式にサポートされているものではなく、オンライン機能に支障をきたす可能性があります。
