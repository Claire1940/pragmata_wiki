## IDUS AIとEDIシステムの理解

『PRAGMATA（プラグマタ）』の世界において、IDUSは単にステージの最後に戦うボスではありません。月面施設全体を支配する遍在的な脅威です。このAIはEDI（Electronic Defense Interface：電子防衛インターフェース）システムを利用して、あなたの進行状況をナレーションし、さらに重要なことに、あなたが「未登録のゲスト」として検知されるたびに「セクターガード・プロトコル」を初期化します。

本作の戦闘がユニークなのは、それが施設の電力網と密接に結びついている点です。施設がロックダウン状態にあるとき、IDUSの力は最大化され、こちらの攻撃能力は制限され、防御的でステルス重視のプレイスタイルを余儀なくされます。ダイアナと協力して主電源を復旧させるにつれ、戦闘のダイナミクスは生存維持から積極的な反撃へと変化していきます。

> **💡 ヒント:** EDIシステムの音声ガイダンスを注意深く聞いてください。AIが「不正アクセスを検知しました」とアナウンスしたときは、エリートユニットがあなたの正確な座標に配備されようとしている直接的な警告です。

## 基本メカニクス：エネルギー閾値

戦闘効率は、そのセクターで回復させたエネルギーの割合に直結しています。**pragmata idus**が完全な制御権を握っている状態（エネルギー0-30%）では、ヒューの銃器の威力は低下し、ジェットパックのチャージ速度も著しく低下します。電力ノードを再接続することだけが、戦況を対等にする唯一の方法です。

| エネルギーレベル | ステータス効果 | 推奨される戦術 |
| :--- | :--- | :--- |
| **0% - 30%** | **ロックダウン有効** | ステルスを優先。ダイアナを使って電力ノードを偵察する。 |
| **31% - 60%** | **攻撃警告** | EDIがセクターガードを配備。遮蔽物を見つけて防衛に徹する。 |
| **61% - 90%** | **電力再接続中** | 攻撃バフが有効。ヒューの銃器で敵の波を殲滅する。 |
| **100%** | **ロックダウン解除** | エリア確保。体力と弾薬が完全に回復する。 |

## 高度な戦闘：オーバードライブシステム

圧倒的な数のセクターガードに対抗するため、プレイヤーはオーバードライブシステムを活用しなければなりません。このメカニクスは、ヒューがスーツを通じて利用できる、施設のエネルギーの一時的なサージ（急増）を表しています。オーバードライブゲージを溜めるには、パリィを成功させ、高エネルギーな戦闘フローを維持する必要があります。

最強のツールは「オーバードライブ・フォンコール」と呼ばれる、戦闘能力を飛躍的に高めるユニークなアビリティです。この発動中、ヒューはダメージ耐性が向上し、IDUSに制御されたロボットの重装甲を貫通できる特殊な「ヒート（熱）」ベースの攻撃が可能になります。

### オーバードライブ・アビリティ一覧

| アビリティ | 効果 | 最適な使用場面 |
| :--- | :--- | :--- |
| **オーバードライブ・ダッシュ** | 敵の背後へ瞬時にテレポート | 遠距離のミサイルバッテリーとの距離を詰める際。 |
| **ヒートブラスト** | 広範囲のエネルギー爆発 | 狭い通路でスカウトドローンの群れを掃討する際。 |
| **プライマリ・セバー** | 強力な単体攻撃 | 電力をリセットしようとするエリートプロテクターの無力化。 |

> **警告:** セクターエネルギーが40%未満の状態でオーバードライブを発動すると、「ブラウンアウト（電圧低下）」状態を引き起こす可能性があります。アビリティ終了後、ヒューは数秒間動けなくなり、追撃に対して無防備になります。

## 敵の種類と対抗策

**pragmata idus**の軍勢は、小型で俊敏なドローンから、巨大で固定された砲撃ユニットまで多岐にわたります。各ユニットは厳格なロジックパターンに従っており、それを理解すれば隙を突くことができます。

### セクターガードのバリエーション

1.  **スカウトドローン**: IDUSの「目」です。攻撃力は低いですが、EDIシステムにあなたの位置を知らせ、即座に増援を呼びます。隠密性を維持するため、最優先で破壊してください。
2.  **ミサイルバッテリー**: 強力な火力支援を行います。標準的な実弾攻撃に対しては装甲を持っていますが、冷却ベントという致命的な弱点があります。一斉射撃の後に「ヒート」サインが出るのを待ち、露出したベントを攻撃して大ダメージを与えましょう。
3.  **エリートプロテクター**: 最も危険な通常敵です。彼らの目的は「プライマリ・コネクションの切断」であり、電力復旧の進行状況をリセットしようとします。動きは遅いですが体力が高く、ダイアナが防衛システムをハッキングした後でなければ効果的にダメージを与えられません。

| 敵ユニット | 主要攻撃 | 弱点 |
| :--- | :--- | :--- |
| **スカウトドローン** | スキャン / 警告 | **低耐久**。連射性能の高い武器に弱い。 |
| **ミサイルバッテリー** | 多連装ミサイル斉射 | **オーバーヒート**。リロード中にベントが露出する。 |
| **エリートプロテクター** | コネクション切断 | **鈍重**。オーバードライブ攻撃に非常に弱い。 |

## 主人公のシナジー：ヒューとダイアナ

『PRAGMATA（プラグマタ）』のゲームプレイの核となるのは、ヒューとダイアナの関係性です。ヒューが銃器やジェットパックで物理的な戦闘を担当する一方で、ダイアナは**pragmata idus**のデジタル防衛をバイパスできる唯一の存在です。

戦闘中、プレイヤーは両方のキャラクターを同時に操作できます。装甲をまとったロボットに直面した際、ヒューの弾丸は弾き返されてしまいます。そこでダイアナを配置してハッキングを開始させる必要があります。これによりリアルタイムのパズルが発生し、グリッド上のカーソルを誘導してターゲットタイルに到達させます。

ダイアナがハッキングしている間、ヒューは彼女のボディーガードを務めなければなりません。パズル中もゲームは停止せず、周囲の世界は動き続けています。ヒューは飛来する弾丸を回避し、近接攻撃をパリィして、ダイアナが中断されないように守り抜く必要があります。ハッキングが完了するとロボットの装甲が無効化され、ヒューが破壊できる輝く弱点が露出します。

## 勝利のための最終戦闘ヒント

*   **熱（ヒート）管理**: ヒューと敵の両方が熱を発生させます。ヒューがオーバーヒートするとスタン状態になります。可能な限り環境を利用して熱を逃がしてください。
*   **電力を優先**: セクター内のすべてのロボットを倒したくなりますが、主な目標は常に電力ノードです。エネルギーレベルが高まれば、すべての戦闘が大幅に楽になります。
*   **視線を切る**: EDIシステムの赤いスキャンラインがヒューに重なったときは、ミサイルのロックオンが迫っている合図です。即座にジェットパックを使用して遮蔽物の後ろに隠れ、ロックを解除しましょう。
*   **エネルギードロップを回収**: 倒した敵はエネルギーシャードをドロップします。これらは戦闘中に体力を回復し、オーバードライブゲージを溜めるために不可欠です。

## FAQ

**Q: Pragmata IdusでAIがプライマリ・コネクションを切断するのを防ぐにはどうすればいいですか？**

A: 「不正アクセス」の警告が表示されたら、起動したばかりの電力ターミナルに向かっているエリートプロテクターを探してください。これらのターゲットを即座に無力化する必要があります。オーバードライブ攻撃の「プライマリ・セバー」を使用するのが、進行状況をリセットされる前に阻止する最短の方法です。

**Q: 60%の攻撃警告に対処する最善の方法は何ですか？**

A: 電力60%では、EDIシステムが非常に攻撃的になります。最善の戦略は、次のノードへの移動を一旦止め、防御しやすい場所を見つけることです。まず襲来するセクターガードの波を殲滅し、「攻撃警告」ステータスが消えてから前進してください。

**Q: 戦闘中にダイアナを操作することはできますか？**

A: はい、このゲームはデュアルコントロールが可能です。ヒューが主な戦闘員ですが、敵の防御を無効化するために、ダイアナをハッキングポイントへ移動させ、リアルタイムでグリッドパズルを完了させる必要があります。

**Q: セクターをクリアせずに体力を回復する方法はありますか？**

A: はい。電力復旧を100%にすると全回復しますが、破壊したスカウトドローンやミサイルバッテリーからエネルギードロップを回収することで少量の体力を回復できます。また、ダイアナがハッキングシーケンス中にオプションのボーナスノードからヘルスキットを見つけることもあります。
