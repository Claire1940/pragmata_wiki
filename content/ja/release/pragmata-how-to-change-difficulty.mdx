## PRAGMATA 難易度の変更方法：ステップバイステップの手順

月面ドローンが手強すぎると感じたり、ハッキングパズルで進行が止まってしまったりした場合は、設定を調整できます。なお、難易度を下げることはいつでも可能ですが、難易度を上げるには、リーダーボードの整合性を保つために特定の条件を満たすか、チャプターを最初からやり直す必要がある場合が多いので注意してください。

設定を調整するには、以下の手順に従ってください：
1. ゲームを一時停止して**メインメニュー**を開く。
2. **オプション（Options）**タブに移動。
3. **ゲームプレイ設定（Gameplay Settings）**を選択。
4. **難易度（Difficulty）**の切り替え項目を探す。
5. 好みのレベルを選択して変更を確定する。

多くの場合、特定の戦闘やハッキングシーケンスで何度も失敗すると、ゲーム側から自動的に「難易度を下げる」プロンプトが表示されます。これは、ヒューがクレイドルの迷路のようなセキュリティノードを操作するダイアナを護衛しなければならない、緊迫した場面で特に役立ちます。

## 難易度がハッキングと戦闘に与える影響

PRAGMATAのコアとなるループは、ダイアナが基地のシステムをハッキングしている間、ヒューが彼女を守るというものです。**PRAGMATAの難易度変更**をマスターすることは、単に敵の体力を変えるだけではありません。ハッキングミニゲームの内容も大きく変化します。高難易度では、迷路内の緑色のゴールが動的に移動し、ハッキングを無効化するノードがより頻繁に出現します。

### 戦闘のスケーリング
ヒューはデコイ・ジェネレーターやステイシス・ネットなど、さまざまなツールを使用できます。難易度が上がると、機械仕掛けの「クリーンアップ・クルー（Cleanup Crews）」などの敵は、これらのツールに対してより早く適応するようになります。例えば、ノーマルでは10秒間機能するデコイも、ハードのエリート敵相手には3秒間しか気を引けないかもしれません。

| メカニクス | ストーリーモードの影響 | ハードモードの影響 |
| :--- | :--- | :--- |
| **ステイシス・ネット** | 敵を15秒間拘束 | 敵を5秒間拘束 |
| **デコイ・ジェネレーター** | 高いヘイト生成 | 1回使うと敵に無視される |
| **武器の耐久度** | 高い耐久度 | 武器の破損速度が30%アップ |
| **ダイアナのハッキング** | 迷路に障害物なし | 動くレッドゾーンの障害物 |

## アンノウン・シグナル（Unknown Signal）モード：究極の挑戦

標準的な8時間のキャンペーンをクリアしたプレイヤーには、「アンノウン・シグナル」モードが決定的なポストゲームコンテンツとして用意されています。このモードはいわゆる「強くてニューゲーム（New Game Plus）」として機能し、難易度が大幅に上昇します。このモードを開始する前に**PRAGMATAの難易度変更**の仕組みを理解しておくことは不可欠です。初回プレイにはなかったランダムな敵のスポーンや、より攻撃的なAI挙動が導入されるからです。

アンノウン・シグナルモードでは、ヒューとダイアナが獲得したすべてのアップグレードを駆使する必要があります。シェルターでギフトを贈ることで築き上げた「見知らぬ家族」の絆レベルは、防御力やハッキングスピードをわずかに上昇させるメカニカルな利点となり、生存のためにはほぼ必須の要素となります。

> **警告:** アンノウン・シグナルモードのプレイ中に難易度を下げると、そのシグナルの完了に関連する特定の報酬を受け取ることができなくなります。

## 体験のバランス調整

PRAGMATAは「パレットクリーナー（お口直し）」、つまり[Crimson Desert](https://steamcommunity.com/app/2407170)のような100時間を超える巨大なオープンワールドとは対照的な、テンポの良い直線的な体験として設計されています。ゲームが短めであるため、多くのプレイヤーがプレイ時間を延ばすために**PRAGMATAの難易度変更方法**を調べています。難易度を上げることで、批評家が指摘した「見た目以上に高いスキル上限」とより深く向き合うことになるからです。

行き詰まった場合は、難易度を下げる前に以下の戦術的な調整を検討してみてください：
*   **ポジショニング:** ヒューの機動力が鍵です。ダイアナがハッキングしている間、一箇所に留まらないようにしましょう。
*   **ツールの相乗効果:** デコイを設置する*前に*ステイシス・ネットを使用して、クラウドコントロールを最大化します。
*   **ハッキング・ノード:** 迷路内の黄色いノードを探してください。ハッキングが完了すると、周囲のすべての敵に大ダメージを与えることができます。

## 難易度設定のトラブルシューティング

特定のボスが「壁」のように感じられたとき、プレイヤーはよく**PRAGMATAの難易度変更**について検索します。メニューのオプションがグレーアウトしている場合、それは現在「引き返し不能」な戦闘シーケンスの中にいる可能性が高いです。現在の戦闘を終わらせるか、クレイドルのシェルターにある直前のチェックポイントをロードして、設定メニューへのフルアクセスを取り戻してください。

| 問題 | 考えられる原因 | 解決策 |
| :--- | :--- | :--- |
| **オプションがグレーアウト** | 戦闘中のロック | 戦闘を終えるかチェックポイントをロード |
| **「ハード」オプションがない** | 初回プレイの制限 | プロローグを完了してすべて解放 |
| **設定が保存されない** | クラウド同期の競合 | ゲームを再起動し、メインメニューから適用 |

## FAQ

**Q: ゲームの途中でPRAGMATAの難易度を変更できますか？**

A: はい、基本的にはゲームプレイ設定メニューからいつでも難易度を下げることができます。ただし、難易度を上げる場合は、現在のチャプターをやり直すか、ニューゲームプラス（アンノウン・シグナル）を開始するまで待つ必要がある場合があります。

**Q: 難易度はストーリーの結末に影響しますか？**

A: いいえ、難易度に関わらずヒューとダイアナの核心的な物語は変わりません。ただし、高難易度でゲームをクリアすると、ユニークな外見アイテムや実績が解除されることがあります。

**Q: 「ストーリー」難易度の主な違いは何ですか？**

A: ストーリー難易度では、ダイアナのハッキング迷路に障害物がなくなり、ヒューの武器の耐久度が大幅に向上します。リソース管理を気にすることなく、物語に集中して楽しむことができます。

**Q: PRAGMATAに「パーマデス（永久死）」モードはありますか？**

A: 発売時点では公式なパーマデス設定はありませんが、アンノウン・シグナルモードでは、クレイドルでの究極の試練を求めるベテランプレイヤー向けに、特定のモディファイアとして「ワンライフ・チャレンジ」が用意されています。
