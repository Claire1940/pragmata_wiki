## PRAGMATA トロフィー・ロードマップと概要

具体的な要件に入る前に、トロフィーハントの全体像を把握することが重要です。PRAGMATAは、少なくとも2回のプレイを想定して設計されています。1回目は「未登録のゲスト」とダイアナの物語を楽しみ、2回目は高難易度設定で戦闘系のチャレンジを回収するためです。

| トロフィーカテゴリー | 合計数 | 推定難易度 |
| :--- | :--- | :--- |
| **プラチナ** | 1 | 6/10 |
| **ゴールド** | 3 | 高 |
| **シルバー** | 12 | 中 |
| **ブロンズ** | 35 | 低 |

> **⚠️ 警告：** 「クレイドル・セキュリティ・プロトコル」に関連するいくつかのトロフィーは、特定の環境インタラクションを完了せずに物語を進めてしまうと取り逃す可能性があります。新しいセクターを開始する際は、常にバックアップセーブを作成しておきましょう。

### ステップ1：ストーリークリアと基本探索
最初の優先事項はメインキャンペーンをクリアすることです。これにより、IDUSユニットとの初遭遇やクレイドル・セキュリティ・プロトコルの最初の突破など、ストーリー進行に関連するブロンズおよびシルバートロフィーのほとんどが自然に解除されます。この周回では、移動メカニクスとダイアナとの絆システムを習得することに集中してください。

### ステップ2：収集アイテムとサイドミッション
ストーリークリア後、またはクリーンアップ中に、地球に関連するすべての記念品を見つける必要があります。これらのアイテムは「眺めの良い家」で見つかることが多く、主人公の過去の背景を知ることができ、ゴールドトロフィー「青き惑星の記憶」の獲得に必要です。

## 戦闘およびスキル関連のPRAGMATAトロフィー

PRAGMATAの戦闘はタクティカルで、ダイアナの安全を確保しながらリソースを管理する必要があります。いくつかのトロフィーは、特定の戦闘実績や装備の可能性を最大限に引き出すことに関連しています。これらの**PRAGMATAトロフィー**を獲得するには、さまざまな武器タイプや環境トラップを試す必要があります。

| トロフィー名 | 要件 | タイプ |
| :--- | :--- | :--- |
| **プロトコル・ブレイカー** | 環境トラップを使用してIDUSユニットを50体撃破する。 | シルバー |
| **ルナ・エース** | 1回の戦闘中に50倍のコンボを維持する。 | シルバー |
| **未登録の伝説** | 難易度「エクステンション（Extinction）」でゲームをクリアする。 | ゴールド |
| **守護精霊** | ダイアナがダメージを受けることなく、主要な戦闘を完了する。 | シルバー |

「未登録の伝説」トロフィーを解除するには、敵の攻撃パターンを深く理解する必要があります。IDUSユニットは高難易度では非常に攻撃的で、その存在は常に安全を脅かします。彼らに対しては、直接対決するよりも「クレイドル」の防衛機構を利用する方が効果的な場合が多いです。

## 探索および収集アイテムガイド

PRAGMATAの世界には、地球での生活を彷彿とさせる隠されたディテールが満載です。これらを見つけることが、探索テーマの**PRAGMATAトロフィー**を獲得する主な方法です。これらのアイテムのほとんどは月面基地の目立たない場所にあり、ダイアナの助けを借りて解くパズルに隠されていることが多いです。

### 「眺めの良い家」の見つけ方
最も感動的なトロフィーの一つは、主人公のシミュレートされた地球の家を見つけることです。このエリアには複数の収集アイテムがあり、重要なロア（設定）が語られます。

1.  **セクター7アクセスポートを特定する：** これは2体目のIDUSボス戦の後に見つかります。
2.  **セキュリティ・プロトコルを解除する：** エアロック近くの端末を使用して、クレイドルのロックダウンをバイパスします。
3.  **リビングルームを探索する：** 暖炉の上にある「色あせた写真」を探すと、特別なダイアログがトリガーされます。
4.  **バルコニーを確認する：** シミュレートされた海の景色を見ることは、トロフィー「ノスタルジー」の獲得に不可欠です。

> **💡 ヒント：** スキャナーを頻繁に使用してください。バイザーをレベル3にアップグレードすると、収集アイテムは壁越しに見えるかすかな青いパルスを発するようになります。

## シークレットトロフィー

カプコンは、特定の条件を満たしたときにのみ明らかになるシークレットトロフィーをいくつか用意しています。これらには「イースターエッグ」や、ゲーム世界との一風変わったインタラクションが含まれます。

*   **静かの海：** ダイアナと一緒に月面から地球を10分間眺める。
*   **安全第一：** アラームを一度も鳴らさずにクレイドルのレーザーグリッドを通り抜ける。
*   **未登録のゲスト：** かつてのクレイドル科学者の隠されたIDカードを見つける。

ゲームの開発に関する詳細情報や公式アップデートについては、[カプコン PRAGMATA 公式サイト](https://www.capcom.com/pragmata/)で最新ニュースやパッチノートを確認してください。

## プラチナ獲得のための高度な戦略

**PRAGMATAトロフィー**リストを真に極めるには、効率的なプレイスタイルを最適化する必要があります。「エクステンション（Extinction）」難易度での周回は、ほとんどのプレイヤーにとって最大の難関です。この設定では体力が乏しく、敵は標準的な攻撃に対して非常に高い耐性を持っています。

| 戦略要素 | 推奨事項 |
| :--- | :--- |
| **主兵装** | 遠距離から安全に攻撃できる**パルスライフル**に集中する。 |
| **ダイアナのアビリティ** | かすり傷を最小限に抑えるため、**シールドプロジェクション**を優先する。 |
| **リソース管理** | **エナジーセル**はボス戦専用に温存する。 |
| **移動** | IDUSの範囲攻撃を避けるために**低重力ダッシュ**を活用する。 |

「クレイドル・セキュリティ・プロトコル」のトロフィーは、プレイヤーが最後に獲得することが多いものです。これらはゲーム中に点在する特定の端末を操作する必要があります。序盤のチャプターで見逃した場合、回収するにはニューゲーム+（New Game+）を開始する必要があるでしょう。幸いなことに、ゲームは複数の周回にわたって進行状況を追跡するため、全**PRAGMATAトロフィー**の獲得はより管理しやすくなっています。

## FAQ

**Q：取り逃す可能性のあるPRAGMATAトロフィーはありますか？**

A：はい、特定の環境インタラクションや「クレイドル」セキュリティ端末に関連するいくつかのトロフィーは、ストーリーを急いで進めると取り逃す可能性があります。次のセクターに移動する前に、各セクターを徹底的に探索することをお勧めします。

**Q：プラチナトロフィーには何回の周回が必要ですか？**

A：通常、最低2回の周回が必要です。1回目はストーリーと収集アイテムのため、2回目は最高難易度で「未登録の伝説」トロフィーを解除するためです。

**Q：ニューゲーム+でトロフィーを獲得できますか？**

A：はい、PRAGMATAはニューゲーム+でのトロフィー進行を完全にサポートしています。これは戦闘関連の実績を回収し、装備のアップグレードを最大化するための最良の方法です。

**Q：ゲーム内で最も難しいトロフィーは何ですか？**

A：多くのプレイヤーは、最高難易度でクリアする「未登録の伝説」が最も挑戦的だと感じています。次いで、攻撃的なIDUSのAIによって途切れやすい高コンボを要求される「ルナ・エース」が挙げられます。
