## プラグマタ体験版 エンディング攻略：全8種類のイラスト

ダイアナのスケッチギャラリーをすべてアンロックするには、特定の条件下で体験版をクリアする必要があります。これらの条件の中には互いに排他的なものもあるため、すべてを見るには通常3〜4回の周回プレイが必要です。ゲームは、総被ダメージ、経過時間、特定の武器の使用状況などの指標を追跡し、最終リザルト画面でどのイラストが表示されるかを決定します。

| イラストNo. | 条件名 | 達成基準 |
| :--- | :--- | :--- |
| **No. 1** | **パシフィスト・サポート** | ダイアナのアビリティのみを使用。強制的な最初の1発を除き、射撃禁止。 |
| **No. 2** | **エリート・エンカウント** | エレベーター内でレアな**オレンジ/赤のハイレベルロボット**を出現させ、撃破する。 |
| **No. 3** | **コンプリーショニスト** | すべての**イエローキューブ**を回収し、施設内のすべての大型宝箱を開ける。 |
| **No. 4** | **パーフェクト・ガード** | ダメージを1ポイントも受けずに体験版全体をクリアする。 |
| **No. 5** | **スピード・デーモン** | 合計**10分未満**で最終リザルト画面に到達する。 |
| **No. 6** | **クラウド・コントロール** | **グリーンガン**の1ショットで2体の敵を拘束し、無力化する。 |
| **No. 7** | **スカベンジャー** | 環境内に散らばっている全リソースの80%以上を回収する。 |
| **No. 8** | **サバイバー** | 特殊な条件を満たさない標準的なクリア。 |

> **💡 ヒント:** イラストNo. 1の場合、最初のカットシーンを発生させるために冒頭で正確に1発だけ撃つ必要があります。その直後、銃器を装備解除し、進行にはダイアナのハッキングとサポートスキルのみを頼りにしてください。

## エンディング No. 2の秘密：エリート・エレベーター・シーケンス

イラストNo. 2は、ボスアリーナへ向かう最後のエレベーター上昇中に特定の「バラエティ・キル」シーケンスを必要とするため、最もレアな**プラグマタ体験版 エンディング**報酬とされています。通常、エレベーターには標準的なドローンと二足歩行ロボットが出現します。しかし、特定の行動を正確に行うと、ゲームは報酬としてエリートのオレンジロボットを出現させます。

エレベーター搭乗中は、以下の手順に正確に従ってください：
1.  **ドローンの排除:** 標準のライフルまたはピストルを使用して、出現する飛行ユニットを素早く処理します。
2.  **ダイアナへの指示:** 最初の地上ロボットをターゲットにし、ダイアナのハッキング能力を使用して完全に無力化します。
3.  **グリーンガンでのトドメ:** グリーンガン（静止/制御用）に切り替え、2体目の標準ロボットにトドメを刺します。
4.  **エリートの出現:** 正しく行われれば、2体の高レベルなオレンジロボットが出現します。これらを倒すと、クリア時にNo. 2のイラストが確定します。

## 10分間チャレンジのためのスピードラン戦略

No. 5の**プラグマタ体験版 エンディング**を達成するには、「スピード・デーモン」のアプローチが必要です。これは、必須ではない戦闘を無視し、月面施設のレイアウトを完全に把握することを意味します。最も時間を浪費するのは、パワーセルの手動再起動と「緊急ロックダウン」シーケンスです。

| セグメント | 目的 | 時短テクニック |
| :--- | :--- | :--- |
| **初期突破** | 第1波をクリア | **オーバードライブ**を即座に発動し、数秒で部屋を制圧する。 |
| **パワーセル 1** | プロトコルの再ルーティング | 脇道は無視。3つのメイン端末へ一直線にダッシュする。 |
| **パワーセル 2** | 手動再起動 | ウサギ型の敵は動きが遅いため、戦わずに横をダッシュで通り抜ける。 |
| **最終エレベーター** | 侵入者の排除 | エレベーターの上昇を止められないよう、ドローンを最優先で処理する。 |

> **⚠️ 注意:** パワー端末に到達するために下層のメンテナンスシャフトに降りると、上層に戻ることはできません。複合ランを試みている場合は、飛び降りる前に必要なリソースをすべて集めているか確認してください。

## 探索とスカベンジング：すべてのイエローキューブを見つける

「コンプリーショニスト」（No. 3）または「スカベンジャー」（No. 7）のリザルトを目指すプレイヤーにとって、探索は不可欠です。月面基地は垂直構造になっており、ヒューがダイアナを高い足場へ持ち上げたり、彼女の「リフト」能力を使って隠れた梁に到達したりする必要があります。イエローキューブには重要な伝承（ロア）と高級素材が含まれており、大型の宝箱は破壊可能な壁の向こうに隠されていることが多いです。

No. 7の**プラグマタ体験版 エンディング**を確実にするには、徹底的な探索が必要です。HUDにはライブカウンターが表示されませんが、通路にある光る素材ドロップはすべて拾うようにしましょう。木箱や樽を壊すことも重要で、これらは総リソース数に大きく貢献します。

## 戦闘の習熟：パシフィストとクラウド・コントロール

『プラグマタ』の戦闘は、ヒューとダイアナのパートナーシップを中心に設計されています。「クラウド・コントロール」（No. 6）のランでは、グリーンガンの習熟度を示す必要があります。この武器は2体の敵を連結させるテザーを放ち、衝突させたりダメージを共有させたりすることができます。この条件を発生させるには、2体のロボットが近接しているとき（理想的には最初のロックダウン後の2回目の主要エンカウント時）を狙い、両者を繋ぐ一撃を放ってください。

パシフィストラン（No. 1）は、おそらく**プラグマタ体験版 エンディング**を体験する上で最もユニークな方法です。ヒューの頑強な体を盾にしてダイアナを守りつつ、彼女が環境ハッキングを行って脅威を無力化する「守護者」としてのプレイを強いられます。ダイアナのサポートアビリティにはクールダウンがあるため、管理を誤ると無防備になる可能性がある点に注意し、彼女のエネルギーレベルを慎重に管理する必要があります。

武器のスケーリングやフレームデータに関するより詳細な技術情報については、[プラグマタ公式サイト](https://www.capcom-games.com/pragmata/)をチェックし、製品版に向けた最新のメカニクス変更を確認してください。

## クリア後のコンテンツと「らくがきスーツ」報酬

8枚のイラスト以外にも、体験版を初めてクリアするとヒュー用の「らくがきスーツ」がアンロックされます。このコスチュームアイテムには、ダイアナが描いたと思われる様々な落書きがヒューのEVAスーツ全体に施されています。月面基地の重苦しい雰囲気に、ちょっとした遊び心を加えてくれます。

さらに、体験版ではヒュー（CV：田中美央）とダイアナ（CV：悠木碧）の深い絆が示唆されています。探索中の何気ない会話に注意を払うと、彼らの過去や、地球へ帰還するという最終目的についてのヒントが見つかるでしょう。体験版では制限されている「エスケープハッチ」のメカニズムは、製品版における「拠点への帰還」ループを示唆しており、安全地帯でダイアナと交流したり、装備をアップグレードしたり、アンロックしたスケッチを鑑賞したりできる可能性があります。

## FAQ

**Q: 1回のプレイで複数の プラグマタ体験版 エンディング イラストをアンロックすることは可能ですか？**

A: はい、特定の条件は重複させることができます。例えば、「パーフェクト・ガード」（No. 4）を達成しつつ、「スピード・デーモン」（No. 5）の時間制限をクリアすることは可能です。しかし、「パシフィスト」（No. 1）と「クラウド・コントロール」（No. 6）は武器の使用条件が矛盾するため、同じプレイで両立させることはできません。

**Q: 環境トラップからのダメージは No. 4 イラストの条件に影響しますか？**

A: はい、体験版の開始から最終ボスの撃破まで、ヘルスバーが少しでも減少すると「パーフェクト・ガード」の報酬からは除外されます。蒸気噴出や放電エリアを移動する際は、ダイアナのシールドアビリティを使用することを強くお勧めします。

**Q: すでにアンロックしたイラストはどこで確認できますか？**

A: 現在の「Sketchbook」体験版では、イラストはクレジット後の最終リザルト画面でのみ表示されます。再度見るには、同じ条件で体験版をクリアするか、コンソールやPCのスクリーンショット機能を使用してギャラリーに保存しておく必要があります。

**Q: ヒューの「らくがきスーツ」はどうやって入手しますか？**

A: どの**プラグマタ体験版 エンディング**イラストを獲得したかに関わらず、体験版を初めてクリアすると自動的に報酬として与えられます。ニューゲームを開始する前に、メインメニューから装備することができます。
