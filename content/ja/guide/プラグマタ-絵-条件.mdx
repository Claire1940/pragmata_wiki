## プラグマタ 絵 条件とアンロック要件の理解

プラグマタのエンディングイラストは単なる装飾ではありません。特定のゲームプレイの節目や「クリアスタイル」に紐付いています。「SKETCHBOOK」として知られる体験版では、最終画面に8つの異なるパターンが表示されます。これらのパターンは、環境とのインタラクション、戦闘効率、そしてステージ中のダイアナとの関係性によって決定されます。

希少なイラストの「**プラグマタ 絵 条件**」を満たすには、多くの場合、スピードと徹底した探索のバランスが必要です。急いでクリアしたプレイヤーは標準的なイラストを目にすることになりますが、月面施設の隅々まで探索したプレイヤーは、より詳細で物語的に重要なアートワークを報酬として受け取ることができます。

### 8つのエンディングイラストパターン

| パターンID | イラストの内容 | 主な解放条件 | 難易度 |
| :--- | :--- | :--- | :--- |
| **パターン 1** | 標準的な主人公 | 基本ランクでステージクリア | 初級 |
| **パターン 2** | ダイアナの笑顔 | ダイアナが無傷の状態でクリア | 中級 |
| **パターン 3** | 月の地平線 | 10分以内にステージクリア | 中級 |
| **パターン 4** | コンバットギア | セクター内のすべてのオプション敵を撃破 | 上級 |
| **パターン 5** | 秘密の部屋 | すべての隠しデータポイントを調べる | 上級 |
| **パターン 6** | 守護者と子供 | 「同期」ゲージを最大にする | 超上級 |
| **パターン 7** | 猫（ニコラ） | メンテナンスダクト内の隠し猫を見つける | 隠し |
| **パターン 8** | 全キャスト | Sランクを達成し、上記の全条件を満たす | マスター |

> **💡 ヒント:** パターン6の「**プラグマタ 絵 条件**」を安定して満たすには、ダイアナが「怯え」状態になることなく、彼女のサポートアビリティを少なくとも10回使用するようにしてください。

## クリア後のコンテンツ：ジムポッドミッション

最初のスタッフロールが流れた後、ゲームは大きく広がります。「秘密の部屋（Secret Chamber）」は、クリア後の進行における中心的な拠点となります。本編の最終ギャラリーで「**プラグマタ 絵 条件**」を完全に満たすには、ジムポッド（Jimpods）に関わる必要があります。これらは高難易度のチャレンジを提供する特殊なミッション端末です。

ジムポッドのミッションをクリアすることが、秘密の部屋のより深い階層をアンロックする唯一の方法です。進行に合わせて、最も重要なロア（背景設定）が隠された奥の部屋へアクセスできるようになります。

### ジムポッドの進行手順

1.  **初期起動:** 初回クリア後、秘密の部屋にあるジムポッドを見つける。
2.  **ミッション完了:** ジムポッドのインターフェース内で、すべての戦闘およびパズル試練を成功させる。
3.  **部屋のアンロック:** すべてのミッションをクリアすると、秘密の部屋の奥にある扉が開く。
4.  **最終報酬:** 奥の部屋にある端末にアクセスし、カスタムモジュール「ブラックボックス」を入手する。

## トラムの復旧とセクター探索

100%のコンプリートを達成し、真エンディングの「**プラグマタ 絵 条件**」を満たすには、月面トラムシステムを復旧させる必要があります。これにより、月面基地のさまざまなセクター間をファストトラベルできるようになります。各セクターには「セクター進行度」トラッカーがあり、収集品の発見、装備のアップグレード、以前のボスの「強化版」の撃破によって埋めていく必要があります。

### セクター進行チェックリスト

| 目的 | 100%達成条件 | 報酬 |
| :--- | :--- | :--- |
| **探索** | 全マップのサブエリアを訪問 | 強化マップデータ |
| **装備** | すべての主要ツールをレベルアップ | **フィラメントキャノン** |
| **ボス再戦** | すべての「強化版」ボスを撃破 | **オブシディア**スキン |
| **データログ** | 全50個のロアエントリーを収集 | **ブラックネオン**シェーダー |

> **⚠️ 警告:** 強化版のボスは攻撃パターンが大幅に変化しており、体力も増加しています。セクター3の再戦に挑む前に、少なくともレベル4のアーマーを装備しておくことをお勧めします。

## 真エンディングをアンロックする方法

真エンディングへの道は、プレイヤーのスキルの究極の試練です。単に基本的な「**プラグマタ 絵 条件**」を満たすだけでは不十分です。メインストーリーで遭遇したすべてのボスの「強化版」を計画的に撃破していく必要があります。これらの再戦をすべて完了して初めて、「真のラスボス」が出現します。

### 真エンディングへのワークフロー

*   **ステップ 1:** メインストーリーを1回クリアし、クリア後の状態をアンロックする。
*   **ステップ 2:** 利用可能なすべてのエリアでセクター進行度100%を達成する。
*   **ステップ 3:** 序盤のボスの強化版を撃破し、「真のラスボス」出現イベントを発生させる。
*   **ステップ 4:** 真のラスボスを撃破して最終シネマティックを鑑賞し、究極のエンディングイラストをアンロックする。

真エンディングの最終的な「**プラグマタ 絵 条件**」は、セーブファイル全体の合計「死亡回数」を追跡するため、非常に困難であることで知られています。複数回死亡していてもエンディングを見ることは可能ですが、スケッチブックの最後の黄金のフレームを入手するには、真のラスボスを「ノーデス（死亡なし）」でクリアする必要があります。

## 必須装備とモジュール

難易度の高いパターンの厳しい「**プラグマタ 絵 条件**」を満たすには、入手可能な最高の装備が必要です。ゲームでは、強力な武器や、機能的なバフも提供する装飾モジュールによって、徹底した探索が報われます。

### 主なアンロック報酬

*   **フィラメントキャノン:** 工業セクターの進行度を100%にすることで入手できる高出力エネルギー武器。真のラスボスのシールドを破壊するために不可欠です。
*   **オブシディア:** ステルス性を高める特殊なアーマーコーティングで、特定のエンディングイラストの「無ダメージ」条件を満たしやすくします。
*   **ブラックネオン:** ビジュアルを刷新するだけでなく、壁越しに隠された収集品をハイライトします。
*   **カスタムモジュール「ブラックボックス」:** 秘密の部屋をクリアした際の究極の報酬。同期ゲージを大幅にブーストし、パターン6の入手をはるかに容易にします。

## FAQ

**Q: デモ版の隠し要素である第8パターンの「プラグマタ 絵 条件」は何ですか？**

A: 第8パターンをアンロックするには、SKETCHBOOKデモをSランクでクリアし、隠し猫（ニコラ）を見つけ、ミッション全体を通してダイアナがダメージを一切受けないようにする必要があります。これは入手が最も難しいイラストです。

**Q: 「ブラックボックス」モジュールはエンディングイラストに影響しますか？**

A: ブラックボックス自体がイラストを変えるわけではありませんが、「守護者と子供」パターンの「**プラグマタ 絵 条件**」を満たすのが非常に楽になります。同期ゲージが安定し、ダメージを受けてもゲージが下がりにくくなるためです。

**Q: ゲームクリア後にエンディングイラストを変更することはできますか？**

A: はい。秘密の部屋からいつでも最終ミッションをリプレイできます。プレイスタイルを変える（例：スピードランを目指す、または完全探索を行う）ことで、異なる「**プラグマタ 絵 条件**」を発生させ、ギャラリーの新しいイラストをアンロックできます。

**Q: PS5版とSwitch 2版でエンディングイラストに違いはありますか？**

A: 「**プラグマタ 絵 条件**」とイラスト自体は、2026年に発売されるPlayStation 5版とNintendo Switch 2版を含むすべてのプラットフォームで共通です。ただし、PS5版はギャラリービューにおいて、より高解像度のアセットが使用されています。
