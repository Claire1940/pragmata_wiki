カプコンが贈る最新SF大作、月面コロニーの美しくも不気味な重力に抗う廃墟を探索するには、単なる戦闘スキル以上のものが必要です。それは、**PRAGMATA（プラグマタ）の全エンディングの出し方**を深く理解することです。心温まる真エンディングを目指すのか、それとも物悲しい「放棄（Abandonment）」ルートを辿るのか、キャンペーン中のあなたの選択がヒューとダイアナの運命を決定します。このガイドでは、2026年に発売された本作で100%コンプリートを達成し、すべての結末を見届けるための**全エンディング回収方法**を徹底解説します。

最良の結末を迎えるためには、最後の選択肢を選ぶだけでは不十分です。探索の徹底度、ダイアナとのシナジー（同調率）、そして月に散らばる「失われた地球の歴史」をどれだけ解明できたかの積み重ねが重要になります。メインストーリーを急いで進めると、デフォルトの結末に辿り着き、多くの謎が残されたままになってしまいます。

## 3つの主要な物語の結末

『PRAGMATA』の本編では、物語は3つの異なるルートに分岐します。これらはダイアナとの「同調率（Synchronization Rate）」と、最終幕までに特定のサイドオブジェクティブを完了しているかによって決定されます。

| エンディング名 | 雰囲気 | 主な条件 |
| :--- | :--- | :--- |
| **デフォルト・エンディング** | ほろ苦い | 標準的な探索でゲームをクリア（同調率100%未満）。 |
| **放棄（Abandonment）エンディング** | 悲劇的 | ダイアナとの同調率が低い（40%未満）、および複数の護衛シークエンスに失敗。 |
| **真（True）エンディング** | 勝利・大団円 | **同調率100%達成**、全20個のメモリフラグメント収集、および3つの特定サイドミッション完了。 |

「バッド」エンディングや「放棄」エンディングを避けるためには、戦闘中にダイアナの安全を最優先にする必要があります。彼女が過度なダメージを受けたり、重要な環境パズルで彼女のハッキング能力を活用できなかったりすると、同調率が低下し、二人の旅は冷酷な結末を迎えることになります。

## PRAGMATA 全エンディング回収：真エンディングへの道

真エンディングの解放は、多くのプレイヤーにとっての最終目標です。このルートでは、地球の状態や月面コロニーの真の正体について、最も多くのコンテキストが語られます。これを確実にするには、綿密な探索が必要です。

### 1. 同調率100%を達成する
ダイアナとの絆は「同調ゲージ」で管理されています。以下の行動で上昇させることができます：
*   セーフゾーンでの休憩中にダイアナと交流する。
*   戦闘で「デュアルテック（Dual-Tech）」フィニッシュを成功させる。
*   ダイアナがダメージを受けずに端末をハッキングできるように守る。
*   居住区の廃墟で見つかる「旧地球（Old Earth）」のアーティファクトを彼女に渡す。

### 2. 全20個のメモリフラグメントを収集する
これらのホログラフィック・データドライブは、ゲーム内で最も重要な収集アイテムです。多くの場合、垂直方向の移動パズルや無重力地帯の奥深くに隠されています。

> **💡 ヒント:** 多くのメモリフラグメントは、最初に訪れた時点では入手不可能です。**キネティック・グラップル**やアップグレードされた**エグゾスーツ・スラスター**を入手した後、高所のキャッシュに到達するために、以前のエリアに戻る必要があることがよくあります。

### 3. 主要なサイドミッションを完了する
真エンディングの「フラグ」となる特定のサイドミッションが3つあります：
*   **深宇宙中継器の調整（The Deep Space Relay Calibration）：** セクター4で発生。
*   **居住区廃墟の捜索（The Residential Ruins Search）：** 3つの暗号化ログを特定。
*   **セクター7兵器庫のハッキング（The Sector 7 Armory Hacking）：** 最大までアップグレードされたハッキングモジュールが必要。

## 重要な取り返しのつかない要素（ポイント・オブ・ノーリターン）

『PRAGMATA』はセミオープンワールドを採用していますが、物語が進むにつれて特定のセクターが永久にロックされます。これらのエリアでメモリフラグメントや暗号化ログを取り逃すと、その周回では真エンディングに到達できなくなります。

| チャプター / セクター | 進行トリガー | 進行後の影響 |
| :--- | :--- | :--- |
| **第3章：セクター4** | 通信中継器の起動 | 「旧地球」居住区の廃墟がロックされる。 |
| **第6章：セクター7** | リニア（Mag-Lev）への搭乗 | 初期の工業地帯への逆走ができなくなる。 |
| **第9章：セクター9** | **軌道エレベーターへの進入** | ゲーム全体の最終的なポイント・オブ・ノーリターン。 |

セクター9の軌道エレベーター制御ポッドを操作する前に、警告プロンプトが表示されます。これが、ファストトラベルを使って以前のセクターに戻り、取り逃したアイテムを回収する最後のチャンスです。

## 8つのスペシャル・エンディング・ピクチャーの解放

物語の結末に加えて、コンプリートを目指すプレイヤーはギャラリー内の8つのユニークな「エンディング・ピクチャー」を解放できます。これらは元々体験版/デモで導入されたものですが、2026年の製品版でも100%クリアの基準となっています。これらの写真はストーリーの選択ではなく、特定のプレイスタイルや実績を表しています。

| 写真番号 | 獲得条件 |
| :--- | :--- |
| **1** | **平和主義プレイ：** ダイアナのアビリティのみを使用（導入部以降、銃を使用しない）。 |
| **2** | **テクニカル・キル：** ラスボスのロボットを3種類の異なる武器タイプで倒す。 |
| **3** | **収集家：** 1回のプレイですべてのイエローキューブと大きな宝箱を回収する。 |
| **4** | **戦闘の達人：** 最終連戦をノーダメージでクリアする。 |
| **5** | **スピードランナー：** 最終章を10分以内にクリアする。 |
| **6** | **群衆制御：** スタティックガン（緑の銃）の1発で2体の大型ロボットをスタンさせる。 |
| **7** | **スカベンジャー：** 大量の資源を回収する（エリア全体の約80%以上）。 |
| **8** | **通常クリア：** 最終章を通常通りクリアする。 |

**写真 #2** の獲得には、特定のトリガーで苦労するプレイヤーが多いようです。最終決戦に向かうエレベーターの中で、ドローン1機とロボット2体と対峙します。「オレンジ・ロボット」を出現させ、写真を確実に解放するには、ドローンをピストルで倒し、1体目のロボットをダイアナのアビリティで倒し、2体目のロボットをスタティックガンの環境ダメージを利用して倒す必要があります。

## 装備と宇宙服の条件

ほとんどのエンディングはストーリー主導ですが、装備を整えることで攻略が大幅に楽になります。特定のスーツには、ノーダメージクリアなどの難しい写真条件を助けるパッシブボーナスがあります。

*   **標準EVAスーツ:** ヒューの初期スーツ。ボーナスなし。
*   **ルナ・スカベンジャー:** 15個のサルベージ契約を完了すると解放。**インベントリ容量が20%増加**し、「収集家」の実績が容易になる。
*   **スターライト・ドレス（ダイアナ）:** 全20個のメモリフラグメントを見つけると解放。**同調ゲージの回復速度が10%向上**する。
*   **コンプリート・スーツ:** 真エンディング達成後に解放。戦闘時以外で**スタミナが無限**になる。

## 「バッド」エンディングのトラブルシューティング

もし「放棄」エンディングのループから抜け出せない場合、通常は同調率が不足していることが原因です。『PRAGMATA』では、第3章、第6章、第9章で発生する「ストーリーフラグ」に基づいてダイアナとの関係性が計算されます。急いで進めすぎてしまった場合は、以前のセーブデータからやり直すか、強くてニューゲーム（New Game Plus）に入って物語の軌道を修正する必要があります。

> **⚠️ 警告:** RE Engineのセーブデータを手動で書き換えて同調率を「強制」しようとすると、ファイルが破損する可能性が高いです。取り逃した目標を修正するには、一度クリアした後に解放される「チャプターセレクト」機能を使用することを強くお勧めします。

## よくある質問（FAQ）

**Q: 1周目のプレイで真エンディングを見ることはできますか？**

A: はい、1周目でも真エンディングに到達することは可能ですが、非常に徹底した探索が必要です。全20個のメモリフラグメントを収集し、第9章で軌道エレベーターに乗る前にダイアナとの同調率を100%にする必要があります。

**Q: 初期のセクターでメモリフラグメントを取り逃した場合はどうなりますか？**

A: 第9章のポイント・オブ・ノーリターンを越えていなければ、ファストトラベルで以前のセクターに戻ることができます。ただし、セクター4の一部エリアは第3章以降ロックされます。そこを取り逃した場合は、100%コンプリートのために周回プレイ（New Game Plus）で回収する必要があります。

**Q: 難易度設定はエンディングの種類に影響しますか？**

A: いいえ、どの難易度設定でもすべての物語エンディングを解放できます。ただし、「N7-ヴァンガード・スーツ」などの特定の外見アイテムは、難易度「ハードコア」でのクリアが必要です。

**Q: エンディングは全部でいくつありますか？**

A: 主要な物語の結末が3つ（デフォルト、放棄、真）あり、それに加えてコンプリート要素として8つのギャラリー用写真（ミニエンディング）が存在します。**PRAGMATAの全エンディングを極める**には、3つのストーリー結末と8つのギャラリー画像のすべてを解放することを目指しましょう。
