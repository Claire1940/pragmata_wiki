## クレイドル：一元化された成長とアップグレード

『Pragmata』におけるすべてのキャラクター成長は、「クレイドル（またはシェルター）」と呼ばれる拠点で行われます。この安息の地は、ヒューとダイアナが休息し、交流し、そして最も重要な「アップグレード端末」にアクセスするための作戦基地として機能します。成長とは単にダメージ数値を上げることではなく、環境や敵との関わり方を変えるユーティリティスキルの解放も含まれます。

アップグレードシステムは、ヒューの身体的戦闘能力、武器のMODスロット、そしてダイアナのハッキング効率に焦点を当てた、いくつかのカテゴリに分かれています。これらの強化を解放するには、ミッション中に発見したり、激しいボス戦の報酬として得られる素材やデータフラグメントを収集する必要があります。

| アップグレードカテゴリ | 重点分野 | 主なメリット |
| :--- | :--- | :--- |
| **戦闘機動** | ヒューの移動 | 回避や時間の流れを遅くするカウンターを解放。 |
| **システムエンジニアリング** | 武器の効率性 | MODスロットの増加と弾薬補充率の向上。 |
| **ハッキングプロトコル** | ダイアナのアビリティ | ハッキンググリッドとオーバードライブゲージの改善。 |
| **サバイバルロジスティクス** | 体力と回復 | 回復アイテムの効果を高める。 |

## 戦闘を生き抜くためのヒューの必須アップグレード

ヒューはエンジニアであり、彼のスキルツリーは戦闘に対する実践的なアプローチを反映しています。従来のシューティングゲームのように武器のダメージだけに集中するのではなく、この**Pragmataスキルツリーガイド**では機動力とユーティリティを優先することを推奨します。弾薬は乏しく、マガジンを使い切ると武器が壊れてしまうことが多いため、ヒュー自身の能力こそが唯一の頼みの綱となります。

### ジャスト回避（ラストセカンド・ドッジ）
ゲーム序盤で最も重要なアップグレードの一つが、時間の流れを遅くする回避です。正確なタイミングで発動すれば、ヒューは飛んでくる弾丸や近接攻撃をかわしながら、周囲の世界を一時的にスローダウンさせることができます。これにより、位置を立て直したり、敵の露出したコアに精密なショットを叩き込んだりするための重要な隙を生み出すことができます。

### MODスロット拡張
ヒューの武器は、属性効果やステータスブーストを付与する様々なMODでカスタマイズ可能です。最初はスロットへのアクセスが制限されています。早めに「MODスロット拡張」ノードに投資することで、メインのピストルに複数のメリットを重複させることができ、ショックウェーブガンやスタシスネットなどのより重く特殊な武器が壊れた際の、頼れるバックアップになります。

> **💡 ヒント:** 長期戦での武器破損の頻度を減らすため、少なくとも1つのMODスロットは常に「弾薬効率（Ammo Efficiency）」に割り当てておきましょう。

## ダイアナのハッキングとオーバードライブの成長

ダイアナは単なる護衛対象ではありません。彼女は装甲を持つ敵を倒すための鍵となります。彼女の「ハッキングミニゲーム」では、グラフグリッド上のデジタル迷路をナビゲートします。彼女の側のスキルツリーをアップグレードすることで、このグリッドの機能や、戦闘中に彼女が提供できるサポートの質が変化します。

### ハッキンググリッドの最適化
ダイアナがハッキングを開始すると、ノードのグリッドが表示されます。デフォルトでは、単に緑色の出口ノードに到達するだけです。しかし、この**Pragmataスキルツリーガイド**の推奨に従って進めると、グリッド上の「ボーナスノード」を視認し、干渉できる能力を解放できます。

| ノードの色 | 効果 | アップグレード要件 |
| :--- | :--- | :--- |
| **青色ノード** | 接続のための基本パスノード。 | なし（基本能力） |
| **黄色ノード** | 敵の防御力を大幅に低下させる。 | ハッキングアップグレード Tier 1 |
| **赤色ノード** | ハッキングをリセットするトラップノード。 | 回避（プレイヤースキル依存） |
| **緑色ノード** | ハッキングの完了地点。 | なし（基本能力） |

### オーバードライブ・プロトコル
オーバードライブ・プロトコルはダイアナの「アルティメット」アビリティです。敵のハッキングに成功するたびにゲージが溜まります。満タンになると、プレイヤーは広範囲にパルスを放ち（R3 + L3）、周囲のすべての敵をスタンさせ、弱点を露出させることができます。スキルツリーでこのゲージをアップグレードすると、持続時間が増加し、発動に必要な「ハッキングデータ」の量が減少します。

## 武器の希少性と戦略

『Pragmata』のユニークな仕組みとして、ショックウェーブガンのような特殊武器は使い捨てであるという点があります。弾薬を使い切ると武器は壊れるため、戦場で新しい武器を見つけなければなりません。このため、時間経過でリチャージされる唯一のツールである「プライマリウェポン（ヒューのピストル）」のアップグレードがさらに重要になります。

最近の[GameSpotによるハンズオンプレビュー](https://www.gamespot.com/articles/pragmata-hands-on-preview-more-than-a-throwback-shooter/1100-6538761/)によると、この希少性の管理こそがゲームプレイの核心です。限られた特殊弾薬を浪費する前に、ダイアナを使ってハッキングし、敵を「無防備」にする必要があります。

### 序盤のおすすめビルド順序

2026年に旅を始めたばかりのプレイヤーには、最初の数回のクレイドル訪問で以下の優先順位を推奨します：

1.  **タクティカル回避 (ヒュー):** 高ダメージのAI攻撃を避けるために不可欠。
2.  **ハッキング Tier 1 (ダイアナ):** 黄色ノードを解放し、装甲の厚い敵を弱体化させる。
3.  **回復効率 (ヒュー):** 限られた回復アイテムの効果を最大化する。
4.  **オーバードライブチャージ率 (ダイアナ):** より頻繁なクラウドコントロールを可能にする。
5.  **ピストルリチャージ速度 (ヒュー):** 重火器がない時の隙を最小限に抑える。

## 高度な戦術：ヒューとダイアナのシナジー

この**Pragmataスキルツリーガイド**の真の価値はシナジーにあります。例えば、ダイアナがフィールド上の「ボーナスノード」を拾う能力をアップグレードすると、それらのノードがハッキングミニゲーム内に出現するようになります。現実世界で黄色いスクエアを拾っておけば、ハッキング中にデジタルグリッド上でその上を通ることで、敵の装甲を大幅にデバフすることができます。

> **警告:** ハッキンググリッドの操作中は注意が必要です。敵の攻撃によってダイアナの照準がターゲットから外れると、接続が失われます。プレッシャー下でもロックオンを維持できるよう、「安定性（Stability）」アップグレードを優先しましょう。

## FAQ

**Q: Pragmataのスキルツリーのポイントはどうやって獲得しますか？**

A: スキルポイントとアップグレード素材は、主に敵対的AIの撃破、ストーリー目標の達成、月面環境に隠されたデータキャッシュの発見によって獲得できます。ダイアナで難易度の高いハッキング迷路をクリアすることでも、彼女専用のアップグレード用データが追加で得られます。

**Q: スキルを振り直す（リスペック）ことはできますか？**

A: 2026年現在のビルドではかなりのカスタマイズが可能ですが、リスペックの機会はストーリーの特定の区切りに限定されています。序盤はニッチな武器MODに手を出す前に、機動力とハッキング効率に焦点を当ててビルドを計画するのが最善です。

**Q: ダイアナには個別の体力バーがあり、アップグレードできますか？**

A: ダイアナはヒューのシステムの一部として機能します。彼女が「ダウン」したり無力化されたりすると、ヒューが守ったり「修理」したりする必要がありますが、彼女の耐久性は主にヒューの防御アップグレードと、ハッキング系統のツリーにある彼女自身の「シールドプロトコル」スキルに依存します。

**Q: ボス戦で最も重要なスキルは何ですか？**

A: ボス戦では「オーバードライブ・プロトコル」の持続時間アップグレードが間違いなく最も重要です。ボスは複数の装甲レイヤーを持っていることが多く、これらはダイアナのハッキングでしかバイパスできません。ボスをスタンさせ、すべての弱点を同時に露出させることが、高火力武器を効果的に叩き込める唯一のチャンスとなります。
