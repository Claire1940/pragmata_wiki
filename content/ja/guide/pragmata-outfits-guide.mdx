カスタマイズは月面サバイバル体験の核となる要素です。この**PRAGMATA衣装ガイド**は、月面のデジタルの廃墟を探索するキャラクターたちが最高の姿でいられるようサポートします。重量級のエクソレイヤー・パイロットであるヒューとしてプレイする場合でも、謎めいた少女ダイアナとしてプレイする場合でも、本作にはレトロフューチャーな宇宙服から洗練されたハイテク戦術ギアまで、幅広い外見の選択肢が用意されています。このガイドでは、2026年版の最新情報に基づき、ワードローブを埋めるために必要なキャビン・スタンプ・システム、クリア後のチャレンジ、および特別ミッションの報酬について詳しく解説します。

『PRAGMATA（プラグマタ）』において、衣装は単なる見た目の変化ではありません。それはゲームの5つのセクターの進行状況や、シミュレーション内での過酷な戦闘を克服した証でもあります。ほとんどの報酬はシェルター・レベルと「キャビンコイン」の蓄積に関連しているため、最も希少な「ブラックアウト」報酬を解除したい場合は、サイドコンテンツへの参加が不可欠です。以下の戦略に従って、2人のギアをすべて手に入れましょう。

## キャビン・スタンプカード・システムの解説

新しい衣装を入手する主な方法は「キャビン・スタンプカード」システムです。メインストーリーを進めることでシェルターのレベルが上がります。レベルが上がるごとに新しいスタンプカードが利用可能になり、合計で5枚のカードが存在します。スタンプを獲得するには、主に「トレーニング・シミュレーター」のミッションを完了することで集められる「キャビンコイン」を消費する必要があります。

カードの全スロットを埋めるよりも多くのコインが入手可能ですが、最も豪華な報酬を受け取るためには、カードのすべてのスロットを埋める「ブラックアウト」を優先すべきです。

| シェルター・レベル | 解放条件 | 主な報酬タイプ |
| :--- | :--- | :--- |
| **レベル 1** | セクター1完了 | 初期エクソレイヤーのバリエーション |
| **レベル 2** | セクター2完了 | Type B タクティカルスーツ |
| **レベル 3** | セクター3完了 | レトロおよびステルスギア |
| **レベル 4** | セクター4完了 | 終盤のホワイト/ルナセット |
| **レベル 5** | 最初の「未知の信号」をクリア | シャドウおよび「ブラックアウト」限定衣装 |

> **💡 ヒント:** キャビンコインを溜め込みすぎる必要はありません。トレーニング・シミュレーターは繰り返しプレイ可能なので、ブラックアウト報酬のためにスタンプが数個足りない場合は、いつでもコインを稼ぐことができます。

## ヒューの解放衣装

ヒューのワードローブは、重厚なインダストリアル・デザインと戦術的なエクソレイヤー・デザインが中心です。初期装備は標準的な**デリスーツ**ですが、後半の解放衣装は「レギオン」や「アストロクラシック」といったテーマに寄っていきます。多くのプレイヤーは、1960年代のパルプSF風のデザインであるアストロクラシックスーツを、その独特な外見からベストチョイスとして挙げています。

| 衣装名 | 解放方法 | スタイルの特徴 |
| :--- | :--- | :--- |
| **エクソレイヤー** | スタンプカード1（個別枠） | 標準的な装甲プレート |
| **エクソレイヤー Type B** | スタンプカード2（個別枠） | ダークカラーの強化版 |
| **レギオン・ゼロ** | スタンプカード2（ブラックアウト報酬） | 洗練された黒のタクティカルアーマー |
| **アストロクラシック** | スタンプカード3（個別枠） | レトロフューチャーなドーム型ヘルメット |
| **レギオン・ゼロ Type B** | スタンプカード3（ブラックアウト報酬） | ハイコントラストなバリエーション |
| **シャドウ・ウォーカー** | スタンプカード5（個別枠） | 暗く「不気味な」デザイン |
| **アストロクラス Type B** | ラスボス撃破 | 鮮やかなレッドのカラーリング |
| **オブシディア** | 「未知の信号」ミッション完了 | プレミアムな黒曜石マット仕上げ |

**オブシディア**スーツは、非常に難易度の高いクリア後ミッションを完了する必要があるため、ヒュー使いの間では究極のステータスシンボルと見なされています。2026年現在のエンドゲームにおける激しい戦闘の中でも、ひときわ目立つ独特のシルエットを提供します。

## ダイアナの解放衣装

ダイアナの衣装は月面の環境を反映したものが多く、「ルナ（Lunar）」の名を冠した、ヒューの重厚なアーマーに比べて軽快で機敏なデザインが特徴です。彼女の旅は象徴的な**ブルークレイドルジャケット**から始まりますが、後半に解放される「スターリリィ」などは彼女の印象を劇的に変化させます。

| 衣装名 | 解放方法 | スタイルの特徴 |
| :--- | :--- | :--- |
| **ルナ・アクア** | スタンプカード1（ブラックアウト報酬） | ティールカラーのアクセントと独特のヘッドギア |
| **ルナ・ブラック** | スタンプカード3（個別枠） | 印象的な暗い月の美学 |
| **ルナ・ホワイト** | スタンプカード4（ブラックアウト報酬） | 清潔感のある「スパイ・ムーン」風 |
| **ハートビート** | スタンプカード5（最初のビンゴ） | 医療機器をイメージしたテクニカルスーツ |
| **シャドウ・エグゼキューター** | スタンプカード5（個別枠） | ダークでエッジの効いたデザイン |
| **スターリリィ** | スタンプカード5（ブラックアウト報酬） | エレガントなデザインとショートヘア |
| **ピンクポップ** | ラスボス撃破 | 明るくキュートなカジュアルウェア |
| **ブラックネオン** | 「未知の信号」ミッション完了 | 暗い生地に光るアクセント |

> **⚠️ 注意:** スターリリィなどの一部の衣装は、ダイアナの髪型を変更します。デフォルトのロングヘアを好む場合は、ルナ・ホワイトやルナ・ブラックのバリエーションを選ぶと良いでしょう。

## クリア後および特別報酬

スタッフロールが流れた後も、この**PRAGMATA衣装ガイド**は終わりません。「未知の信号」ミッションに関わることでしか入手できないハイティアなセットがいくつか存在します。これらはゲームのメカニクスを完全にマスターしているかを試す、特化したポストゲーム・コンテンツです。

シェルター・レベルを5に上げ、最後のスタンプカードを解放するには、ラスボス撃破後に少なくとも1つの「未知の信号」ミッションをクリアする必要があります。これらのミッションをすべてクリアすると、ヒューの**オブシディア**とダイアナの**ブラックネオン**衣装が与えられます。これらはゲーム内で最も視覚的に印象的なセットとされており、周囲の環境に反応するリアクティブ・テクスチャを備えています。

## 予約特典とバンドル限定衣装

デラックスエディションを購入したか、ゲームを予約注文していた場合、侍や忍者にインスパイアされた衣装にアクセスできる可能性があります。これらはゲーム内の進行システムとは別のもので、通常はゲーム開始時からシェルターのカスタマイズメニューで利用可能です。

*   **侍ヒュー:** 宇宙旅行用にアレンジされた伝統的な甲冑。
*   **忍者ダイアナ:** 日本の伝統的なモチーフを取り入れた、洗練されたステルス重視のスーツ。

さらに、発売と同時にリリースされたスペシャルバンドルには、各キャラクターに2つずつの追加衣装が含まれています。これらはゲームプレイ上の利点はありませんが、標準的な月面基地のスタイルとは異なる多様性を求めるプレイヤーに最適です。

## 衣装の変更方法

『PRAGMATA』での外見の変更はシェルター内で行います。スタンプカードやミッション報酬で衣装を解放したら、以下の手順に従ってください：

1.  **シェルター**（拠点）に戻ります。
2.  トレーニング・シミュレーターの入り口近くにある**カスタマイズ端末**へ向かいます。
3.  編集したいキャラクター（ヒューまたはダイアナ）を選択します。
4.  解放済みのカテゴリーをブラウズし、選択を決定します。

一部の衣装はフルセットですが、他の衣装ではヘッドギアなどの細かな調整が可能な場合があります。月面の旅でお気に入りのスタイルを見つけるために、さまざまな組み合わせを試してみてください。

## FAQ（よくある質問）

**Q: 難易度によってロックされている衣装はありますか？**

A: ほとんどの衣装はスタンプカードとミッションの完了に紐付けられています。ただし、難易度「Lunatic」でゲームをクリアすると隠しコスメティック報酬があるという噂がありますが、このガイドにある主要なセットはどの難易度でも入手可能です。

**Q: 衣装によってステータスボーナスやゲームプレイ上の利点はありますか？**

A: いいえ、『PRAGMATA』の衣装は純粋に見た目だけのものです。体力、防御力、攻撃力が変わることはありません。戦闘能力は、服装ではなく装備のアップグレードやスキルツリーの進行によって決定されます。

**Q: 1回のプレイで取り逃してしまう衣装はありますか？**

A: 永久に取り逃す衣装はありません。クリア後もキャビンコインを稼ぎ、スタンプカードを完成させることができます。ボス関連の報酬も、クリア後に再度挑戦することで入手可能です。

**Q: キャビンコインを早く集める方法は？**

A: 最も効率的な稼ぎ方は、高ランクのトレーニング・シミュレーター・ミッションを繰り返すことです。短時間で「S」ランクをクリアできるミッションに集中し、時間あたりのコイン獲得量を最大化しましょう。
