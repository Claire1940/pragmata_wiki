カプコンの『PRAGMATA（プラグマタ）』は、主人公ヒュー・ウィリアムズと謎のアンドロイドの少女ダイアナとの緻密なシナジーにより、SFアクションジャンルを再定義しました。この**プラグマタ アビリティガイド**をマスターすることは、月面を探索し、「クレイドル」内での命懸けの遭遇を生き抜くために不可欠です。2026年のリリースからの新規プレイヤーであれ、熟練のアクションファンであれ、これら2人のキャラクターがいかに相互作用するかを理解することが勝利への最大の鍵となります。

この**プラグマタ アビリティガイド**では、ヒューの標準的な弾道兵器から、戦況を一変させるダイアナのハイテクなハッキングプロトコルまで、あらゆるツールを解説します。生の火力と戦術的な電子戦のバランスを取ることで、プレイヤーは装甲を纏った「イーター」の弱点を露呈させ、戦場の主導権を握ることができます。以下では、武器管理、ハッキングミニゲーム、そして壊滅的な威力を持つオーバードライブ・プロトコルの詳細について探っていきます。

## ヒュー・ウィリアムズ：弾道兵器と戦闘の基本

ヒューはこのコンビにおける主要なダメージディーラーです。従来のシューティングゲームとは異なり、『プラグマタ』では特殊武器に「フラジリティ（脆さ）」システムを採用しているため、エイムと同じくらいリソース管理が重要になります。ヒューのメインウェポンは多機能なピストルで、弾薬は無限ですが、連続して発射するとエネルギーセルを補充するために短いクールダウン期間が必要になります。

標準的なピストル以外にも、ヒューは環境からより強力なツールを回収できます。これらの特殊武器は高い実用性を備えていますが、大きな注意点があります。弾薬が尽きると武器は壊れてしまい、戦場で代わりの武器を見つけるまで再使用できません。

| 武器名 | タイプ | 主な効果 | 弾薬ロジック |
| :--- | :--- | :--- | :--- |
| **プライマリ・ピストル** | 弾道 | 安定した中距離ダメージ。 | **無限**（時間経過で補充）。 |
| **ショックウェーブ・ガン** | 重火器 | 高ダメージ、敵をよろめかせる。 | **制限あり**（弾切れで武器が破壊）。 |
| **ステイシス・ネット** | タクティカル | 敵をスタンさせ、凍結させる。 | **制限あり**（非致死性の補助ツール）。 |

> **💡 ヒント:** 特殊武器の残弾数には常に注意を払いましょう。ショックウェーブ・ガンが壊れそうなときは、床にある「光」を探してください。それは近くに交換用の武器があることを示しています。

## ダイアナのハッキングプロトコル：プラグマタ アビリティガイド

ヒューが物理的な脅威に対処する一方で、ダイアナは敵の防御をバイパスするために必要な技術的サポートを提供します。「ウォーカー」や「セクターガード」などの多くのアドバンスド・エネミーは、標準的な弾道ダメージを無効化する重装甲を持っています。これらを倒すには、ダイアナがハッキングを開始して敵を「開放」し、ヒューが攻撃できるようにコアを露出させる必要があります。

ハッキングプロセスは、ダイアナがターゲットへの照準を維持している間、プレイヤーがグリッドを操作するミニゲーム形式になっています。ダイアナの視線が遮られたり、攻撃によって中断されたりすると、ハッキングの進行が停滞したりリセットされたりすることがあります。

### ハッキング・グリッド・ノード

ハッキング中、さまざまな色の正方形が表示されたグラフグリッドが現れます。目標は、開始地点から緑色の出口ノードまでの経路を構築することです。

| ノードの色 | 戦闘への影響 | 戦術的優先度 |
| :--- | :--- | :--- |
| **青ノード** | 標準的な接続ポイント。 | 出口へのパスを作るために必要。 |
| **黄ノード** | **防御デバフ**: 敵の装甲を弱体化。 | 高。ここを通ることでヒューのDPSが向上。 |
| **赤ノード** | **ハザード**: システムロックアウトを誘発。 | ハッキング失敗を防ぐため、絶対に避ける。 |
| **緑ノード** | **成功**: ハッキングを完了。 | すべてのハッキングシーケンスの最終目標。 |

## 高度なシナジー：オーバードライブ・プロトコルと弱点

ダイアナが敵のハッキングに成功すると、装甲プレートが移動または格納され、発光する青いコアが露出します。この時こそ、ヒューはショックウェーブ・ガンのような高火力武器に切り替え、ダメージを最大化すべき瞬間です。しかし、この**プラグマタ アビリティガイド**における最強のツールは**オーバードライブ・プロトコル**です。

オーバードライブ・プロトコルは、ダイアナがハッキングを成功させ、ヒューがダメージを与えることでチャージされる「アルティメット」アビリティです。ゲージが満タンになると、**L3 + R3**（コンソールの場合）を押すことで発動できます。

*   **広範囲露出:** オーバードライブは即座に周囲のすべての敵をスタンさせ、手動のハッキングミニゲームなしで弱点を露出させます。
*   **タイムディレーション:** 世界の流れがわずかに遅くなり、ヒューが複数のターゲットを迅速に仕留めることが可能になります。
*   **緊急回避:** 「イーター」に囲まれた際、オーバードライブは仕切り直すための完璧な「脱出カード」として機能します。

> **警告:** オーバードライブのチャージにはかなりの時間がかかります。一般的に、このアビリティは雑魚ドローンではなく、ボス戦や「ウォーカー」ユニットの大群のために温存しておくことが推奨されます。

## リソース管理と生存

『プラグマタ』での生存は、装備を維持する能力にかかっています。特殊武器は壊れやすいため、常にインベントリを回転させる必要があります。ショックウェーブ・ガンやステイシス・ネットがない場合は、ヒューの機動力とプライマリ・ピストルのリチャージ速度に頼らなければなりません。

1.  **絶えずスカベンジする:** クレートや端末ステーションを探しましょう。これらには「シュガーキューブ」（回復アイテム）や弾薬の補充が入っていることが多いです。
2.  **ジャスト回避:** 敵の攻撃の直前に回避をタイミングよく行うと、短いスローモーション効果が発生し、位置調整やハッキング開始の隙が生まれます。
3.  **脱出ハッチ:** ヒューの体力がゼロになった場合、ダイアナは彼を近くの脱出ハッチまでテレポートさせようとします。これはチェックポイントシステムとして機能しますが、ダイアナがクイックハックでハッチを「解錠」するために数秒間静止する必要があります。

## ボス戦タクティカルガイド：「イーター」

巨大な戦車のような「セクターガード」などの『プラグマタ』のボスには、マルチステージのアプローチが必要です。力押しだけでこれらの遭遇を突破することはできません。

*   **ステージ1：観察**。ボスの初期攻撃を回避します。体にある青いノードを探してください。これらがダイアナがターゲットにする必要があるポイントです。
*   **ステージ2：ハッキング**。ヒューがヘイトを買っている間に、ダイアナは複雑なグリッドハックを完了させる必要があります。このフェーズでは、ヒューはステイシス・ネットを使用してボスを静止させるべきです。
*   **ステージ3：コア**。ハッキングが完了すると、ボスは通常崩れ落ちるか、パイロット/コアを露出させます。この時だけが、大ダメージを与えるチャンスです。ここでショックウェーブ・ガンやオーバードライブ・プロトコルを使用して、一気に決着をつけましょう。

| ボスのギミック | 対抗アビリティ | 結果 |
| :--- | :--- | :--- |
| 強固な前面装甲 | ダイアナのハッキンググリッド | 装甲プレートが格納される。 |
| 高速突進攻撃 | ヒューのステイシス・ネット | ボスがその場で凍結する。 |
| 範囲攻撃（スラム） | ジャスト回避 | スローモーションの窓が発生。 |
| シールド生成 | 黄ノードのハック | シールドの耐久値が低下。 |

## FAQ

**Q: 初心者にとって、プラグマタ アビリティガイドで最も重要なことは何ですか？**

A: 最も重要なのは、ヒューとダイアナのシナジーを理解することです。ダイアナがハッキングを完了するまで、ヒューは装甲した敵に大きなダメージを与えることができません。ダイアナがハッキンググリッドを操作している間、彼女を守ることに集中してください。

**Q: ショックウェーブ・ガンのような武器の弾薬が永久になることはありますか？**

A: いいえ。『プラグマタ』では、特殊武器は一時的なパワーアップとして設計されています。弾薬を慎重に管理し、壊れたらすぐにプライマリ・ピストルに切り替える準備をしておく必要があります。

**Q: オーバードライブ・プロトコルを早くチャージするにはどうすればいいですか？**

A: ハッキングミニゲーム中にオプションの「ボーナスノード」（黄色の正方形）を通過したり、戦闘中にジャスト回避を成功させたりすることで、チャージ速度を上げることができます。

**Q: ハッキング中にダイアナが倒されることはありますか？**

A: ダイアナはダメージを受けたり中断されたりすることがあり、それによってハッキングは失敗します。彼女の体力がゼロになると、彼女を蘇生させるかチェックポイントに到達するまで、すべてのハッキングおよびオーバードライブアビリティが使用不可になります。
