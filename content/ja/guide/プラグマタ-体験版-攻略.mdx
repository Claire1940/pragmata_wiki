## 基本メカニクスと操作方法

『プラグマタ』は、伝統的な三人称視点シューティング（TPS）の要素と、高い機動性を備えたプラットフォームアクションを融合させています。ハイテクEVAスーツを着用したエンジニアのヒューは、内部エネルギーを消費して「スペースブースト」やエアダッシュを行います。ダイアナと連携しながら、このエネルギーをいかに管理するかが攻略の第一歩です。

| アクション | 操作 (PS5/PC) | 効果 |
| :--- | :--- | :--- |
| **環境スキャン** | L1 / Q | インタラクティブな端末、アイテム、敵の弱点をハイライトする。 |
| **シフト / 回避** | RB / R1 | 素早いダッシュ。タイミングよく回避するとスローモーションの「ウィッチタイム」が発動。 |
| **エアダッシュ** | A + A / X + X | ジャンプ距離を伸ばし、垂直方向の再配置を可能にする。 |
| **ハッキング指示** | X / □ | ダイアナに特定のノードや端末へのインターフェースを指示する。 |

> **💡 ヒント:** 常にスキャンを有効にしておきましょう。アイテムを見つけるだけでなく、セクターのロックダウンを解除するために必要な「EXEノード」を特定するためにも、**プラグマタ 体験版 攻略**においてスキャンは必須です。

## 戦闘システム：ハッキングと銃撃戦

『プラグマタ』の戦闘は多層的です。単に引き金を引くだけではなく、まずはダイアナのハッキング能力を使って敵の脆弱性を暴く必要があります。ダイアナでアーマーを無力化し、ヒューの重火器で追撃するのが基本のループです。

### ハッキング・ミニゲーム
ダイアナが端末や大型の敵にインターフェースすると、ホログラフィックパネルが表示されます。スタート地点から緑色の「終了（End）」ノードまでパルスを誘導してください。
1.  **装甲剥離:** 緑色のノードに到達すると、敵のシールドが解除されます。
2.  **時間延長:** ハッキング中に青色のノードを通過すると、敵が脆弱な状態になる時間が延長されます。
3.  **弱点ハイライト:** ハッキング完了後、敵のコアが黄色く光ります。ここを狙うことで大ダメージを与えられます。

### ハッキング・ノードの種類
大規模な戦闘中、ダイアナは周囲の環境ノードに接続し、バフやデバフを付与できます。

*   **情報攪乱:** 一定時間、敵が受ける全ダメージを増加させます。
*   **マルチターゲット・リンク:** 近くの敵同士を鎖で繋ぎ、一体に与えたダメージをグループ全体に伝播させます。
*   **オーバードライブ・プロトコル:** ダイアナのアルティメットアビリティ。ゲージが満タンになると、周囲の全脅威を無力化し、飛来する弾丸を空中で停止させます。

## 武器一覧と戦略

2026年の体験版では、4つの異なる武器カテゴリーを使用できます。強力な「アタックユニット」は環境内で見つかる回数限定アイテムであるため、弾薬の管理が極めて重要です。

| 武器タイプ | 主な用途 | リロード / 使用 |
| :--- | :--- | :--- |
| **グリップガン** | 標準的な戦闘。中距離で安定したDPSを発揮。 | **弾数無限** |
| **ショットガン** | 「ラット・イン・ザ・メイズ」ボットに対する高い瞬間火力。 | **手動リロード** |
| **ショックウェーブガン** | クラウドコントロール。小型ユニットの群れを吹き飛ばす。 | **手動リロード** |
| **チャージピアサー** | 遠距離用レールキャノン。厚い装甲板を貫通する。 | **チャージショット** |

**隠し武器のヒント:** **チャージピアサー**は体験版の隠れたMVPです。最上階へのメインリフトに到達する前に、サイドにあるメンテナンス通路を探索して「パーツカートリッジホルダー」を見つけてください。この武器があれば、ボスのパーツを安全な距離から狙撃でき、最終決戦が大幅に楽になります。

## ソラパネル工場 ウォークスルー

体験版のミッションでは、施設の頂上に到達してメイン電力ターミナルを再起動することが目的です。道中は大きく3つのフェーズに分かれています。

### 1. エントランスホール
最初の目的は、3つのEXEノードを見つけることです。スキャン（L1/Q）を使用して、クレートや上部通路の背後に隠されたノードを探してください。3つすべてを起動するとロックダウンが解除されます。天井から降ってくる「ワーカー」に注意し、ダイアナが端末を操作している間はグリップガンで敵を遠ざけましょう。

### 2. メンテナンス通路と垂直アクション
このセクションではプラットフォームアクションが試されます。エアダッシュを使って溝を飛び越えますが、床の「スタシスネット（停滞ネット）」には注意してください。このトラップにかかるとヒューはその場に拘束され、スナイパーボットの餌食になってしまいます。
*   **リフト・シーケンス:** ダイアナがリフトをハッキングして上昇させている間、押し寄せるドローンの波からプラットフォームを守る必要があります。ショックウェーブガンを使えば、精密なエイムを必要とせずにドローンをプラットフォームから叩き落とせます。

> **⚠️ 警告:** リフトの上昇中にプラットフォームから落下すると、直前のエスケープハッチからやり直しになります。足元に注意し、ドローンの排除に集中しましょう。

### 3. エスケープハッチとシェルター
工場内の各所に**エスケープハッチ**があります。これらはチェックポイントとして機能します。ハッチを開けると「シェルター」に戻ることができ、集めた鉱石を使ってヒューのスーツをアップグレードしたり、体力を回復したりできます。ハッチを見つけたら必ず起動しておきましょう。

## ボス攻略: EDIS (セクター防衛プロトコル)

体験版の最後は、高機動ガードボット「**EDIS**」とのバトルです。この戦いでは、これまでに学んだすべてのメカニクスを駆使する必要があります。

*   **フェーズ 1: ミサイル対処。** EDISは「ルナ」ミサイルを一斉射撃してきます。ダイアナが「オーバードライブ・プロトコル」について叫ぶのを待ちましょう。発動するとミサイルが停止し、それらをEDISに打ち返して大きなダウンを奪えます。
*   **フェーズ 2: 物理攻撃。** EDISが突進攻撃を仕掛けてきます。衝突直前にシフト（R1/RB）を使用してウィッチタイムを発動させましょう。時間がスローになっている間にショットガンやチャージピアサーに切り替え、胸部の光るコアを攻撃してください。
*   **ファイナルスタンド:** 体力が10%になると、EDISは「無力化モード」に入り、乱射してきます。残りのアタックユニットの弾薬を使い切り、素早く仕留めましょう。チャージピアサーのフルチャージショットを頭部に叩き込めば、即座に決着がつきます。

## FAQ

**Q: 体験版の異なる絵やエンディングを解放するにはどうすればいいですか？**

A: 最後にダイアナが描く絵は、クリアタイムと戦闘効率に紐付いています。全8バリエーションを見るには、じっくり探索するプレイから10分を切るスピードランまで、複数回クリアする必要があります。

**Q: EDIS戦の最中に回復はできますか？**

A: はい。アリーナの周囲にあるクレートの中に精製鉱石が入っています。ただし、回復アイテムの使用には数秒かかります。ウィッチタイム中や、ミサイルを跳ね返してEDISがダウンしている隙に回復するのがベストです。

**Q: 「マルチハック」能力は何をするものですか？**

A: マルチハックは解放可能なスキルで、ダイアナが複数の端末や敵に同時にリンクできるようになります。**プラグマタ 体験版 攻略**においては、リフトの移動速度を上げたり、リンクしたグループ全体でダメージを共有させてワーカーの群れを素早く殲滅したりするのに役立ちます。

**Q: チャージピアサーは永続的な武器ですか？**

A: 体験版では、チャージピアサーを入手するとショックウェーブガンのスロットが上書きされます。死亡した場合、入手後にチェックポイント（エスケープハッチ）に到達していなければ失われます。製品版では、武器のロードアウトは完全にカスタマイズ可能になる予定です。
