カプコンが贈る最新SFエピックの月面世界を冒険するには、単なる反射神経以上のものが必要です。人間の兵士ヒュー・ウィリアムズと、アンドロイドのパートナーであるダイアナとの相乗効果を深く理解することが求められます。この**PRAGMATAスキルガイド**は、ハッキングミニゲームの複雑な仕組みからハイテク兵器の戦術的な管理まで、「ザ・クレイドル（揺り籠）」の複雑なシステムをナビゲートするのに役立つよう設計されています。基本的な「ウォーカー」に立ち向かう時も、恐るべき「イーター」ガードプロトコルと対峙する時も、アビリティをマスターすることこそが太陽光発電所を生き延び、脱出のための最上階へ到達する唯一の道です。

ゲームの序盤を進めていくうちに、従来のシューティング要素は戦いの半分に過ぎないことに気づくでしょう。この**PRAGMATAスキルガイド**では、初心者の生存者と月面のベテランを分ける不可欠なメカニクスを詳しく解説します。ダイアナのサポート能力とヒューの重火器を活用することで、装甲化された脅威を解体し、環境を有利に利用することができます。2026年に待ち受ける試練に万全の体制で臨むため、以下の戦略に従ってください。

## 二人主人公の戦闘システム

PRAGMATAの戦闘は、ヒューの圧倒的な火力とダイアナのテクニカルなサポートが織りなすダンスのようなものです。ヒューは、機動力を高め重火器の扱いを可能にするエクススーツを着用し、直接的な交戦を担当します。しかし、多くの敵は高度な「イーター」装甲で守られており、標準的な弾薬では効果がありません。ここでダイアナの存在が不可欠になります。

ダイアナはあなたの「戦術的な目」として機能します。彼女は弱点を特定し、敵を「開放」するためのハッキングを開始して、ヒューの武器が核心部のプロセッサに届くようにします。従来のコンパニオンAIとは異なり、ダイアナのアクションはプレイヤーのポジショニングと照準に直接影響されます。ハッキング中に視線を失ったり、ターゲットから離れすぎたりすると、接続が切断され、プロセスを最初からやり直すことになります。

### ヒューの武器アーセナル

ヒューの武器は「プライマリ」と「スペシャル」の2つのタイプに分類されます。太陽光発電所で長期生存するためには、それぞれの弾薬管理を理解することが極めて重要です。

| 武器タイプ | 例 | 弾薬源 | 耐久度 |
| :--- | :--- | :--- | :--- |
| **プライマリ** | タクティカルピストル | 時間経過で自動チャージ | **無限** |
| **近距離** | ショックウェーブガン | 落ちているアイテム / 弾薬箱 | **空になると壊れる** |
| **クラウドコントロール** | ステイシスネット | 落ちているアイテム / 弾薬箱 | **空になると壊れる** |

> **⚠️ 警告:** ショックウェーブガンやステイシスネットなどのスペシャル武器は非常に強力ですが、壊れやすい性質を持っています。弾薬を使い切ると武器自体が壊れてしまうため、環境内で代わりの武器を見つける必要があります。

## ハッキングミニゲームのマスター

ハッキングはこの**PRAGMATAスキルガイド**における核心的なメカニクスです。ダイアナが敵をターゲットすると、ホログラフィックなグリッドが表示されます。敵の防御を突破するには、一連のノードを介してデータストリームを誘導する必要があります。このシステムはリズムパズルやクラシックなゲーム「Snake（スネーク）」に例えられることが多く、経路の選び方によってハッキングの成否と威力が決まります。

### グリッドノードの種類と効果

ハッキンググリッドには、進行を助けるノードや妨げるノードが配置されています。これらを正しくナビゲートできるかどうかが、素早い撃破か、あるいは危険な長期戦かの分かれ目となります。

| ノードの色 | 効果 | 優先度 |
| :--- | : :--- | :--- |
| **青いスクエア** | 標準データノード。ハッキングを進行させるために必要。 | **高** |
| **緑のスクエア** | 出口ノード。ここに到達するとハッキング完了。 | **必須** |
| **黄色のスクエア** | ボーナスノード。敵の防御力を大幅に低下させる。 | **任意/戦略的** |
| **赤いスクエア** | ハザードノード。ハッキング失敗や「ロックアウト」の原因となる。 | **回避** |

ダメージ出力を最大化するには、緑の出口ノードに到達する前に、できるだけ多くの**黄色いノード**を通過するようにしてください。ボーナスノードを多く集めるほど、敵が「開放」された際により脆弱になります。

## 高度なスキルとオーバードライブ・プロトコル

ハッキングと射撃の基本ループに慣れてくると、高度な操作が解放されます。最も重要な防御スキルの一つが**ジャスト回避（パーフェクトドッジ）**です。敵の攻撃が当たる直前にタイミングよく移動することで、ヒューは一時的な集中状態に入り、素早いカウンターショットや即座のダイアナによるハッキング開始が可能になります。

### オーバードライブ・プロトコル（究極アビリティ）

オーバードライブ・プロトコルはいわば「切り札」です。このゲージは、ダイアナがハッキングを成功させたり、ヒューが露出したコアにダメージを与えたりすることで蓄積されます。ゲージが満タンになったら、コントローラーの**L3 + R3**（または指定のキー）を押して発動できます。

*   **効果範囲:** オーバードライブは周囲のすべての敵をスタンさせる強力なパルスを放ちます。
*   **自動露出:** 爆発に巻き込まれた敵は即座に「開放」状態になり、ハッキングミニゲームをスキップできます。
*   **戦術的利用:** ボス戦や、高ティアのウォーカーに完全に囲まれた時のために温存しておきましょう。

> **💡 ヒント:** 「セクターガード」のボス戦でオーバードライブ・プロトコルを使用すると、メインタンクやパイロットを即座に露出させることができ、スペシャル武器の全弾を叩き込んで最大のDPSを出すことが可能です。

## 生存と探索の戦略

太陽光発電所や月面の「ザ・クレイドル」といったPRAGMATAの環境は、垂直性に富み、隠されたリソースで溢れています。探索は戦闘スキルと同じくらい重要です。

1.  **手動再起動:** 多くのエリアが緊急ロックダウン状態にあります。メインパワー端末を見つけ、ダイアナが手動再起動を行う間、彼女を守り抜く必要があります。
2.  **エスケープハッチ:** これらはチェックポイントとして機能します。戦闘で倒れた場合、最後に起動したハッチに転送されます。新しいセクターに移動する前に、必ずダイアナにハッチをハッキング（**L1長押し**）させてください。
3.  **ルート管理:** レベルのあちこちに散らばっている「特別なクリスマスプレゼント」やサプライクレートを探しましょう。これらにはダイアナ用の角砂糖や、ショックウェーブ、ステイシス武器に不可欠な弾薬が入っています。

## 敵の分析：イーターとの対峙

PRAGMATAにおける主な敵は、暴走したボットやセキュリティプロトコルの集合体である「イーター」です。小型で俊敏なウォーカーから、巨大な多層階建ての「ガンダムスタイル」ロボットまで多岐にわたります。

*   **ウォーカー:** 一般的な雑魚敵です。簡単なハッキングとプライマリピストル数発で倒せます。
*   **セクターガード:** これらのボスは複数段階のハッキングを必要とします。多くの場合シールドを持っており、ダイアナがハッキングを開始できるようになる前に、ショックウェーブガンでシールドを削る必要があります。
*   **未登録のゲスト:** これらの高速ユニットはダイアナを妨害しようとします。ステイシスネットを使用して足止めし、邪魔されずにハッキンググリッドに集中できるようにしましょう。

## 戦闘フローのまとめ

ゲームで優位に立つために、常に以下の優先順位で戦闘ループを回してください：
1.  **脅威の特定:** ヒューのスキャナーを使用して、最も強力な敵を見つける。
2.  **サポートの開始:** ダイアナがターゲットへの視線を確保できる位置に移動する。
3.  **パズルフェーズ:** ハッキンググリッドを完了させ、防御デバフのために黄色いノードを優先する。
4.  **実行フェーズ:** 高ダメージのスペシャル武器に切り替え、光るコアを狙い撃つ。
5.  **再配置:** プライマリ武器がリチャージされる間、ジャスト回避を使って反撃を避ける。

## FAQ

**Q: 入力した年齢はPRAGMATAのゲームプレイに影響しますか？**

A: いいえ、開始時に入力する年齢はデータ収集目的のものであり、難易度、ストーリー、戦闘メカニクスが変化することはありません。

**Q: ショックウェーブガンが壊れたらどうなりますか？**

A: 弾薬がなくなると武器は壊れ、インベントリから消滅します。その武器タイプを再び使用するには、環境内を探索して新しいショックウェーブガンのピックアップを見つける必要があります。

**Q: すべての青いノードを通らなくてもハッキングを完了できますか？**

A: はい、緑の出口ノードまでの経路を作成するだけで完了できます。ただし、より多くのノード（特に黄色いもの）を通ることで強力な戦闘ボーナスが得られ、その後の戦闘が非常に楽になります。

**Q: 赤いノードに引っかかった場合、ハッキングをリセットする方法はありますか？**

A: 赤いノードに入ったり集中を失ったりすると、ハッキングは失敗します。通常はすぐに再開できますが、一部の高度な敵は短期間ハッキングが無効になる「ロックアウト」期間を持つ場合があります。この**PRAGMATAスキルガイド**の経路ヒントを活用して、失敗を回避しましょう。
