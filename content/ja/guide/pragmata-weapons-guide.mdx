## メイン武器：ビルドの基礎

メイン武器は弾薬が時間経過でリチャージされるため、最も信頼できるツールです。重攻撃ユニットに比べると基本的なものに見えるかもしれませんが、ゲーム全体を通して使い続けることができます。一般的に、プレイスタイルに合わせて「グリップガン」か「パルスカービン」の2つの主な選択肢があります。

### グリップガン vs パルスカービン
グリップガンは万能なオールラウンダーとして機能します。確かな精度と、それなりのヘッドショット倍率を誇ります。一方、パルスカービンはフルオートの野獣であり、単発の威力よりもスピードと属性の蓄積を優先します。

| 武器 | タイプ | 主な利点 | 最適な用途 |
| :--- | :--- | :--- | :--- |
| **グリップガン** | セミオートピストル | 信頼性の高い精度と弾薬効率 | 一般的な遭遇戦とヘッドショット狙い |
| **パルスカービン** | フルオート | 標的への素早い**Heat（熱）**蓄積 | クリティカル重視のビルドと素早い敵 |

> **💡 ヒント:** クリティカルヒット中心のビルドを好むなら、敵のオーバーヒートゲージを非常に早く溜められるパルスカービンが優れた選択肢となります。

## 攻撃ユニット：重火器

攻撃ユニットは、大きな脅威を排除するために必要な瞬間火力とユーティリティを提供します。これらのユニットはチャージが必要だったり弾薬が限られていたりすることが多いため、配置とタイミングが重要になります。

### 遠距離およびエリア制圧
距離を保ちたいプレイヤーにとって、**チャージピアサー**はおそらくゲーム内で最強の単発武器です。3段階までチャージ可能なレールキャノンとして機能し、複数の敵を貫通して背後に隠された弱点を突くことができます。

小規模な「雑魚敵」の群れに圧倒されそうな時は、**ホーミングミサイル**が優れたクラウドコントロールを提供します。複数のターゲットをロックオンすることで、数秒で部屋を掃討できますが、主要なボスに対しては**フォトンレーザー**に比べて集中ダメージは劣ります。

| 攻撃ユニット | ダメージタイプ | 主な特徴 |
| :--- | :--- | :--- |
| **ショックウェーブガン** | ショットガン / 拡散 | 発射体数が多く、至近距離で壊滅的な威力を発揮 |
| **チャージピアサー** | レールガン / 貫通 | 最強の単発火力。敵を貫通してヒットする |
| **フォトンレーザー** | 継続ビーム | 最高の純粋DPS。ターゲットを拘束する必要がある |
| **ホーミングミサイル** | 爆発 / 範囲攻撃 | マルチロック可能。雑魚敵の掃討に最適 |
| **ジャックハンマー** | 近接 / パイルバンカー | クリア後武器。「オープン」状態を無視して大ダメージを与える |

## 戦術および防衛ユニット

セカンドユニットこそが、あなたの戦術的柔軟性を真に決定づけるものです。これらのツールを使用することで戦場を操作し、重火器が輝くための隙を作り出すことができます。

### 戦術的ユーティリティ
**ライオットブラスター**は、その怯ませ（スタガー）能力の高さからコミュニティで人気があります。うまく命中させればほとんどの標準的な敵をノックダウンでき、ショックウェーブガンで距離を詰めるチャンスが生まれます。あるいは、**ステイシスネット**はボス戦において、ターゲットを静止させてフォトンレーザーのバーストを当てるのに最適です。

*   **コードジェネレーター:** 敵のハッキングを妨害する「ブロッカー」ノードを無効化するために不可欠です。
*   **粘着爆弾:** ハッキングマトリックスを縮小させ、激しい銃撃戦の中でも素早いハッキング実行を可能にします。
*   **ハッキングマイン:** 近接トラップにハッキングノードを「プリロード」しておくことができます。

### 防衛オプション
PRAGMATAにおける防御は、単にブロックすることではなく、方向を逸らすことにあります。**ドローンハイヴ**は攻撃的な防衛として機能し、ハッキングに集中している間、敵を自動的にターゲットにするファンネルを展開します。

> **警告:** **インパクトバリア**は非常にシチュエーションを選びます。飛び道具や空中戦の敵からは守ってくれますが、自身の機動力も制限されます。主にミサイルの多い遭遇戦で使用してください。

## 高度なハッキングメカニクス

**PRAGMATA武器ガイド**の柱となるのがハッキングシステムです。敵をハッキングすると「オープン（Open）」状態になり、受けるダメージが大幅に増加し、さまざまなステータス効果が適用されます。

### 必須ハッキングノード
ノードはハッキング中に適用する特定の効果です。通常、デジタル攻撃をカスタマイズするために3つのスロットがあります。

1.  **デコード:** ターゲットに与えるすべてのダメージを最大50%増加させます（1.5倍の補正）。
2.  **混乱 (Confuse):** 「フリーズ」よりも優れた代替案で、約40秒間持続し、敵がプレイヤーを無視したり仲間を攻撃したりするように仕向けます。
3.  **露出 (Expose):** その後のハッキングで壊滅的なクリティカルダメージを発生させます。
4.  **チェイン:** 実行するたびに威力が増していく、連続した高速ハッキングを可能にします。

### ハッキングモードの比較
ハッキングモードは、「オープン」状態の挙動やダメージの計算方法を決定します。

| モード | フォーカス | メカニズム |
| :--- | :--- | :--- |
| **オフェンシブ** | バランス重視 | 追加ショットごとにハックダメージが約8%増加 |
| **ストライク** | 瞬間火力 | オフェンシブの累積ダメージの2倍。実行後にターゲットが閉じる |
| **ハイブリッド** | 継続 | 特定のノードをヒットすることでハッキングタイマーを延長できる |
| **コンバスト** | クリティカル | **Heat**蓄積を最大化。カービンビルドに必須 |
| **ブースト** | ハッキング特化 | アクティブなハックを大幅に強化。銃を撃たずにエリートを倒せる |

## 2026年版最適化ロードアウト

装備を最大限に活用するには、武器の属性特性とハッキングモードの相乗効果を狙うべきです。例えば、「熱（Heat）」ビルドでは、パルスカービン、Heatハッキングノード、そしてコンバストモードを利用して、高ダメージのクリティカル爆発を頻繁に発生させます。

より戦術的なアプローチとしては、遠距離キル用のチャージピアサー、クラウドコントロール用のホーミングミサイル、そして敵を近づかせないための混乱ノードを組み合わせた「レールハッカー」セットアップがあります。これにより、他の敵が混乱している間に価値の高いターゲットを仕留めることができます。

## よくある質問（FAQ）

**Q: PRAGMATA武器ガイドにおいて、初心者にとって最高の武器は何ですか？**

A: グリップガンとライオットブラスターの組み合わせが最も信頼できる開始時のコンボです。グリップガンは扱いやすい反動とリチャージ可能な弾薬を持ち、ライオットブラスターは近づきすぎた敵をノックダウンさせることで安全を確保してくれます。

**Q: ジャックハンマーはどうやってアンロックしますか？**

A: ジャックハンマーはクリア後の報酬です。これは「強くてニューゲーム（NG+）」用に設計されており、敵がオープン状態であるかどうかにかかわらず、強力な「パイルバンカー」ダメージを与えることができます。

**Q: クラウドコントロールには「フリーズ」と「混乱」のどちらが良いですか？**

A: 一般的には「混乱」の方が優れていると考えられています。フリーズは約8秒間敵をその場に固定しますが、混乱は40秒間持続し、敵のヘイト（敵対心）を自分から逸らしたり、敵のバフを奪ったりすることさえ可能です。

**Q: なぜハッキングマトリックスがリセットされるのですか？**

A: デフォルトでは、敵から直接攻撃を受けるとハッキングの進行状況がリセットされます。これを防ぐには、攻撃を受けてもハッキングを完了できる**ハイパーフォーカス**MODを見つけて装備する必要があります。
