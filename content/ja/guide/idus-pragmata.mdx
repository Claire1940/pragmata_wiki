## Idus Pragmataのコア・メカニクス

**Idus Pragmata**の基本的なループは、電力の復旧とセクターの移動を中心に展開します。従来のシューティングゲームとは異なり、プレイヤーの攻撃能力は、現在滞在しているセクターのエネルギー状態に直接紐付いていることが多いです。電力が低いときは防御の選択肢が限られるため、戦闘に対してより慎重なアプローチを強いられます。

### エネルギー閾値と戦闘効率
プライマリ電力を再接続する際、エネルギーの復旧率に応じて攻撃ポテンシャルが上昇することに気づくでしょう。エネルギーレベルが低いと「不正アクセス」として検知されやすく、セクターガード・プロトコルが作動します。敵の波に対して攻勢をかけるタイミングを計るには、これらのパーセンテージを監視することが不可欠です。

| エネルギーレベル | ステータス効果 | 戦術的推奨事項 |
| :--- | :--- | :--- |
| **0% - 30%** | ロックダウン有効 | ステルスとパワーノードの発見に集中する。 |
| **31% - 60%** | 攻撃警告 | セクターガードの配備に備え、カバーを利用する。 |
| **61% - 90%** | 電力再接続中 | 攻撃能力が強化される。積極的に敵を掃討する。 |
| **100%** | ロックダウン解除 | 主要目標達成。エリアの安全確保完了。 |

> **💡 ヒント:** 大規模な敵グループと交戦する前に、常にプライマリ電力の再接続を優先してください。エネルギーレベルが高いほど、主要な攻撃アビリティのクールダウンが大幅に短縮されます。

## 戦闘プロトコルとEDIシステム

EDIシステムは、**Idus Pragmata**内においてナレーターであると同時に、ゲームプレイ上の障害としても機能します。これは施設のセキュリティAIとして、プレイヤーを脅威と識別し、「セクターガード・プロトコル」を開始します。EDIが発する音声や視覚的な手がかりを理解することで、戦闘において一瞬の隙を突くアドバンテージを得ることができます。

### セクターガード・プロトコルへの対処
EDIが「未登録のゲスト」を検知すると、ターゲットのスキャンを開始します。これが移動の合図です。立ち止まっているプレイヤーは、長距離ミサイルの一斉射撃や局所的なエネルギー爆発を含む施設の自動防衛システムの格好の標的となります。

1. **ターゲット識別**: 赤いスキャンラインが表示されたら、すぐに視線を遮るもの（遮蔽物）に隠れてロックオンを妨害してください。
2. **ミサイル回避**: セクターガードはしばしばクラスター状にミサイルを発射します。「オーバードライブ」ダッシュを使用して、飛来する弾丸の間を縫うように移動しましょう。
3. **カウンター攻撃**: 敵の「ヒート（熱）」シグネチャに注目してください。一斉射撃の後に敵がオーバーヒートすると、クリティカルダメージを与えやすくなります。

## 高度な戦闘戦術

基本的な射撃や回避以外にも、**Idus Pragmata**には「オーバードライブ・フォンコール」のようなユニークなメカニクスが導入されています。この能力はプレイヤーのパワーを一時的に急上昇させ、セクターガードの部屋全体を一掃できるような壊滅的な攻撃を可能にします。

### オーバードライブ・システム
オーバードライブ・コールを開始するには、通常、成功したパリィや高エネルギー状態での戦闘によってチャージされるフルメーターが必要です。発動すると、プレイヤーは強化状態に入り、ダメージ耐性が向上し、特別な「ヒート」ベースの攻撃が使用可能になります。

| アビリティ | 効果 | 最適な使用場面 |
| :--- | :--- | :--- |
| **オーバードライブ・ダッシュ** | ターゲットの背後へ瞬時にテレポート | ミサイルを放つガードとの距離を詰める。 |
| **ヒートブラスト** | 範囲（AOE）エネルギー爆発 | 低レベルドローンの群れを一掃する。 |
| **プライマリ・セバー** | 単体への高ダメージ攻撃 | エリートセクターガードやボスを無力化する。 |

> **警告:** エネルギーレベルが40%未満のときにオーバードライブを使用すると、一時的な電力不足（ブラウンアウト）が発生し、アビリティ終了後に数秒間動けなくなる可能性があります。

## 敵のタイプと対応策

**Idus Pragmata**のスケッチブック・デモに登場する敵は主にロボットであり、厳格なロジックパターンに従っています。これらのパターンを学習することで、動きを予測し、より困難な遭遇に備えてリソースを温存することができます。

### セクターガードのバリエーション
施設は、検出された「不正アクセス」のレベルに基づいて異なるタイプのユニットを配備します。基本ユニットは数に頼りますが、高度なユニットは特定のカウンター戦略を必要とする洗練された武器を使用します。

*   **スカウトドローン**: EDIにあなたの位置を通知する高速移動ターゲット。増援を防ぐために、これらを最優先で破壊してください。
*   **ミサイルバッテリー**: 静止していますが致命的です。標準的な弾道攻撃に対して装甲を持っているため、倒すには「ヒート」管理が必要です。
*   **エリートプロテクター**: これらのユニットは「プライマリ接続を切断（セバー）」し、迅速に対処しないと電力復旧の進行状況をリセットしてしまいます。

| 敵ユニット | 主要攻撃 | 弱点 |
| :--- | :--- | :--- |
| **スカウトドローン** | スキャン/警告 | 低体力。速射に弱い。 |
| **ミサイルバッテリー** | マルチミサイル斉射 | リロード中に露出する冷却ベント。 |
| **エリートプロテクター** | 接続の切断 | 移動が遅い。オーバードライブ攻撃に弱い。 |

## デモクリアのための最終ヒント

デモを完了するには、攻撃性と忍耐のバランスが必要です。ダイアログで「エネルギーレベルが回復した」と表示されたら、それは多くの場合、ボス戦や主要なロックダウンイベントが終了間近であるサインです。この時間を利用して残りのリソースを収集し、最後の攻勢に備えましょう。

*   **ダイアログを聞く**: EDIや仲間は、しばしば攻撃が来る際に音声で合図をくれます（「来るぞ！」「これを食らえ！」など）。
*   **ヒートを監視する**: プレイヤーと敵の両方が熱を発生させます。このリソースを管理することで、銃撃戦の最中にスタン状態になるのを防ぐことができます。
*   **動き続ける**: セクターガードの「薄汚い連中（Dirty bastards）」はあなたを追い詰めようとします。常に脱出ルートを確保しておきましょう。

## FAQ

**Q: Idus Pragmataで60%攻撃警告に対処する最善の方法は何ですか？**

A: 60%攻撃警告が作動すると、EDIはより頻繁にセクターガードを配備します。最善の戦略は、パワーノードの近くで防御しやすい場所を見つけ、再接続プロセスを続行する前に最初の波をクリアすることに集中することです。

**Q: オーバードライブ・フォンコールはどうやって発動しますか？**

A: オーバードライブ・システムはアクティブな戦闘を通じてチャージされます。メーターが満タンになったら、画面上のプロンプトに従ってコールを開始してください。これによりステータスが大幅に上昇し、短期間ユニークなアビリティが解放されます。

**Q: EDIがプライマリ接続を切断するのを止めることはできますか？**

A: はい。「不正アクセスを検知」という警告が表示されたら、電力ターミナルを操作しようとしているエリートプロテクターを探してください。これらのターゲットを即座に無力化することで、接続の切断を防ぐことができます。

**Q: 戦闘中に体力を回復する方法はありますか？**

A: 体力は主に、倒した敵がドロップするエネルギーを回収するか、セクター内で100%の電力復旧を達成することで回復します。100%復旧時にはプレイヤーと分隊全員が完全に回復します。
