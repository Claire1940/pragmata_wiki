## ダイアナのスケッチブックの仕組みを理解する

『Pragmata』体験版において、「スケッチブック」は物語の補完とやり込み要素のハブとして機能します。主人公に同行する謎の少女ダイアナは、プレイヤーのパフォーマンスや遭遇した敵、特定の環境トリガーに基づいて絵を描きます。従来の達成項目（アチーブメント）とは異なり、これらの絵は特定の戦闘シークエンス中に隠されたゲームプレイ基準を満たすことで「解放」されます。

最も入手が難しいのはダイアナの「2枚目の絵」であり、これは隠しボスとの遭遇に直結しています。これを発生させるには、体験版中盤のエレベーターシークエンスにおいて、厳格な戦闘ルールを守らなければなりません。これらの**Pragmata体験版の絵の出現条件**を満たせなかった場合、エレベーターはボーナスエンカウントが発生しないまま目的地に到着してしまい、その周回では最終的なスケッチブックの項目がロックされたままになります。

## エレベーターでの遭遇：ステップバイステップ解放ガイド

隠しコンテンツの核心は、エレベーターに乗り込んでくる3体のセキュリティボットの処理方法にあります。隠しボスを解放し、**Pragmata体験版の絵の出現条件**を満たすには、「戦術的な多様性」を示す必要があります。ゲーム側は、3体のボットそれぞれに対して、どの武器で「とどめを刺したか」を追跡しています。

### 武器の多様性に関する条件

隠し要素を発生させるには、すべてのキルを単一の武器に頼ることはできません。兵装をローテーションさせる必要があります。このルールは「とどめの一撃」にのみ適用されます。敵を弱らせる際はお好みの武器を使用して構いませんが、撃破する瞬間の武器が各敵で異なっている必要があります。

| ボットの順番 | 推奨される削り武器 | 必須のトドメ武器 |
| :--- | :--- | :--- |
| **1体目** | ピストル（通常射撃） | **ピストル**（最後の一撃） |
| **2体目** | ピストル / 近接攻撃 | **ショックウェーブガン** |
| **3体目** | ショックウェーブガン | **ハッキング・インターフェース** |

> **💡 ヒント:** ショックウェーブガンの爆風ダメージには注意してください。ハッキングで倒そうとしていた弱ったボットを誤って破壊してしまうと、シークエンス失敗となり、体験版を最初からやり直す必要があります。

### 詳細な戦闘ストラテジー

1.  **1体目のボット:** 最も単純な段階です。標準のピストル射撃で体力を削ります。機体から火花が出始めたら、頭部ユニットを狙って正確に最後の一撃を放ち、キルを確定させます。
2.  **2体目のボット:** 戦法を変えましょう。環境オブジェクトや軽い近接攻撃で体力を減らします。破壊寸前になったらショックウェーブガンに切り替えます。至近距離で一発放てば、2つ目の固有キル条件が満たされます。
3.  **3体目のボット:** 多くのプレイヤーが失敗する難所です。ピストルを使って体力を極限まで減らしてください。「ハッキング」のプロンプトが表示されるか、ボットがスタン状態になったら、ハッキングを開始します。ミニゲームを成功させてボットを破壊すれば、3つ目の固有武器フィニッシュとしてカウントされます。

## ダイアナの2枚目の絵を解放する

3つの異なる武器を使って3体のボットを無事に排除すると、エレベーターの挙動が変化します。通常のフロア出口に向かう代わりに「侵入者検知（Intruder Detected）」の警告が鳴り響きます。これにより、強化されたセキュリティユニットである隠しボスとの戦闘に突入します。

このボスを倒すことが、2枚目の絵に関連する**Pragmata体験版の絵の出現条件**の主要な要件です。戦闘後のカットシーンでダイアナがスケッチしている姿が見られ、メニューに新しい項目が追加されます。

| スケッチの項目 | 題材 | 主な条件 |
| :--- | :--- | :--- |
| **絵 #1** | 宇宙飛行士 | オープニングのチュートリアルを完了する。 |
| **絵 #2** | 隠された守護者 | エレベーターのボット3体を**異なる武器**で倒す。 |
| **絵 #3** | 月の地平線 | ノーダメージで最終展望地点に到達する。 |

## コンプリートを目指すための高度な戦闘のコツ

『Pragmata』の戦闘は、計算された物理ベースの挙動が特徴です。**Pragmata体験版の絵の出現条件**を一貫して満たすには、敵ボットのスタッガーゲージ（よろめきゲージ）をマスターする必要があります。

*   **スタッガー管理:** 敵には隠された「体勢」バーがあります。ショックウェーブガンを使用すると、このバーを急速に蓄積できます。ハッキングでとどめを刺したい場合は、まずショックウェーブガンでスタンさせ、その際に敵の体力を削りきらないよう注意しましょう。
*   **弾薬の節約:** 体験版ではリソースが限られています。1体目のボットにショックウェーブのチャージを無駄遣いしないようにしましょう。2体目のキルと、3体目をスタンさせるために必ず必要になります。
*   **ダイアナの支援:** ダイアナは時折、環境内のトラップをハイライトしてくれます。通常プレイでは有用ですが、エレベーターシークエンス中は使用を避けてください。環境キルは「固有武器」によるキルとしてカウントされないためです。

> **警告:** 3体のうち2体に対して同じ武器を使用してしまうと、隠しボスは出現しません。武器のローテーションは100%正確に行う必要があります。

## 技術要件と設定

これらのチャレンジに挑む際は、最高の体験を得るために[Pragmata公式サイトの設定](https://www.capcom.com/pragmata/)を参照し、高フレームレート向けに最適化されているか確認してください。3体目のボットでのハッキングインターフェースのタイミングには精度が求められます。

*   **入力遅延:** モニターを「ゲームモード」に設定し、ハッキングのミニゲームのタイミングが正確に反映されるようにします。
*   **視覚的な明瞭さ:** モーションブラーをオフにすると、ボットの体力が減った際の火花（とどめを刺せる合図）が確認しやすくなります。

## 隠し敵のロア（設定）と考察

**Pragmata体験版の絵の出現条件**を満たすことで遭遇できる隠しボスは、『Pragmata』の広大な世界観を垣間見せてくれます。この「隠された守護者（Hidden Guardian）」は標準的なセキュリティボットよりも高度なAIを搭載しており、不規則な移動パターンや、ピストルダメージを通す前にショックウェーブガンで破壊しなければならないシールドを駆使してきます。

この敵を観察することは、物語の背景を理解したいプレイヤーにとって不可欠です。ダイアナが描くこの敵の絵には、しばしば不可解な記号が含まれており、2026年初頭の体験版リリース以来、コミュニティではその解読が試みられています。

| 敵の種類 | 弱点 | 行動パターン |
| :--- | :--- | :--- |
| **標準型ボット** | ヘッドショット | 積極的な近接突進。 |
| **ショック・ボット** | ハッキング | 鈍重で強力な遠距離攻撃。 |
| **隠された守護者** | ショックウェーブ + ピストル | 高い機動力と防御シールド。 |

## 完了手順のまとめ

すべての**Pragmata体験版の絵の出現条件**を満たしたか確認するため、セッションを終了する前に以下のチェックリストを実行してください。

1.  スケッチブック体験版を最初から開始する。
2.  カットシーンをスキップせずにエレベーターセクションに到達する（スキップするとバグが発生するという報告があるため）。
3.  ボット1をピストルで撃破する。
4.  ボット2をショックウェーブガンで撃破する。
5.  ボット3をハッキングスキルで撃破する。
6.  直後に出現する隠しボス「隠された守護者」を倒す。
7.  ポーズメニューからダイアナのスケッチブックを開き、新しいアートが追加されているか確認する。

## FAQ

**Q: 1体目のボットの体力を削るためにショックウェーブガンを使ってもいいですか？**

A: はい。敵の体力を減らすためにどの武器を使っても構いません。**Pragmata体験版の絵の出現条件**において重要なのは、どの武器が「最後の一撃（とどめ）」を与えたかという点だけです。

**Q: 正しくボットを倒したはずなのに、隠しボスが出現しません。なぜですか？**

A: 多くの場合、偶発的なダメージが原因です。ボットがプレイヤーの武器による直接攻撃ではなく、状態異常（炎上や感電）や環境爆発で死亡した場合、「固有武器によるキル」としてカウントされないことがあります。最後の一撃は非常に慎重に行ってください。

**Q: Pragmata体験版には全部で何枚の絵がありますか？**

A: 現在、収集できる主な絵は3枚です。最も入手が難しいのは2枚目の絵で、これにはエレベーターでの特定の武器ローテーションによるキルが必要です。

**Q: 体験版でスケッチブックを完成させると報酬はありますか？**

A: ゲーム内アイテムなどの報酬はありませんが、スケッチブックを完成させると開発チームからの特別な「Thank You」画面が表示され、ダイアナの出自に関する追加のロア（設定）情報を得ることができます。
