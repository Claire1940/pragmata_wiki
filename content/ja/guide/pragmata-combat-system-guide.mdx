## 武器とリソース管理

ヒューの兵装は、永続的な基本武器と一時的な特殊兵装の2つのカテゴリに分けられます。激しい遭遇戦で攻撃の手を休めないためには、「損壊」メカニズムを理解することが不可欠です。

### 基本武器 vs 特殊兵装

基本武器（通常は汎用性の高いピストル）は、最も信頼できるツールです。自己補充型の弾薬システムを備えており、マガジンを使い切ってもバックグラウンドで自動的にリチャージされるため、完全に無防備になることはありません。

一方、特殊兵装は「ハイリスク・ハイリターン」の原則で動作します。ショックウェーブガンやステイシスネットなどのツールはリチャージされません。弾薬を使い切ると武器自体が壊れてしまうため、戦場で代わりのものを探す必要があります。

| 武器タイプ | 基本武器（ピストル） | ショックウェーブガン | ステイシスネット |
| :--- | :--- | :--- | :--- |
| **射程** | 中〜遠距離 | 近距離 | 中距離 |
| **主な機能** | 安定したDPS | 怯ませ＆高ダメージ | 群衆制御/スタン |
| **弾薬源** | 自動リチャージ | 床のドロップ品 | 床のドロップ品 |
| **耐久性** | 無限 | **空になると壊れる** | **空になると壊れる** |

> **警告:** 特殊兵装を温存しすぎないでください。使い切ると壊れますが、敵から頻繁にドロップするため、「完璧な瞬間」を待つよりも、敵の群れを一掃するために積極的に使用する方が得策です。

## ハッキングのメカニズム：ダイアナの役割

**プラグマタ戦闘システムガイド**における最大の特徴は、ハッキングインターフェースです。2026年の世界で遭遇する敵の多くは分厚い外装を持っており、標準的な射撃はほとんど通用しません。大きなダメージを与えるには、ダイアナを通じてハッキングを開始する必要があります。

### ハッキングの開始方法
プロセスを開始するには、ヒューがダイアナを護衛しながら、彼女をターゲットに向けさせる必要があります。接続が確立されると、画面にグリッド形式のミニゲームが表示されます。これは「ライブ」戦闘メカニクスであり、グラフを操作している間も敵は動き続け、攻撃してきます。

1.  **視線を維持する:** ダイアナの狙いが敵から外れると、接続が中断されます。中断した場所から再開できますが、その遅延が致命傷になることもあります。
2.  **グリッドをナビゲートする:** カーソルを動かして一連の青いノードを通過し、最終的に緑の出口ノードに到達する必要があります。
3.  **赤いトラップを避ける:** 注意マークの付いた赤いノードは、進行を妨げたりハッキングを失敗させたりします。

### ハッキングノードの種類

| ノードの色 | 戦闘への効果 | 優先度 |
| :--- | :--- | :--- |
| **青い正方形** | 標準的な進行ノード。ゴールに到達するために必要。 | **高** |
| **緑の正方形** | ハッキングを完了させ、敵の弱点を露出させる。 | **必須** |
| **黄色の正方形** | 床に落ちているボーナスノード。敵の防御力を下げる。 | **任意/戦略的** |
| **赤い正方形** | 警告ノード。失敗や進行のリセットを引き起こす。 | **回避** |

## 高度な戦闘：オーバードライブとシナジー

ハッキングを成功させるたびに、ダイアナの**オーバードライブ・プロトコル**ゲージが溜まっていきます。これは、ヒューが敵に完全に包囲されたような状況で真価を発揮する「アルティメット」能力です。

### オーバードライブ・プロトコルの発動
ゲージが満タンの時に**R3 + L3**を同時に押すと、ダイアナが巨大なエネルギーバーストを放出します。この攻撃には2つの目的があります。
*   広範囲内のすべての敵を即座にスタンさせます。
*   短時間、個別のハッキングミニゲームをスキップして、エリア内のすべての敵の弱点を自動的に「開放」します。

> **ヒント:** オーバードライブ・プロトコルは、ボス戦のフェーズや、装甲兵が多すぎて個別のハッキングが困難な「伏兵」の部屋のために取っておきましょう。

## 生存とチェックポイント

最善を尽くしても、月面の環境は過酷です。『PRAGMATA』では、「脱出ハッチ」を中心としたユニークなチェックポイントシステムを採用しています。

ヒューの体力が尽きても、伝統的な「ゲームオーバー」画面にはなりません。代わりに、最後に起動した脱出ハッチにテレポートされます。ただし、これらのハッチは自動的には有効になりません。ハッチを見つけたら、**L1**を長押ししてダイアナにハッキングを促す必要があります。これによりチェックポイントが「初期化」され、進行状況が保存されます。

## セクションまとめ：戦闘の流れ

**プラグマタ戦闘システムガイド**で優れた成果を出すには、以下の基本的な戦闘ループに従ってください。
1.  最も危険な装甲兵を特定する。
2.  **ステイシスネット**を使用してターゲットをその場に固定する。
3.  **L2**でロックオンし、ダイアナにハッキンググリッドを開始させる。
4.  他の敵を避けながらグリッドを操作し、床に**黄色いノード**があれば回収する。
5.  ハッキングが完了し「内部」が露出したら、**ショックウェーブガン**または**ピストル**に切り替えて敵を仕留める。

## FAQ

**Q: なぜショックウェーブガンがインベントリから消えたのですか？**

A: 『PRAGMATA』では、ショックウェーブガンのような特殊兵装は弾数に基づいた耐久性制限があります。弾を使い切ると武器は壊れます。その武器タイプを再び使用するには、戦場で新しいものを見つける必要があります。

**Q: ハッキングシステムを使わずにゲームをクリアできますか？**

A: 雑魚敵の一部は標準的な射撃で倒せますが、**プラグマタ戦闘システムガイド**で強調している通り、エリートや装甲兵は大幅なダメージ軽減を持っています。ハッキングこそが弱点を露出し、「大ダメージ」を与える唯一の効率的な方法です。

**Q: ハッキング中にダイアナの狙いが遮られたらどうなりますか？**

A: ハッキンググリッドが一時停止、または切断されます。L2でロックオンを再確立する必要があります。幸い、通常はグリッドの最初からやり直すのではなく、直前の進行状況から再開できます。

**Q: ハッキンググリッドに黄色いノードをもっと出現させるには？**

A: 黄色いノードは、戦闘中に環境の床に落ちている物理的なピックアップアイテムです。ハッキングを開始する前にそれらの上を歩いて回収しておくと、次のハッキンググリッドに「スロット」され、そこを通過することで敵の防御力を大幅に弱体化させることができます。
