カプコンの待望のSFアドベンチャー『プラグマタ（Pragmata）』の月面の謎が、ついに「Sketchbook」体験版を通じてその姿を現しました。多くのプレイヤーは単にクレジットに到達するだけで満足するかもしれませんが、この「バーティカル・スライス（垂直切り出し版）」の真の深みは、あらゆる**プラグマタ 体験版 エンディング**のバリエーションを追い求めることにあります。体験版を完全にマスターするには、コンパニオンキャラクターであるダイアナが描いた8つのユニークなシークレットイラストをアンロックする必要があります。これらの報酬は単なる見せかけではなく、ゲームの戦闘、探索、スピードメカニクスの習熟度の証となります。

特定の**プラグマタ 体験版 エンディング**を達成するには、運以上のものが必要です。ゲームは、被ダメージ、総プレイ時間、ダイアナのサポート能力と自身の武器をいかに効果的に使い分けたかなど、さまざまな指標でプレイヤーのパフォーマンスを追跡します。「パーフェクトガード」ランを目指す場合でも、捉えどころのない「エリートエンカウント」の発生を試みる場合でも、このガイドでは、2026年4月24日の製品版発売前にギャラリーを埋めるために必要な正確な手順を解説します。

## 全8種類のエンディングイラスト：解放条件

全8種類のイラストをアンロックするには、複数回のプレイが必要です。「パシフィスト（平和主義）」ランや「クラウドコントロール」チャレンジなど、一部の条件は根本的に異なるプレイスタイルを要求されるため、一度のプレイで同時に達成することはできません。以下の表は、最終リザルト画面に表示される各報酬の具体的な基準をまとめたものです。

| イラスト No. | 条件名 | 達成のための具体的な要件 |
| :--- | :--- | :--- |
| **No. 1** | **パシフィスト・サポート** | ダイアナの能力のみを使用。最初の強制射撃以外、銃を撃たない。 |
| **No. 2** | **エリート・エンカウント** | 最後のエレベーターで高レベルのオレンジ/レッドロボットを出現させ、撃破する。 |
| **No. 3** | **コンプリーショニスト** | 体験版内のすべてのイエローキューブを回収し、すべての大型宝箱を開ける。 |
| **No. 4** | **パーフェクトガード** | ダメージを一切受けずに体験版をクリアする。 |
| **No. 5** | **スピードデーモン** | 総プレイ時間**10分未満**で最終リザルト画面に到達する。 |
| **No. 6** | **クラウドコントロール** | グリーンガンを使用して、1発のショットで2体の敵を拘束する。 |
| **No. 7** | **スカベンジャー** | 施設内に散らばる全一般リソースの80%以上を回収する。 |
| **No. 8** | **サバイバー** | 他の特殊条件を満たさずに体験版を通常クリアする。 |

> **💡 プロのヒント：** イラストNo. 1の場合、最初のカットシーンを発生させるために最初に1発だけ撃つ必要があります。その直後、誤ってパシフィストランを無効にしないよう、すぐに銃器の装備を外してください。

## シークレットエンディング No. 2（エリート・エンカウント）の発生方法

2番目のイラストは、最終ボスへと続くエレベーターのシークエンス中に隠された「バラエティキル」カウンターに依存するため、入手が最も難しいとされています。通常、エレベーターには標準的なドローンや基本的なロボットが出現します。しかし、ツールキット全体を使いこなすことで、希少な高レベルの「オレンジロボット」を強制的に出現させることができます。

この**プラグマタ 体験版 エンディング**報酬を確保するには、エレベーター内で以下の手順を正確に実行してください：

1.  **ドローンの排除：** 標準のピストルまたはライフルを使用して、出現する飛行ユニットを素早く倒します。
2.  **最初のロボットを無力化：** 射撃する代わりに、ダイアナにハッキング能力を使うよう指示し、最初の地上型ロボットを無力化します。
3.  **2体目のロボットを仕留める：** **グリーンガン**（静止/制御銃）に切り替え、テザーショットを使用して2体目の標準ロボットを倒します。
4.  **出現を確認：** 正しく行われれば、2体の高レベルオレンジロボットがワープしてきます。それらを倒すと、No. 2のイラストが確定します。

## スピードランの極意：イラスト No. 5

10分以内に体験版をクリアするのは、月面施設のレイアウトに精通している必要がある困難なタスクです。不要な戦闘は無視し、「緊急ロックダウン」端末とパワーセルパズルに完全に集中しなければなりません。

| 目的 | 必要なアクション | 時短戦略 |
| :--- | :--- | :--- |
| **初期侵入** | 最初の侵入者ウェーブを排除。 | すぐに**オーバードライブ**を使用して、数秒で部屋を制圧する。 |
| **パワーセル 1** | 上位3つのプロトコルを再ルーティング。 | 小さな箱は無視して走り抜け、リソースの回収で立ち止まらない。 |
| **パワーセル 2** | セルの手動再起動。 | ウサギ型の敵は無視する。彼らの索敵範囲は狭い。 |
| **エレベーター移動** | 侵入者を排除して進み続ける。 | エレベーターの進行を止める可能性があるドローンを最優先する。 |

> **⚠️ 警告：** 一度メンテナンスシャフトに降りると、上の階層に戻ることはできません。最終シュートに飛び込む前に、タイムや回収目標を達成しているか確認してください。

## 探索とリソース管理（No. 3 & No. 7）

月面基地は、環境ストーリーテリングと隠されたキャッシュで満たされています。「コンプリーショニスト」報酬（No. 3）を獲得するには、すべてのイエローキューブを見つける必要があります。これらのキューブは伝承（ロア）を提供し、破壊可能な壁の向こう側や、ダイアナの「リフト」能力が必要な高い梁の上に隠されていることが多いです。

対照的に、「スカベンジャー」エンディング（No. 7）は、希少性よりも量に焦点を当てています。一般的なクラフト材料の大部分を拾う必要があります。HUDにはパーセンテージカウンターは表示されませんが、次の進行端末を操作する前にすべての通路を探索するのが良い目安です。

## グリーンガンによる戦闘マスタリー（No. 6）

グリーンガンは、テザー（紐）を発射するユニークな戦術ツールです。敵をスタンさせるためにも使用できますが、イラストNo. 6では「リンク（連結）」させる必要があります。2体の敵が並ぶのを待ち、それらをテザーで繋ぐショットを放つことで、衝突させたりダメージを共有させたりします。

これを達成するのに最適な場所は、最初の緊急ロックダウンを解除した直後の2番目の遭遇エリアです。狭い通路に複数のロボットが出現するため、1発の精密なショットで2つのユニットを「コントロール」する絶好の機会となります。

## FAQ

**Q: 1回のプレイで複数のエンディングイラストをアンロックできますか？**

A: はい、多くの条件は併用可能です。例えば、「パーフェクトガード」（No. 4）を維持しながら「スピードラン」（No. 5）を達成できます。ただし、「パシフィスト」（No. 1）と「クラウドコントロール」（No. 6）は、後者が特定の武器を使用する必要があるため、組み合わせることはできません。

**Q: 最終ボス戦でダメージを受けると、No. 4のイラストは無効になりますか？**

A: はい。No. 4の**プラグマタ 体験版 エンディング**報酬を確保するには、体験版開始から最終リザルト画面まで、被ダメージゼロの記録を維持する必要があります。このチャレンジでは、ダイアナのシールド能力を頻繁に使用することを強くお勧めします。

**Q: すでにアンロックしたイラストはどこで見られますか？**

A: 現在「Sketchbook」体験版では、イラストはゲーム終了後のサマリー画面でのみ表示されます。再度確認するには、その条件でもう一度クリアするか、手動でキャプチャしたプラットフォームのスクリーンショットギャラリーから閲覧する必要があります。

**Q: エレベーターでのオレンジロボットの出現はランダムですか？**

A: いいえ、確定的な出現条件があります。最初の3体の敵を3つの異なる方法（標準の銃、ダイアナの能力、グリーンガン）で倒すことで、「バラエティボーナス」が発生し、毎回確実にエリートエンカウントを誘発できます。
