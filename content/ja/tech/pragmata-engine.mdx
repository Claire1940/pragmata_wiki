## RE ENGINEの力：Pragmataにおけるパスレイトレーシング

『PRAGMATA』の視覚的スペクタルの中心にあるのは、RE ENGINE内でのカプコンによるパスレイトレーシングの高度な実装です。従来のラスタライズやハイブリッド・レイトレーシングのアプローチとは異なり、パスレイトレーシングは光の伝搬をより正確にシミュレートし、信じられないほどリアルなライティング、反射、影を実現します。この手法は何百万もの光の経路を追跡し、現実世界での光の振る舞いを再現することで、「生き生きとしたシーン」と「高品質な画像」を作り出します。

RE ENGINE開発のキーマンであるカプコンの三嶋仁志氏は、『PRAGMATA』（および『バイオハザード レクイエム』）におけるパスレイトレーシングが、標準的なレイトレーシングに対してどのような利点があるかを強調しました。レイトレーシングは通常、反射や影といった特定の光の相互作用に焦点を当てますが、パスレイトレーシングは光をより包括的かつ物理的に正確に表現します。これにより、より深みとリアリズムのある優れた視覚体験がもたらされます。

パスレイトレーシングが『PRAGMATA』のビジュアルをどのように高めているかの比較は以下の通りです。

| 機能 | パスレイトレーシング (Pragmata) | レイトレーシング (一般的) |
|---|---|---|
| **ライティングの正確性** | リアルタイムで物理的に正確なグローバルイルミネーション | リアルタイム、間接照明にはラスタライズとのハイブリッドが多い |
| **影** | より詳細で柔らかく、リアルに拡散する | 良好だが、時として不自然または精度に欠けることがある |
| **反射** | より鮮明で安定しており、極めて正確 | 一部のRE ENGINEタイトルでは粒状感が出たり詳細を欠くことがある |
| **間接照明** | 完全にシミュレートされ、優れた品質とバウンス光を実現 | 近似値であったり動的でないことが多く、リアリズムに欠ける |
| **オクルージョン** | より厳密でリアルなアンビエントオクルージョン | 良好だが、フルパスレイトレーシングの微妙な深みには及ばない |
| **デノイジング** | 特にDLSS RRにより、より安定して効果的 | 実装によってはアーティファクトが目立つことがある |

## 舞台裏：RE ENGINEのパスレイトレーシング革新

Pragmataエンジンの印象的なビジュアルとパフォーマンスは、カプコンのRE ENGINEにおける広範な技術革新と最適化の結果です。カプコンと協力したNVIDIAのCalvin Shu氏は、GDCでこれらの進歩の多くを詳しく説明し、効率的にリアルタイム・パスレイトレーシングを達成した方法に焦点を当てました。

重要な要素は、NVIDIA DLSS Ray Reconstruction (DLSS RR) との統合です。このAI搭載技術はレイトレーシング画像を強化するために設計されており、パスレイトレーシングされたシーンで画質を維持するために不可欠な優れたデノイジングと再構築を提供します。

RE ENGINEのために開発された主な最適化技術は以下の通りです。

| 最適化 | 目的 | ビジュアルへの影響 | パフォーマンスへの影響 |
|---|---|---|---|
| **簡略化されたBSDF (RIS)** | レンダリング中の光サンプリングを効率的に評価 | 計算負荷を抑えつつ視覚品質を維持 | ストレスデモで約15%のパフォーマンス向上に寄与 |
| **ライトグリッドカリング** | 不要な光源をカリングして点光源の計算を加速 | 正確な影と効率的なライティング計算を保証 | 高速化の最も重要な要因として特定 |
| **輝度補正** | 高強度のスポットライトやIBLによる分散の問題に対処 | 一貫したライティングを実現し、DLSS RRの品質を向上 | 多様なシーンで安定したパスレイトレーシング結果を得るために不可欠 |
| **発光ポリゴンサンプリング** | 発光光源から発生するノイズを低減 | より鮮明で安定した発光エフェクト | Walkerのエイリアス法を用いた効率的なサンプリング |
| **ReSTIR GI (Resampled Real-time Global Illumination)** | DLSS RRを安定させ、間接照明のノイズを低減 | 間接光におけるゴーストやぼやけを最小限に抑える | 全体的な画質と時間的安定性を向上 |
| **ガイドバッファとディスオクルージョンマスク** | 複雑なシナリオでの特定のDLSS RRアーティファクトを解決 | より鮮明な髪、正確なガラスの反射、正確な雨や水たまり | 視覚的な忠実度を維持し、ぼやけを防ぐために重要 |
| **髪の毛用Strand BVH** | 低重力環境での髪の束を管理 | リアルな髪の動きと正確なセルフシャドウを可能に | 以前のRE ENGINEタイトルで見られた視覚的な不整合を解消 |
| **DXR 1.2とSER (Shader Execution Reordering)** | GPUスループット向上のための最新レイトレーシングAPI | 複雑なレイワークロードに対するハードウェアの有効活用 | オーバーヘッドを抑えた「光速のスループット」を提供 |

特筆すべき課題の一つは、特に環境の間接照明に関するアーティストの期待に応えることでした。エンジンチームはテクニカルアーティストと密接に協力し、ReSTIR GIを有効にした際のオクルージョンの微妙な変化などの問題を解決しました。また、『PRAGMATA』独自の低重力環境はキャラクターの髪に特殊な解決策を必要とし、パスレイトレーシングされたライティングと矛盾しないリアルな動きと影を確保するために「Strand BVH」が必要となりました。

パフォーマンスの改善は継続的に行われ、ライトループの統合やDXR 1.2とSERの活用といった最適化により、初期のストレステストでは4K DLAAシーンで21ミリ秒から13.3ミリ秒への短縮が見られました。これらの努力は、Pragmataエンジンで洗練された視覚的に素晴らしい体験を届けるというカプコンのコミットメントを浮き彫りにしています。

## システム要件とエクスペリエンスの最適化

Pragmataエンジンが提供する高度なグラフィックスを十分に堪能するには、システム要件と最適化のヒントを理解することが重要です。デモ版はプレイヤーが期待できる内容の洞察を与えてくれ、スムーズな1080p 60 FPS体験に向けた推奨スペックが提示されました。

**最低システム要件 (ターゲット: 1080p, 約45 FPS):**
*   **GPU:** NVIDIA GeForce GTX 1660 6GB または AMD Radeon RX 5500 XT 8GB
*   **CPU:** Intel Core i5-9400 または AMD Ryzen 3 3100
*   **RAM:** 8GB

**推奨システム要件 (ターゲット: 1080p, 60 FPS):**
*   **GPU:** NVIDIA GeForce RTX 2060 Super 8GB または AMD Radeon RX 6600 8GB
*   **CPU:** Intel Core i7-10700 または AMD Ryzen 5 5500
*   **RAM:** 16GB

**レイトレーシング推奨 (ターゲット: 1080p, 60 FPS、RT有効):**
*   **GPU:** NVIDIA GeForce RTX 3060 12GB または AMD Radeon RX 6700 XT 12GB

> **💡 最適化のヒント:** レイトレーシングで最高の視覚体験を得るには、GPUに少なくとも12GBのVRAMがあることを確認してください。VRAMが懸念される場合は、DLSSやFSRを「バランス」または「パフォーマンス」モードで使用することで、品質を大きく損なうことなく高いフレームレートを維持できます。これらのアップスケーリング技術はPragmataエンジンにおいて非常に効果的で、レイトレーシングを有効にした1440pでもフレームレートを100 FPS以上に押し上げることがよくあります。ご自身の環境に合わせて、ビジュアルの忠実度とパフォーマンスの完璧なバランスを見つけるために、様々な設定を試してみてください。

8GBのVRAMを搭載したカードでも『PRAGMATA』をレイトレーシング有効で動作させることは可能ですが、長時間のプレイや複雑なシーンでは、テクスチャのちらつきやわずかなパフォーマンスの低下など、VRAMに関連する軽微な問題が発生する可能性があることに注意してください。一貫した高品質のレイトレーシング体験には、12GB以上のVRAMを搭載したGPUを選択するのが理想的です。

デモ版のゲームのデフォルト設定は、ミドルレンジのハードウェアでもレイトレーシングを有効にした「ほぼ高設定」を優先していました。これは、Pragmataエンジンが箱出しの状態で印象的に見えるように設計されている一方で、優れたパフォーマンスも提供していることを示しています。

## FAQ

**Q: Pragmataエンジンとは何ですか？**

A: Pragmataエンジンとは、ゲーム『PRAGMATA』のために大幅に強化されたカプコン独自の「RE ENGINE」を指します。高度なリアルタイム・パスレイトレーシング技術による極めてリアルなライティング、反射、影に加え、様々なパフォーマンス最適化が特徴です。

**Q: Pragmataはレイトレーシングとパスレイトレーシングのどちらを使用していますか？**

A: 『PRAGMATA』はフルパスレイトレーシングを採用しています。これはレイトレーシングの進化形です。パスレイトレーシングは従来のレイトレーシングよりも包括的に光をシミュレートし、より詳細な影、鮮明な反射、優れた間接照明をもたらします。

**Q: Pragmataの一般的なシステム要件は？**

A: 推奨される1080p 60 FPS体験には、RTX 2060 Super 8GBまたはRX 6600 8GBのGPU、Intel Core i7-10700またはRyzen 5 5500のCPU、および16GBのRAMが必要です。レイトレーシングを利用する場合は、RTX 3060 12GBまたはRX 6700 XT 12GBが推奨されます。

**Q: RE ENGINEはどのようにしてPragmataの高度なグラフィックスと良好なパフォーマンスを両立させていますか？**

A: RE ENGINEは、光サンプリングのための簡略化されたBSDF、ライトグリッドカリング、輝度補正、効率的な発光ポリゴンサンプリング、ReSTIR GIなどの数多くの最適化を採用しています。また、NVIDIA DLSSやAMD FSRといったアップスケーリングソリューション、DXR 1.2とShader Execution Reordering (SER) などの最新APIを最大限に活用し、スムーズなパフォーマンスを確保しています。
