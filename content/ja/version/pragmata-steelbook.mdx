## プラグマタ スチールブックを探す

パッケージメディアのコレクターにとって、**プラグマタ スチールブック**は本作のプレゼンテーションの頂点と言えます。歴史的にカプコンは、米国のBest Buyや英国のGAMEといった大手小売店と提携し、予約特典としてスチールブックを提供してきました。これらのケースは通常、高品質なエンボス加工のアートワークが施され、月の表面やヒューの船外活動（EVA）スーツの複雑なディテールが描かれることが多いです。

> **注意：** スチールブックの数量はほぼ常に限定されています。小売店が特典として**プラグマタ スチールブック**を掲載している場合、発売日の数週間前に完売することも珍しくないため、早めに予約を確保することを強くお勧めします。

初期の小売販売分を逃してしまった場合、二次市場で**プラグマタ スチールブック**が単体で販売されているのを見つけることができるかもしれませんが、プレミアム価格を支払うことになるでしょう。購入の際は、スチールブックが「G2」サイズ（PS5/Xbox用の標準Blu-rayサイズ）なのか、あるいはNintendo Switch 2用の小型サイズなのかを必ず確認してください。

## 予約特典：ヒュー＆ダイアナ用セット

*   **ヒュー用コスチューム：** 「ネオ・ブシドー」セット。EVAスーツ用の侍風アーマープレートが特徴。

重要な点として、これらのコスチュームは純粋に外見のみの変更です。防御力の上昇、戦闘能力の向上、難易度の変化はありません。カプコンは、これらのアイテムが後日デジタルストアで個別に販売される可能性も示唆しているため、レビューを待ってから購入したい場合は、スキン目的だけで無理に予約する必要はありません。

## デラックスエディション：シェルター・バラエティパックの価値は？

『PRAGMATA』のデラックスエディションは、スタンダード版より10ポンド/10ドルほど高価です。この追加投資により、ミッションの合間にヒューとダイアナが過ごす拠点「シェルター」に焦点を当てた「シェルター・バラエティパック」を受け取ることができます。

| アイテム | 内容 |
| :--- | :--- |
| **追加コスチューム** | 両メインキャラクターの追加ビジュアルバリエーション。 |
| **グリップガンスキン** | ヒューの主要ツールのユニークな外観変更。 |
| **BGMトラック** | シェルター内で再生できる代替背景音楽。 |
| **ダイアナ用ジェスチャー** | ダイアナAI用の新しいエモートとインタラクション。 |
| **アートライブラリ** | コンセプトアートや世界観設定データのデジタルコレクション。 |

これらの特典は、キャラクターのカスタマイズや世界観の深掘りを楽しむプレイヤーにとっては魅力的ですが、新しいミッションやストーリーコンテンツを追加するものではありません。「フォトモード」を愛用する方や、キャラクターメニューで多くの時間を過ごす方にとって、デラックスエディションは堅実な選択です。しかし、アクションや「クレイドル」セキュリティプロトコルの謎解きだけを目的にしているなら、スタンダード版を選んでもゲームプレイを逃すことはありません。

## ゲームプレイメカニクス：期待される内容

『PRAGMATA』は、システム主導型のシングルプレイヤーSFアクションゲームです。基本的なループは、危険な環境の探索、物理ベースのパズルの解決、そしてタクティカルな戦闘で構成されています。ヒューとダイアナの関係性が体験の中核であり、ダイアナは単なる同行者ではなく、ヒューには不可能な方法で環境と干渉するゲームプレイメカニクスの重要な一部です。

ゲームは雰囲気と物語の謎を重視しています。厳格なセキュリティプロトコルに支配された世界における「未登録のゲスト」として、グリップガンと環境への適応力を駆使して生き残らなければなりません。これらのコアシステムはDLCや特別版に縛られておらず、どのエディションを購入してもすべてのプレイヤーが完全なストーリーを体験できます。

## FAQ

**Q: 予約すれば必ずスチールブックが付いてきますか？**

A: いいえ。**プラグマタ スチールブック**は通常、特定の小売店限定の特典です。購入予定の店舗（Amazon、ゲオ、楽天ブックスなど）で、金属製ケースが特典に含まれているか確認する必要があります。

**Q: フィギュア付きのコレクターズエディションはありますか？**

A: カプコンはまだ大規模な「プレミアム・コレクターズエディション」の詳細を発表していませんが、通常は2026年4月24日の発売日が近づくと発表されます。これらには通常、**プラグマタ スチールブック**、ダイアナやヒューのスタチュー、物理的なアートブックが含まれます。

**Q: 発売後に予約特典のコスチュームを入手できますか？**

A: カプコンは「ネオ・ブシドー」および「ネオ・クノイチ」セットが後日別途販売される可能性があると述べています。ただし、発売時に入手する確実な方法は現在のところ予約のみです。

**Q: Nintendo Switch 2版も同じ内容ですか？**

A: はい、ゲームプレイの内容はすべてのプラットフォームで同一です。グラフィックの解像度などは異なる場合がありますが、ストーリー、ミッション、および利用可能なエディション（**プラグマタ スチールブック**の可能性を含む）は共通です。
