## 『プラグマタ』Switch 2 パフォーマンスガイド：技術解説

Nintendo Switch 2版の本作は、主にRE ENGINEの効率性により、技術的な驚異となっています。カプコンのタイトルが大幅な妥協を必要とすることが多かった過去の世代とは異なり、新ハードウェアでは「現世代機」に近い感覚を実現しています。ゲームは60 FPSの出力を目標としていますが、この目標の安定性はTVモードか携帯モードかによって大きく異なります。

### TVモード vs. 携帯モードのパフォーマンス

TVモードでプレイする場合、ハードウェアは現代のディスプレイに合わせてより高い解像度を出そうとします。しかし、これは安定性を犠牲にします。静かなエリアでは安定して60 FPSを維持しますが、複数の敵が登場する激しい戦闘シーケンスでは、フレームレートが50台前半から40台後半に低下することがあります。

逆に、携帯モードは一般的に優れたプレイ方法と考えられています。Switch 2の可変リフレッシュレート（VRR）のおかげで、内部解像度がわずかに下がっても、ゲームプレイは非常にスムーズに感じられます。携帯モードでは画像が少し「粗く」あるいはソフトに見えるかもしれませんが、一貫した動きが得られるため、多くのテスターにとって好ましい選択肢となっています。

| 機能 | TVモード | 携帯モード |
| :--- | :--- | :--- |
| **目標フレームレート** | 60 FPS（可変） | 60 FPS（安定） |
| **パフォーマンスの最低ライン** | 激しい戦闘中で約48 FPS | VRRサポートにより約55 FPS |
| **画質** | 高解像度、よりシャープなテクスチャ | 低解像度、やや「粗い」見た目 |
| **安定性** | 開けた場所で時折スタッターが発生 | 非常にスムーズな体験 |

> **⚠️ 警告：** 現在、フレームレートを30 FPSに固定するオプションはありません。フレームレートの変動に敏感な方は、VRR機能を活用するために携帯モードでプレイすることを強くお勧めします。

## 戦闘メカニクスと2人のキャラクターのシナジー

『プラグマタ』は単なる標準的なサードパーソン・シューティングではありません。独自の「スプリット・プロタゴニスト（分離型主人公）」システムを採用したSFアクションアドベンチャーです。プレイヤーは、乗組員と離ればなれになった宇宙飛行士ヒュー・ウィリアムズを操作し、背中には謎のアンドロイドの少女ダイアナを背負っています。この設定は単なる物語の味付けではなく、戦闘ループの核心です。

### ハッキングのミニゲーム

ダイアナは戦術的なアドバンテージとなります。ヒューが伝統的な銃撃と移動を担当する一方で、ダイアナは即座にAIアンドロイドをハッキングできます。これには、攻撃を避けながら同時に「ミニゲーム」に取り組む必要があります。画面上のアイコンの小さな迷路を通り抜け、クリティカルヒットのチャンスをアンロックしなければなりません。

| メカニクス | 説明 | 戦闘への影響 |
| :--- | :--- | :--- |
| **通常射撃** | ヒューのメイン武器による攻撃 | ダメージは小さく、ボスに対しては単独では効果が薄い |
| **ダイアナのハッキング** | リアルタイムの迷路ミニゲーム | 弱点を露出させ、**クリティカルヒット**を可能にする |
| **スラスターブースト** | 未来的なジャンプと回避のメカニクス | アスレチック要素や範囲攻撃の回避に不可欠 |
| **スキャナー** | 環境分析ツール | 隠されたボタンやパズルの解法を明らかにする |

両方のキャラクターを効果的に使うことは、まるで「脳の2つの部分を同時に使う」ような感覚です。ハッキングのメカニクスを無視すると、基本的な敵にさえ微々たるダメージしか与えられないことに気づくでしょう。ヒューで射撃するリズムを保ちつつ、ダイアナでハッキングのルートを計画することが、より「派手な」遭遇戦を生き延びる鍵となります。

## ビジュアルの忠実度とRE ENGINE

ゲームのビジュアル・アイデンティティは無機質でクリーンであり、映画『エイリアン』のような古典的なSF映画を彷彿とさせます。環境は白く、清潔で未来的であることが多く、これがRE ENGINEのライティング効果を際立たせています。最も印象的なビジュアル機能の一つはキャラクターのレンダリングで、特にダイアナの髪の物理演算はSwitch 2のパワーのベンチマークとなっています。

しかし、PlayStation 5版と比較すると、テクスチャ解像度やシャドウ品質に顕著な低下が見られます。Switch 2版も健闘していますが、60 FPSを維持するためにカプコンがいくつかの妥協を強いられたことは明らかです。

### Switch 2プレイヤー向けの最適化のヒント

この**『プラグマタ』Switch 2 パフォーマンスガイド**に基づき、最高の結果を得るために以下の最適化戦略に従ってください：

1.  **携帯モードを優先する：** 小さな画面が低解像度をカバーし、VRRサポートにより大型テレビよりもフレームレートが大幅に安定して感じられます。
2.  **高品質のオーディオを使用する：** 声優の演技には賛否両論ありますが、環境音は最高クラスです。ヘッドホンを使用して、敵の動きやハッキングの合図をより正確に把握しましょう。
3.  **スキャナーを使いこなす：** 多くの「ロックされた」エリアは、実際には遠隔ボタンをスキャンする必要があるパズルです。これらを遠くから起動することが、アスレチックセクションを進むために頻繁に要求されます。
4.  **回避を恐れない：** ヒューのスラスターは非常にレスポンスが良いです。Switch 2の60 FPS環境では、攻撃的なAIボットに対する最も信頼できる防御手段は回避です。

## レベルデザインと探索

ゲームは、直線的な戦闘アリーナと複雑なアスレチック、パズル解決のバランスが取れています。以前はロックされていたエリアを、新しい能力やダイアナのアップグレードされたハッキングノードを使って再訪することがよくあります。レベルデザインはしばしば「ロックイン」され、次のリフトやドアが利用可能になる前に、部屋のすべての敵を倒す必要があります。

> **💡 ヒント：** ミニゲーム中にオプションのハッキングノードを探しましょう。基本的なハッキングで即座にダメージを与えることもできますが、特殊なアイコンを通る長いルートを選ぶと、大規模な環境トラップや範囲攻撃を誘発できることがあります。

## FAQ

**Q: 『プラグマタ』はSwitch 2で4Kに対応していますか？**

A: いいえ、ゲームは60 FPSの目標を維持するために動的解像度スケーリングを使用しています。TVモードでは携帯モードよりも高い解像度に達しますが、ネイティブ4K画像は出力しません。

**Q: ダイアナだけでゲームをプレイすることはできますか？**

A: 現在のビルドに基づくと、プレイヤーはヒュー・ウィリアムズを操作し、ダイアナは背中でサポートキャラクターとして機能します。ただし、特定のハッキングやパズルシーケンスでは役割が入れ替わるセクションがあります。

**Q: Switch 2版のパッケージ版は発売されますか？**

A: はい、カプコンは2026年4月24日にパッケージ版とダウンロード版の世界同時発売を認めています。

**Q: このガイドで言及されているフレームレートの低下を修正するにはどうすればいいですか？**

A: フレームレートの低下を軽減する最も効果的な方法は携帯モードでプレイすることです。これにより、ハードウェアのVRR機能を活用して変動を滑らかにできます。TVモードでプレイする場合は、熱によるスロットリングを防ぐために本体の通気を良くしてください。
