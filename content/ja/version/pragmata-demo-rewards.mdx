## 隠しエレベーターエネミーのアンロック方法

入手が最も困難な報酬は、ダイアナの2枚目の絵です。これは、エレベーター上昇中の隠された遭遇戦に直結しています。ほとんどのプレイヤーは標準的な3体のボットに遭遇してそのまま進んでしまいますが、非常に特定の戦闘プロトコルに従った場合にのみ出現する4体目の「隠しエネミー」が存在します。

この遭遇を発生させるには、最初に出現する3体のエレベーターボットを、それぞれ**異なる武器**でとどめを刺す必要があります。もし2体以上のボットに対して同じ武器で最後の一撃を与えてしまうと、隠しエネミーは出現せず、2枚目の絵の条件を満たすことができません。

### 隠し要素アンロック・ステップバイステップガイド

1.  **1体目のボットと交戦:** どのような方法で体力を削っても構いませんが、最後の一撃は必ず**ピストル**で与えてください。
2.  **2体目のボットと交戦:** ターゲットを弱らせた後、武器を**ショックウェーブガン**に切り替えてとどめを刺します。
3.  **3体目のボットと交戦:** 残りのツールを使ってHPを削り、**ハック**または特定の環境インタラクションを使用して撃破します。
4.  **出現の確認:** 3体すべてを異なるフィニッシュ技で破壊すると、エレベーターが停止し、隠しエネミー「イントルーダー（侵入者）」ボットが出現します。

> **警告:** 範囲攻撃（AoE）を使用する際は注意してください。偶発的な爆発で2体のボットを同時に倒してしまった場合、ゲームはそれを同一のダメージソースとみなし、隠しエネミーの出現条件は失敗となります。

## 武器管理と戦闘戦略

**プラグマタ体験版報酬**を確実に手に入れるためには、体験版の戦闘システムのニュアンスを理解する必要があります。「マルチハック」能力は特に有用ですが、隠し要素をアンロックしようとする際には諸刃の剣となる可能性があります。

| 武器/アビリティ | 最適な用途 | とどめを刺す能力 |
| :--- | :--- | :--- |
| **ピストル** | 単体への精密射撃。 | **高** - 最初のキルでコントロールしやすい。 |
| **ショックウェーブガン** | クラウドコントロールと怯ませ。 | **中** - 高ダメージ。爆発半径に注意。 |
| **マルチハック** | 複数の敵に削りダメージを与える。 | **低** - ターゲットを弱らせるのに最適。 |
| **近接攻撃** | 弱った敵へのトドメ。 | **高** - コントロールされたキルに非常に信頼できる。 |

他の敵と共通のダメージソースでとどめを刺さない限り、条件は維持されます。3体のボットに同時にダメージを分散させるのではなく、1体ずつ集中して倒すのが良い戦略です。これにより、流れ弾や二次爆発による偶発的なキルを防ぐことができます。

## ダイアナのスケッチブック：主要な収集品

1枚目の絵は「スキャンと収集」メカニクスのチュートリアルとして機能するため、見逃すことはまずありません。しかし、2枚目の絵は体験版コンテンツの「真」のエンディングを象徴しています。隠されたエレベーターボスを倒すことで、月面に立つ謎の人物を描いたこのスケッチが自動的にアンロックされます。

> **ヒント:** ゲームを一時停止し、「データ」タブに移動することで、いつでも進行状況を確認できます。スケッチブックの2番目のスロットがまだロックされている場合は、体験版を再起動してエレベーターのチャレンジに再挑戦する必要があります。

## 体験版コンプリートのための高度なヒント

3つの武器の条件管理に苦戦している場合は、コミュニティで報酬確保のために使われている以下の高度な戦術を検討してください。

*   **削りダメージのコントロール:** 各ボットの体力の最初の90%はピストルを使用してください。これにより、ダメージ出力を完全にコントロールできます。ボットのヘルスバーが赤く点滅し始めてから、「フィニッシャー」武器に切り替えます。
*   **マルチハックの安全性:** 初期の噂に反して、マルチハック自体が複数の敵にトドメを刺さない限り、マルチハックによるダメージは条件を「壊す」ことにはなりません。敵のグループを弱らせるために安全に使用できます。
*   **素早いリスタート:** 誤って2体のボットを同じ武器で倒してしまったことに気づいたら、体験版を最後までプレイする必要はありません。メインメニューから「エレベーター接近」のチェックポイントを素早くロードして、やり直すことができます。

## FAQ

**Q: プラグマタ体験版報酬はどの難易度でも入手できますか？**

A: はい、ダイアナの絵は「Normal」と「Hard」の両方の難易度でアンロック可能です。ただし、プロトタイプスーツ・スキンは、利用可能な最高難易度で体験版をクリアすることに厳密に紐付けられています。

**Q: 隠しの絵を入手した後、保存するために体験版を最後までプレイする必要がありますか？**

A: はい。セーブデータに絵の情報が確実に更新されるよう、「Thank You For Playing」画面まで到達することを強くお勧めします。ボス戦の直後にアプリケーションを終了すると、進行状況が失われる可能性があります。

**Q: 体験版ではいくつの武器が利用可能ですか？**

A: エレベーターチャレンジでは、標準ピストル、ショックウェーブガン、ハックツールの3つの主要な武器タイプが利用可能です。隠しエネミーを出現させるには、これら3つすべてをフィニッシャーとして活用する必要があります。

**Q: すべてのロア端末を見つけることによる報酬はありますか？**

A: 体験版においてロア端末による特定の「アイテム」報酬はありませんが、すべてを収集することで、最終カットシーン中に主人公とダイアナの間の隠しダイアログがアンロックされます。
