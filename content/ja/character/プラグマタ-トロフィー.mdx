ついにカプコンのSF大作が発売されました。コンプリートを目指すプレイヤーにとって、**プラグマタ トロフィー**（Pragmata Trophy）リストは、やりごたえがありつつも達成感のある100%クリアへの道筋を示してくれます。荒廃した月面を探索する場合でも、謎の少女ダイアナを守る場合でも、効率的なプレイには各実績の条件を理解することが不可欠です。本ガイドでは、**プラグマタ トロフィー**の要件を深く掘り下げ、最難関の戦闘遭遇戦の戦略や、見つけにくい収集品をすべて見つけるためのヒントを提供します。

『プラグマタ』のトロフィーリストは、ストーリーの進行、高度な戦闘チャレンジ、そして細部まで行き届いた探索のバランスが取れています。切望されるプラチナトロフィーを獲得するには、ゲーム独自の「ハッキング」メカニクスをマスターし、特殊な装備モジュールをアップグレードし、クリア後のコンテンツ「アンノウン・シグナル（Unknown Signal）」を生き抜く必要があります。以下では、2026年版のリストをクリアするために、トロフィーをレアリティ別に分類し、具体的な手順を解説します。

## プラグマタ トロフィー：レアリティと内訳

具体的な要件に入る前に、実績リストの全体的な構成を確認しておくと役立ちます。他のカプコンの主要タイトルと同様に、本作もカジュアルなクリアからハードコアな極め込みまでを網羅した標準的な構成となっています。

| トロフィーランク | 数量 | 主な目的 |
| :--- | :--- | :--- |
| **プラチナ** | 1 | 他のすべてのトロフィーを獲得することで授与。 |
| **ゴールド** | 2 | メインストーリーの完了と「アンノウン・シグナル」のマスター。 |
| **シルバー** | 12+ | 戦闘実績（レッドゾーン）と収集のマイルストーン。 |
| **ブロンズ** | 25+ | ストーリーの節目、基本的なアップグレード、初回の特定アクション。 |

> **💡 ヒント：** ほとんどのブロンズトロフィーはストーリー進行に紐付いているため、取り逃すことはありません。ただし、シルバーの「ALL Bingo!」トロフィーには注意が必要です。これにはキャンペーンを通じて「キャビン・スタンプボード」に継続的に取り組む必要があります。

## メインストーリーと進行状況のトロフィー

実績リストの中核は、主人公ヒューが「データセンター」や月面の「シェルター」を旅する過程に紐付いています。これらの節目は、あなたの**プラグマタ トロフィー**獲得への道のりの骨格となります。

| トロフィー名 | レアリティ | 獲得条件 |
| :--- | :--- | :--- |
| **Encounter（邂逅）** | ブロンズ | 初めてシェルターに到達する。 |
| **Power Restored（電力復旧）** | ブロンズ | 月面施設の電力を正常に復旧させる。 |
| **Promise of Two（二人の約束）** | ゴールド | 難易度を問わずゲームをクリアする。 |
| **A Worthy Man（ふさわしき者）** | ゴールド | クリア後のミッション「アンノウン・シグナル」を正常にクリアする。 |

「アンノウン・シグナル」は、プレイヤーのスキルの究極の試練と広く見なされています。これはボスラッシュや高難易度の戦闘ガントレットとして機能し、スタッフロールの後にアンロックされます。これに備えるため、最初のプレイ中に「ファームウェア」と「カスタムモジュール」を最大化することに集中しましょう。

## 戦闘とスキルベースの実績

『プラグマタ』には、環境そのものがプレイヤーに対する武器となる高強度の汚染エリア「レッドゾーン」が登場します。これらのゾーンを制圧することは、**プラグマタ トロフィー**体験の重要な要素です。

### レッドゾーンの攻略
「腕に自信あり（シルバー）」トロフィーを獲得するには、レッドゾーンを制圧する必要があります。これは単に敵を倒すだけでなく、戦術的な移動、環境ハッキング、そして仲間の保護を組み合わせる必要があります。

1. **ハッキングノードを優先する**：スキャナーを使用して、敵のタレットを逆利用できるノードを特定します。
2. **モジュールの相乗効果**：ダイアナとの「シンクロ率」を高めるモジュールを装備し、強力な連携攻撃を繰り出しましょう。
3. **クラウドコントロール**： 「リード・アース」キャニスターを使用して、迫りくるロボットの脅威に対して一時的な障壁や罠を作成します。

### ハッキングと装備の極意
ゲーム後半を生き抜くには、装備への投資が不可欠です。「最強の武器」や「最大モジュール」のトロフィーは、月面の廃墟やアースメモリの断片で見つかる素材を集めることで獲得できます。

> **注意：** 序盤で間違ったモジュールをアップグレードすると、中盤の難易度が大幅に上がる可能性があります。まずは「ハッキング速度」と「エネルギー効率」のモジュールに集中し、プレッシャーのかかる状況下でも環境と対話できるようにしましょう。

## 探索と収集ガイド

**プラグマタ トロフィー**リストの大部分は、隠されたアイテムの発見に費やされます。ゲームの世界には、人類文化の名残であり、伝承（ロア）やアップグレードポイントを提供する「アースメモリ」の断片が散りばめられています。

| 収集アイテム | 目的 | 関連トロフィー |
| :--- | :--- | :--- |
| **アースメモリ** | 世界観の補完とアップグレードポイント | メモリコレクター (シルバー) |
| **ミニキャビン** | ファストトラベルと休憩所 | 第二の我が家 (ブロンズ) |
| **スタンプボード** | ミニゲームの進行 | **ALL Bingo!** (シルバー) |
| **ハッキングノード** | 戦闘の優位性 | マスターデコーダー (ブロンズ) |

すべての「ミニキャビン」を見つけることは、トロフィーだけでなく生存のためにも不可欠です。これらのキャビンはチェックポイントとして機能し、装備の入れ替えやキャビン・スタンプボードの操作が可能です。ボードをすべて埋めると「ALL Bingo!」トロフィーを獲得できますが、これには特定のセクターで「パーフェクトハッキング」や「ノーダメージ戦闘」などの特定のタスクをこなす必要があります。

## プラチナトロフィー獲得のための高度な戦略

**プラグマタ トロフィー**のプラチナを獲得するには、おそらく1周半のプレイが必要になります。「二人の約束」はどの難易度でも獲得できますが、「アンノウン・シグナル」を攻略するには、ニューゲームプラス（NG+）で見つかるステータスや装備が必要になることが多いです。

1. **1周目（ストーリー重視）**：物語を楽しみ、できるだけ多くのアースメモリを見つけることに集中しましょう。この段階では、戦闘の「Sランク」トロフィーを気にする必要はありません。
2. **回収フェーズ**：ラスボスに挑む前に、以前のセクターを再訪して残っているレッドゾーンを制圧し、キャビン・スタンプボードを完成させましょう。
3. **ポストゲーム/NG+**：フルアップグレードした装備で「アンノウン・シグナル」に挑みます。最後の試練をクリアすると、「PRAGMATA」プラチナトロフィーが解除されます。

ゲームのメカニクスや公式アップデートの詳細については、[プラグマタ公式サイト](https://www.capcom.co.jp/pragmata/)で最新の開発者インサイトやパッチノートを確認してください。

## FAQ

**Q: 取り逃す可能性のあるトロフィーはありますか？**

A: 現在、ほとんどのトロフィーはチャプターセレクトやクリア後に獲得可能です。ただし、特定の「キャビン・スタンプボード」のタスクは、特定のストーリー展開中に完了する方が簡単です。セクター固有のチャレンジに遅れないよう、新しいミニキャビンに到達するたびにスタンプボードを確認することをお勧めします。

**Q: 最も獲得が難しいトロフィーは何ですか？**

A: 一般的に「ふさわしき者（ゴールド）」が最難関とされています。これには、最も攻撃的な敵AIが登場し、回復リソースが限られている「アンノウン・シグナル」をクリアする必要があります。この実績には、適切なモジュールの最適化が不可欠です。

**Q: 難易度設定はトロフィーリストに影響しますか？**

A: 「二人の約束」トロフィーは、イージーを含むどの難易度でも獲得できます。ただし、高難易度が独自の報酬を与えるかどうかは明記されていませんが、通常、高難易度でプレイするとより多くのアップグレード素材を入手でき、「最大アップグレード」トロフィーの獲得を早めることができます。

**Q: 収集トロフィーにはいくつのアースメモリが必要ですか？**

A: データセンターと月面に50個のアースメモリの断片が散らばっています。基本的なアップグレードにはすべて必要というわけではありませんが、「メモリコレクター」トロフィーには、すべてのユニークな断片を100%発見する必要があります。
