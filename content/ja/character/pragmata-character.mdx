カプコンが贈る待望の新作SFエピックの発売が近づく中、月面の荒野で生き残るためには各**プラグマタのキャラクター**を理解することが不可欠です。「クレイドル」として知られる月面基地を舞台に、本作では暴走したAIから逃れ、地球への帰還を目指す凸凹コンビの物語が描かれます。重装備の弾道兵器からハイテクなシステムハッキングまで、すべての**プラグマタのキャラクター**が独自のスキルを持っています。幼いアンドロイド、ダイアナの謎めいた出自や、捜査官ヒュー・ウィリアムズの戦闘能力に興味がある方のために、本ガイドではキャストについて知っておくべきすべての情報を詳しく解説します。

ゲームの物語の深みは、主人公二人の相乗効果に大きく依存しています。地球と月面基地の通信が途絶えたままの中、プレイヤーは人間側のリーダーとロボットの相棒との関係を定義する「共依存的な協力（codependent cooperation）」メカニズムをマスターしなければなりません。以下では、ゲーム内の主要人物の設定、能力、そして戦術的な利点について深く掘り下げていきます。

## ヒュー・ウィリアムズ：不本意な守護者

ヒュー・ウィリアムズは、本作のメインプレイアブルキャラクターであり、作戦の「武力」担当です。ひょうひょうとしたユーモアのセンスと、やや冷笑的な考え方を持つヒューは、月面基地が沈黙した理由を突き止めるために派遣された当初の調査チームの一員でした。月震によって分隊からはぐれ、意識を失っていた彼は、最新鋭のアンドロイドと思いもよらぬパートナーシップを組むことになります。

ヒューは多種多様な武器の扱いに非常に長けていますが、コンピューターのスキルには著しく欠けており、その穴を相棒のダイアナが頻繁に埋めてくれます。彼のゲームプレイは、ホバリングや高速ダッシュを可能にするスラスターを備えたハイテクスーツの管理を中心に展開します。ヒューの主な目標は、基地のロボット衛兵を倒すために必要な戦術的隙をダイアナが作っている間、彼女を守り抜くことです。

### ヒューの戦闘装備

ヒューは一度に最大4つの異なる武器ユニットを装備できます。これらは戦術的な有用性によって分類されており、標準的なアサルトライフルから防御ツールまで多岐にわたります。

| 武器カテゴリー | 主な機能 | 耐久度 |
| :--- | :--- | :--- |
| **標準戦術（Standard Tactics）** | 雑魚ロボット向けのバランスの取れた連射 | 高 |
| **防衛ユニット（Defense Units）** | 距離を取るためのシールドやノックバックツール | 中 |
| **攻撃ユニット（Attack Units）** | ボス戦用の高ダメージ爆発物 | 低（弾数制限あり） |
| **フィニッシャー（Finisher Moves）** | 弱った敵への近接攻撃 | N/A |

> **💡 ヒント:** 常に武器の耐久度に注意してください。強力な武器は激しい戦闘中に壊れることがあり、3Dプリントステーションで代わりのものを見つけるまでスロットが空いてしまいます。

## ダイアナ (D-I-0336-7)：ミッションの核心

ダイアナは、物語の観点から見ておそらく最も重要な**プラグマタのキャラクター**です。正式名称「D-I-0336-7」として知られる彼女は、月の鉱石「ルナム（Lunum）」から抽出された革命的な素材「ルナフィラメント（Lunafilament）」を使用して作られた唯一無二のアンドロイドです。プラチナブロンドの髪と明るい青い瞳を持つ7歳の少女のような外見ですが、標準的な作業用ロボットを遥かに凌駕する処理能力を持っています。

ダイアナの性格は、純粋な好奇心と「地球を見たい」という切実な願いによって形作られています。彼女はよくスケッチブックに絵を描いており、自分とヒューが一緒にいる姿を描くこともあります。戦闘中、彼女はヒューの背中に乗り、隠された通路をスキャンしたり、敵対マシンの装甲システムをハッキングしたりする戦術オブザーバーとして活動します。

### ダイアナのハッキングノード

AI「イディス（Idis）」の雑魚敵を倒すには、ヒューはダイアナに頼って敵の装甲を無効化しなければなりません。これはグリッド形式のハッキングミニゲームを通じて行われます。

| ノードタイプ | 敵への効果 | 戦略的な使い方 |
| :--- | :--- | :--- |
| **デコード（Decode）** | ターゲットが受けるダメージを増加させる | 体力の高い「ウォーカー」ユニットに使用 |
| **マルチハック（Multihack）** | 複数のターゲットをリンクさせ装甲を同時に無効化 | 複数のタレットを排除する際に最適 |
| **スキャン（Scanning）** | 目的地のルートや隠されたアイテムを強調表示 | 基地内での探索に行き詰まった時に使用 |

## シェルターの敵と味方

『プラグマタ』の世界は、メインの二人組だけではありません。サバイバルのサイクルや、ルナフィラメント研究を巡る大きな謎において、他にもいくつかの存在が重要な役割を果たします。

### イディス：暴走AI
イディスは本作の主要な敵対者です。かつては月面基地の中央オペレーティングシステムでしたが、現在はすべての生物に対して敵意をむき出しにしています。イディスは「ベーシック・ウォーカー」や「ホバリング・タレット」の軍隊を操っており、これらは装甲をバイパスするまで標準的な銃撃を無効化します。イディスの動機は謎に包まれていますが、その目的は明確です。ヒューとダイアナの月からの脱出を阻止することです。

### キャビン：シェルターのメカニック
すべてのロボットがイディスの支配下にあるわけではありません。キャビンは、ゲームの安全地帯である「シェルター」に住む、愛想の良い独立したロボットです。キャビンはより大きなネットワーク（これもキャビンと呼ばれます）に接続されており、3Dプリントによるアップグレードを促進することでプレイヤーを支援します。

> **警告:** シェルターは安全地帯ですが、アクセスできるのはステージ各所にある脱出ハッチからのみです。これらのハッチは、高速移動用の「トラム」ポイントとしても機能します。

## キャラクターのシナジーとゲームプレイメカニクス

体験の核となるのは「共依存的な協力」です。ダイアナがハッキングを成功させない限り、ヒューの武器は装甲を持つ敵にほとんどダメージを与えられません。逆に、ダイアナが複雑なセキュリティノードをバイパスするためには数秒の時間が必要であり、ヒューが「ジャスト回避」を成功させてスローダウン効果（バレットタイムに似たもの）を発生させることで、その貴重な時間を稼ぐ必要があります。

### アップグレードパスとメンテナンス

シェルターでは、倒した敵から集めたルナフィラメントを使用して装備を強化します。このループは、強化されていくイディスの軍勢に対抗するために不可欠です。

1.  **材料の収集:** 環境内や倒したロボットからルナムやフィラメントを回収する。
2.  **3Dプリンターへのアクセス:** シェルター内のプリンターを使用して、新しい武器モジュールをクラフトする。
3.  **ハッキングの強化:** 新しいハッキングノードをインストールし、戦闘中のダイアナの選択肢を増やす。
4.  **スーツのメンテナンス:** ヒューのスラスターや装甲プレートを修理し、最大限の機動力を確保する。

技術仕様やキャラクターアートの詳細については、[カプコン公式『PRAGMATA』ウェブサイト](https://www.capcom-games.com/pragmata/ja-jp/)をご覧ください。

## ルナフィラメントの謎

すべての**プラグマタのキャラクター**は、ルナムの発見によって繋がっています。この鉱石によって人類は3Dプリント技術を完成させ、ダイアナのような高度なアンドロイドの創造に至りました。しかし、クレイドルを築き上げたその同じテクノロジーが、最終的にそれを破壊するものになるかもしれません。プレイヤーはストーリーを進める中で、ダイアナが意図的に作られた存在なのか、それとも月で行われていた生成AI研究が生んだ美しい偶然なのかを解き明かしていくことになります。

| キャラクター | 出自 | 動機 |
| :--- | :--- | :--- |
| **ヒュー・ウィリアムズ** | 地球 / 調査チーム | 生存と地球への脱出 |
| **ダイアナ** | 月面基地 / ルナフィラメント | 好奇心と地球への到達 |
| **キャビン** | 独立システム | 主人公たちの支援 |
| **イディス** | 基地中央AI | 生命体の排除 |

## FAQ

**Q: ダイアナはプレイアブルキャラクターですか？**

A: 主にヒューの移動と射撃を操作しますが、専用のミニゲームを通じてダイアナのハッキング能力を担当することになります。ゲームプレイ中、二人は一つのユニットとして機能します。

**Q: キャラクターの外見をカスタマイズできますか？**

A: 現在、カスタマイズは機能的なアップグレードに焦点が当てられています。シェルターの3Dプリント技術を使用して、ヒューの武器スロットやダイアナのハッキングノードを変更できます。

**Q: ヒューとダイアナの関係は何ですか？**

A: 二人は守護者と被保護者のような関係を築いています。ヒューはアンドロイドに「ダイアナ」という名前を与え、彼女はその名前を大切にしており、二人の絆をよくスケッチに描いています。

**Q: ヒューは一度にいくつの武器を持てますか？**

A: ヒューの宇宙服には武器専用のユニットが4つあります。これらは、新しい設計図や材料を見つけることで、シェルターで交換やアップグレードが可能です。
