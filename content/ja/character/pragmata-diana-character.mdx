## ダイアナというキャラクターを理解する

**ダイアナ**を定義するのは「二面性」です。彼女は過酷な環境に身を置くか弱い子供でありながら、重武装の主人公をも凌駕する技術的能力を備えています。初期のゲームプレイデモから、ダイアナが物語の舞台となる月面施設と深い関わりがあることは明らかです。彼女はしばしば自分の過去を暗示し、「ここで働いていた」ことや、施設の現在の惨状が「自分のせい」である可能性さえ示唆します。

ダイアナの主な役割は「サポートスペシャリスト」です。主人公が重火器や物理的な防御を担当する一方で、ダイアナはデジタルな戦場を管理します。彼女は端末とインターフェースし、セキュリティのロックダウンを解除し、リアルタイムの操作データを提供して脅威を特定します。

| 属性 | 説明 |
| :--- | :--- |
| **役割** | サポートスペシャリスト / 物語の主軸 |
| **主なスキル** | 高度なハッキングとシステム上書き |
| **象徴的なアイテム** | スケッチブック |
| **戦闘スタイル** | 間接的サポート / 主人公の強化 |
| **主要な繋がり** | 「ドクター・アース（Dr. Earth）」およびAI「エディス」との関連の可能性 |

> **警告：** 戦闘シークエンス中はダイアナから離れすぎないようにしてください。彼女はある程度自衛が可能ですが、特定のコンテキストアクションや電力復旧を発動させるには、彼女の近くにいる必要があります。

## ダイアナのゲームプレイメカニズムとサポートの役割

戦闘の最中、**ダイアナ**は戦力を倍増させる「フォース・マルチプライヤー」として機能します。プレイヤーは、ダイアナのハッキングが有効なタイミングに合わせて攻撃を調整することを学ぶ必要があります。例えば、主人公がロックされたエリアや緊急ロックダウンで保護された端末に遭遇した場合、機能を回復できるのはダイアナだけです。

### ハッキングと環境への干渉
ダイアナのハッキングは単なる演出ではなく、核となるゲームプレイメカニズムです。「Sketchbook」デモでは、彼女が発電所の複数の階層にわたってロックダウンを解除する様子が見られます。このプロセスでは、プレイヤーがメイン電力端末を操作するダイアナを守り抜く「マニュアルリブート」が発生することがよくあります。

| アクション | ゲームプレイへの効果 |
| :--- | :--- |
| **端末ハック** | 緊急ロックダウンを解除し、進行ルートを確保する。 |
| **環境スキャン** | インタラクティブなオブジェクトや隠された端末を強調表示する。 |
| **電力復旧** | エネルギーレベルを増加させ（30%から100%）、施設機能を有効にする。 |
| **リフト起動** | エレベーターをハッキングし、垂直方向の移動を可能にする。 |

### オーバードライブ・プロトコル
ダイアナの能力の中で最も強力なツールの一つが**オーバードライブ・プロトコル**です。これが発動すると、主人公の戦闘効率が大幅に向上し、「より強力に、より速く」敵を撃破できるようになります。これは、暴走したと思われる「ウォーカー」や大型のロボットの脅威に対して特に有効です。

## ダイアナの出自にまつわる謎

**ダイアナ**を取り巻く物語は謎に包まれています。様々な会話の中で、ダイアナは自身の「秘密の」仕事や、「ドクター・アース」に連絡を取りたいという願望について言及します。これは、彼女が人工的な構成体であるか、あるいは月面運用のために高度に強化された人間であることを示唆しています。

主人公とダイアナを「未登録のゲスト」と見なすAI、**エディス**の登場が、彼女の物語をさらに複雑にしています。エディスのダイアナに対する敵意は、ダイアナが離反した要素であるか、あるいはシステムから「登録抹消」された元従業員である可能性を暗示しています。

### 物語の重要な手がかり
1.  **スケッチブック：** ダイアナはゲーム中、スケッチブックの中で秘密のプロジェクトに取り組んでいる姿が見られます。この本には、ミッションに不可欠な設計図、記憶、あるいはコードが含まれている可能性があります。
2.  **「私のせい」という発言：** ダイアナはボットや施設の状態に罪悪感を抱いており、彼女が不注意にも機械の「暴走」行動を引き起こした可能性を示しています。
3.  **地球との繋がり：** 両キャラクターとも地球への連絡を切望しており、月面コロニーが孤立し、完全な通信遮断状態にある可能性を示唆しています。

| エンティティ | ダイアナとの関係 |
| :--- | :--- |
| **主人公** | 月面ミッションにおける保護者でありパートナー。 |
| **エディス** | ダイアナのアクセス権を拒否する敵対的なAI。 |
| **ドクター・アース** | ダイアナが緊急に連絡を取る必要がある謎の人物。 |
| **ボット** | 本来人間を攻撃しないはずが、暴走してしまった機械たち。 |

## 戦闘シナジー：ダイアナとの連携

『PRAGMATA』で成功を収めるには、主人公の重装甲と**ダイアナ**の戦術的バフの相乗効果（シナジー）をマスターしなければなりません。戦闘はしばしば「迷路の中のネズミ」のような状況になり、ダイアナがドアを開けたりシステムを再起動したりするのを待つ間、敵の猛攻を凌ぐ必要があります。

### シナジーのための戦術的ヒント
*   **ショットをチャージする：** ダイアナが「こっちに撃って！（shooting to me）」と叫んだら、耐久力の高いターゲットに対して効果的なウォーカーのチャージショットなどを使用しましょう。
*   **防御的なポジショニング：** ダイアナがリフトや端末をハッキングしている間、彼女は無防備になります。彼女と「侵入者検知（Intruder Detected）」ゾーンの間に身を置いて守りましょう。
*   **音声合図を聞き逃さない：** ダイアナはしばしば「今撃って！（Shoot now!）」や「任せて！（On it!）」と叫び、敵の防御が下がった瞬間やシステムの準備が整ったことを知らせてくれます。

> **ヒント：** 道に迷ったときは、スキャン機能（Qキーまたは対応するボタンで起動）を使用してください。ダイアナが進むために必要な特定の端末をハイライトしてくれることがよくあります。

## ビジュアルデザインとキャラクターの進化

**ダイアナ**のビジュアルデザインは、2020年の初公開以来、微妙な変化を遂げてきました。2026年時点のデザインでは、月面でのサバイバル精神が強調されています。彼女の服装は近未来的な技術と子供らしい安心感が混ざり合っており、人間性と『PRAGMATA』の冷徹な機械世界の架け橋としての役割を象徴しています。

彼女のアニメーションは特に注目に値し、子供のような好奇心と、テクノロジーに対するエキスパートレベルの習熟度が融合しています。スケッチブックに秘密を「隠して」いるときも、自信を持ってセクターガードのプロトコルを上書きしているときも、彼女の動きは機械の世界で大人にならざるを得なかった子供の物語を語っています。

## FAQ

**Q: ダイアナは『PRAGMATA』のプレイアブルキャラクターですか？**

A: 主に操作するのは装甲を纏った主人公ですが、**ダイアナ**にコマンドを出したり、ハッキングやオーバードライブ・プロトコルなどの特殊能力を発動させたりすることができます。彼女は伝統的な意味での第2のプレイアブルキャラクターというよりは、常に寄り添うパートナーとして機能します。

**Q: ゲームプレイに出てくる「スケッチブック」とは何ですか？**

A: スケッチブックは物語の重要な仕掛けであり、UI要素でもあります。ダイアナはこれを使って自身の「秘密の」仕事を記録しており、ゲームの結末を握る鍵となる可能性があります。また、カプコンがリリースした主要なゲームプレイデモの名称でもあります。

**Q: ダイアナをアップグレードすることはできますか？**

A: はい、ゲームを進行させて施設の電力をより多く復旧させることで、ダイアナのハッキング速度やオーバードライブ・プロトコルの効力を高めることができます。これは、ゲーム後半で遭遇する「セクターガード」プロトコルに対処するために不可欠です。

**Q: エディスはダイアナとどのような関係ですか？**

A: エディスは施設のセキュリティAIです。彼女は**ダイアナ**と主人公を侵入者と見なしています。ダイアナのハッキングとエディスのセキュリティ対策の対立が、ゲーム内の多くのパズルや戦闘の基礎となっています。
