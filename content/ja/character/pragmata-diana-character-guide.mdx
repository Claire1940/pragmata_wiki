## ダイアナとは？「子供」の殻に隠されたアンドロイド

ダイアナ（個体識別番号：D-I-0336-7）は、「ルナフィラメント」と呼ばれる特殊な素材を使用して作られた唯一無二の「プラグマタ」アンドロイドです。長い髪を持つ少女の姿をしていますが、決して脆弱ではありません。彼女は装甲服に身を包んだ宇宙飛行士ヒュー・ウィリアムズによって発見され、彼から「ダイアナ」という名前を与えられました。彼女は自身のアイデンティティを形成していく中で、この名前を特に大切にしています。

物語において、ダイアナは人間の直感と機械の効率性を結ぶ架け橋のような存在です。彼女は好奇心旺盛で学習が早く、荒廃した月面環境に対して無邪気な驚きを持って反応することがよくあります。しかし、彼女の主な機能は「ヘビーヒッター」としての役割です。ヒューが物理的な移動や基本的な防御を担当する一方で、ダイアナはコロニーの暴走したセキュリティボットを無力化するために必要な技術的な「重労働」を担います。

## プラグマタ ダイアナ キャラクターガイド：核となる戦闘メカニクス

『プラグマタ』の戦闘は、共生的な「デュアルキャラクターシステム」を中心に構築されています。単一のヒーローを操作するのではなく、一方が欠ければ機能しないパートナーシップを管理します。この**プラグマタ ダイアナ キャラクターガイド**では、アクションが加速する中でも冷静な思考を必要とする「エイム、ハック、デストロイ」ループに焦点を当てます。

### エイム、ハック、デストロイのループ
月面基地の敵の多くはシールドや装甲を装備しており、ヒューのライフルによる標準的な射撃は効果が薄い「ピーシューター効果（豆鉄砲現象）」に陥ります。これらの脅威を打ち破るには、以下の手順を踏む必要があります。

1.  **エイム（照準）：** ヒューの照準レティクルを使用してターゲットをロックオンします。
2.  **ハック（ハッキング）：** ダイアナのグリッドベースのハッキングインターフェースを起動し、敵の内部回路や弱点を露出させます。
3.  **デストロイ（破壊）：** 脆弱なウィンドウが開いたら、ヒューの高火力バーストやダイアナのシンクアタックで一気に畳みかけます。

### グリッドベースのハッキングインターフェース
ダイアナがハッキングを開始すると、ターゲット上にグリッド状のインターフェースが表示されます。コントローラーのボタンを使用して、特定のノードを通過するルートを構築する必要があります。パズルの複雑さはゲームの進行に合わせて増していきます。より多くのノードを経由してルートを延長することで、脆弱な状態を長く維持したり、ヒューがより大きなダメージを与えられるようになったりします。

> **💡 ヒント：** ハッキングミニゲームを焦ってはいけません。ハッキング中は無防備になりますが、長いノードルートを完成させることで戦闘を瞬時に終わらせ、結果的に体力を温存できることが多々あります。

## ダイアナのハッキングツールキット：ノードとアップグレード

月面基地を探索する中で、データフラグメントや「ルナスクリップ」を収集し、ダイアナの戦闘サブルーチンをアップグレードできます。彼女のハッキングツールキットは敵に対してだけでなく、環境を操作して有利に進めるためにも使用できます。

### 消耗型ハッキングノード
ダイアナは、最も強力なアビリティを実行するために消耗型のノードを使用します。これらは豊富に入手可能ですが、長時間のボス戦ではリソース管理が重要になります。

| ノードタイプ | アビリティ説明 | 戦術的な影響 |
| :--- | :--- | :--- |
| **マルチターゲット** | 最大5体の敵を同時にロックオン | ドローン群に対するクラウドコントロールに不可欠。 |
| **シールドブレイク** | 重厚な機械装甲を即座にバイパス | 「ハッカー」クラスの敵をスタン状態に追い込む。 |
| **オーバードライブ** | 短時間、ダイアナの出力を強化 | 大規模な範囲攻撃「シンクアタック」を可能にする。 |
| **システムハイジャック** | セキュリティボットを味方に転向させる | 一時的な囮やタンク役を確保する。 |

### 環境ハッキング
ダイアナのインターフェースにより、コロニーのインフラと相互作用できます。これには以下が含まれます：
*   **ミサイルのリダイレクト：** 飛来する発射体の軌道を反転させ、発射元に送り返す。
*   **トラップの無効化：** 床に設置された停滞ネットやレーザーグリッドを停止させる。
*   **ロックのバイパス：** 緊急隔壁を開き、新しいエリアへの到達や圧倒的な敵の波からの脱出を可能にする。

## シナジーとデュアルプレイ：ヒューとダイアナの絆の管理

**プラグマタ ダイアナ キャラクターガイド**の重要な柱の一つは、「シンクエネルギー」の管理です。このエネルギーは、世界を探索し戦闘を成功させることで蓄積されます。シンクメーターが満タンになると、ヒューの物理的な強さとダイアナの技術的な卓越性を組み合わせた破壊的な連携技を繰り出すことができます。

### シンクアタックのバリエーション
実行されるシンクアタックの種類は、ヒューの現在の武器装備とダイアナのアクティブなノードによって決まります。

| 武器の組み合わせ | シンクアタックの結果 | 有効な敵... |
| :--- | :--- | :--- |
| **グレネードランチャー** | **エクスプローシブチェーン** - ダイアナが手榴弾をハックし、連続爆発させる。 | 地上型の「ワーカー」の大群。 |
| **ロングレンジ・ショックウェーブ** | **プレシジョンピアス** - あらゆる装甲を無視する集中ビーム。 | ボスの弱点や「コロッサス」。 |
| **ステイシスネット** | **タイムフリーズ** - 広範囲の全敵を10秒間停止させる。 | オープンアリーナでの高圧的な敵の波の管理。 |

> **⚠️ 警告：** ダイアナがヒューから長時間離れると、彼女が通常管理しているスーツの補助システムが故障し、急速な酸素枯渇と体力減少を招く可能性があります。

## 月面基地での高度な戦術アドバイス

『プラグマタ』で真に秀でるためには、本作を伝統的なラン＆ガンシューティングではなく、リアルタイムの「問題解決演習」として捉える必要があります。

1.  **ヒューを破城槌として使う：** ヒューは威圧感を与えるように設計されています。ダイアナが強力なハッキングシーケンスをチャージしている間、彼を使って敵の注意（ヘイト）を引きつけましょう。
2.  **視覚的な合図に注目：** ダイアナの行動は彼女の「センチエンスレベル（意識レベル）」によって変化します。非戦闘時に彼女が鼻歌を歌ったり、周囲に絵を描き始めたりしたら、それは通常、シンクメーターが満タンで次の戦闘の準備ができていることを示しています。
3.  **ダイアナのアップグレードを優先する：** ダイアナは高ダメージを与える主要なソースであるため、「ルナスクリップ」はヒューのアーマーを強化する前に、まず彼女の戦闘モジュールに費やすべきです。
4.  **冷静さが鍵：** デジタルで再現されたタイムズスクエアのような広いエリアでは、多方向から攻撃を受けます。冷静さを保ち、ヒューのデコイで敵を分散させ、ダイアナのシステムハイジャックで敵の数を減らしましょう。

## FAQ

**Q: 『プラグマタ』でダイアナを直接操作することはできますか？**

A: 移動や基本的な戦闘は主にヒューを操作しますが、多くのシーケンスでダイアナのハッキングインターフェースや特殊武器システムを直接制御することができます。このゲームは、両方のキャラクターを同時に管理する「デュアル主人公」体験として設計されています。

**Q: このガイドにおけるダイアナの名前の由来は何ですか？**

A: ゲーム内のロア（設定）では、ヒューがローマ神話の狩猟の女神にちなんで名付けたと唆唆されています。これは、彼女の小さく無邪気な外見に反して、チームの主要な戦闘員であり「ヘビーヒッター」である役割を暗示しています。

**Q: ダイアナとヒューのコンビで協力プレイ（Co-op）はできますか？**

A: 2026年4月の発売時点では、『プラグマタ』は完全にシングルプレイヤー専用の体験です。開発者は、ヒューとダイアナの絆とメカニカルな相乗効果は物語の中心であり、一人のプレイヤーの視点から体験するのが最適であると強調しています。

**Q: ハッキングノードは一度使うと完全になくなりますか？**

A: ハッキングノードは消耗品ですが、月面基地の至る所で豊富に入手できます。ゲームの最後まで温存するのではなく、さまざまな戦闘戦略を試すために自由に使用することが推奨されています。
