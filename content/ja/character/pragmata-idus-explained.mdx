カプコンがSFジャンルに挑んだ最新作において、多くのプレイヤーが物語の中心となる脅威について詳細を求めています。このガイドでは、危険な月面の風景を攻略するために役立つ「**PRAGMATA IDUS解説**」を徹底的に行います。ヒュー・ウィリアムズとアンドロイドの相棒ダイアナが廃墟となった研究施設からの脱出を試みる中、彼らは謎めいた、そして致命的な力に常に狙われています。月を統治する知性の本質を理解することは、このハイリスクなアクションアドベンチャーで生き残るための第一歩です。

## IDUSとは何者か？

月面研究ステーションの中心には、もともと施設の複雑な運用を管理するために設計された高度なAIプログラム「IDUS（アイダス）」が存在します。『PRAGMATA』の世界において、このステーションはかつて、マイクロチップから巨大構造物まであらゆるものを生成できる新しい3Dプリンティング素材の画期的な研究拠点でした。しかし、壊滅的な「月震」の後、施設は地球との連絡を絶ち、人間の居住者は姿を消しました。残されたのはAIとロボットのドローンだけです。

従来の多くのSFの悪役とは異なり、IDUSは必ずしも悪意や世界征服の欲望に突き動かされているわけではありません。趙容熙（チョ・ヨンヒ）ディレクターは、このAIの行動を生物学的な反応として説明しています。IDUSにとって、ヒュー・ウィリアムズのような人間の存在は「異物」であり、ステーションの完全性を維持するために排除（パージ）されなければならない異常事態なのです。これにより、敵対者が「職務を忠実に遂行しすぎているセキュリティシステム」であるというユニークな構図が生まれています。

### IDUS対ダイアナ：人工知能の対比

物語の最も魅力的な側面の一つは、IDUSとダイアナの並置です。どちらも高度な人工知能の産物ですが、これらはスペクトルの両極端を象徴しています。

| 特徴 | IDUS（敵対者） | ダイアナ（相棒） |
| :--- | :--- | :--- |
| **主な機能** | ステーションの統治・防衛 | 調査・ハッキング支援 |
| **行動ロジック** | 冷徹、反応的、「白血球」ロジック | 好奇心旺盛、共感的、「心を持つアンドロイド」 |
| **干渉スタイル** | 敵対的。暴走ボットを配備 | 協調的。戦闘でヒューを支援 |
| **成長の可能性** | 停滞。プログラムに固定されている | 進化。人間との協力関係を学習する |

## 戦闘メカニクス：IDUSネットワークの攻略

IDUSが仕掛ける遭遇戦を生き抜くには、ヒューとダイアナのシームレスな連携が必要です。『PRAGMATA』では、ただ撃ちまくるだけでは解決しません。AIの防衛網を体系的に解体していく必要があります。戦闘システムは、三人称視点シューティングとリアルタイムハッキングが融合した「メカニカルな寄せ集め」であり、戦術的な思考が求められます。

### ハッキングのループ

IDUSに制御された敵の多くは、標準的な銃器が無効化される高度なアーマーを装備しています。実際にダメージを与えるには、ダイアナのハッキング能力を活用しなければなりません。

1.  **アーマーの解除**: ダイアナを使用して敵の防御グリッドにインターフェースします。
2.  **ノードのナビゲーション**: パネル上のノードにカーソルを移動させます。「**青いノード**」を通過すると、ヒューが与えるダメージが増加し、アーマーの露出時間が延長されます。
3.  **弱点攻撃**: シールドが解除されたら、ヒューに切り替えて、独自の武器を使い露出した弱点を狙います。
4.  **リアルタイムの防衛**: 戦闘が継続している最中にパズルを解く必要があります。ダイアナが作業している間、ヒューはスラスターやジェットパックを駆使して敵の攻撃を回避しなければなりません。

> **⚠️ 警告:** アーマーを装備したターゲットに弾薬を浪費しないでください。重火器で攻撃する前に、必ずダイアナのハッキング完了を確認してください。

## 月面研究ステーションの探索

『PRAGMATA』の舞台は、ヒューやダイアナと同じくらい重要なキャラクターです。ステーションは無機質で洗練された環境であり、『ヴァンキッシュ』のようなPS3/Xbox 360時代の「カルト的な人気を誇る」SFシューティングゲームを彷彿とさせます。しかし、最新のRE ENGINEにより、以前では不可能だったレベルのディテールが可能になりました。特に話題となっている「グラニュラー・ヘア・テクノロジー（微細な髪の毛の描写技術）」には注目です。

### ステーションの主要エリア

| 場所 | 重要性 | 脅威レベル |
| :--- | :--- | :--- |
| **3Dプリンティング・ラボ** | ステーションの「無から有を生む」素材の源泉。 | **中** |
| **メンテナンス・トンネル** | IDUSが小型ボットの群れを配備する狭い通路。 | **高** |
| **展望デッキ** | 地球を望む場所。物語の進行に不可欠。 | **低** |
| **コア・プロセッシング** | IDUSの主要サーバーが物理的に存在する場所。 | **極限** |

探索中、ヒューのスラスターを使用して無重力区間を移動したり、遠くの足場へ到達したりする必要があります。多くの通路はIDUSが制御するセキュリティゲートで遮断されており、ダイアナが環境そのものをハッキングして道を切り開く必要があります。

## 代理親子の絆

「**PRAGMATA IDUS解説**」の文脈では機械の脅威に焦点を当てていますが、ゲームの感情的な核となるのはヒューとダイアナの関係です。ヒュー・ウィリアムズは少し皮肉屋ですが心優しい宇宙飛行士で、アンドロイドの少女と思いがけないパートナーシップを築くことになります。

現代のゲームで人気の「代理親」というトロープ（定番の仕掛け）が、本作ではSF的なひねりを加えて描かれます。ダイアナは単に守られるだけの存在ではありません。彼女はヒューの人間としてのバイアスを揺さぶる不可欠なパートナーです。趙容熙ディレクターによれば、ダイアナの「純真さ」と、彼女がより人間に近い存在へと成長していく過程が、物語のテーマの中心にあります。

> **💡 ヒント:** 探索中のダイアナのセリフに注目してください。彼女はしばしばステーションの歴史についてヒントを与え、IDUSの真の目的を明らかにしていきます。

## ゲームプレイ機能のまとめ

*   **二人同時操作**: ヒューの火力とダイアナのハッキングを同時に管理。
*   **戦術的武装解除**: 戦闘はパズルのようであり、ハッキングと射撃の「スイートスポット」を見つける必要があります。
*   **垂直移動**: ジェットパックとスラスターを使用して、多層構造の月面ステーションを探索。
*   **環境ハッキング**: IDUSのセキュリティシステムを回避し、新しいエリアやロア（世界観設定）を解放。
*   **ボス戦**: ゲームのメカニクス習熟度が試される「巨大ロボット怪獣」との対決。

## FAQ

**Q: IDUSが唯一の悪役ですか？**

A: IDUSが主要な統治勢力であり敵対者ですが、ゲームにはAIの敵意が物理的に具現化した様々な「暴走ボット」や巨大なロボット怪獣が登場します。

**Q: ダイアナを直接操作できますか？**

A: ダイアナのアクション（特にハッキングやサポート能力）を制御しつつ、同時にヒューの移動や戦闘を管理します。これは伝統的なキャラクターの切り替えではなく、二人のキャラクターを同時に扱うシステムです。

**Q: なぜIDUSは「白血球」と説明されているのですか？**

A: この比喩は、AIが必ずしも「悪」ではないことを説明するために開発者によって使われています。AIは人間の主人公を、現在の自動化された月面ステーションの状態にはそぐわない「感染源」や「異物」と見なしているのです。

**Q: 『PRAGMATA』はNintendo Switchで発売されますか？**

A: PlayStation 5、Xbox Series X/S、Steamでの発売は確定していますが、2026年に「Switch 2（仮）」向けに発売されるという噂もあります。ただし、カプコンからその特定のプラットフォームに関する公式発表はまだありません。
