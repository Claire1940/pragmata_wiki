## シェルター拠点：進行と繋がりのハブ

体験の中心にあるのは、しばしば「クレイドル（ゆりかご）」と呼ばれるシェルターです。ここは単にミッションを選択するための静止したメニュー画面ではありません。主人公のヒューと、彼のアンドロイドの相棒であるダイアナが、過酷な月面から逃れて休息を得るための、完全に具現化された物理的な空間です。シェルターはゲームの進行システムの主要なアンカーとして機能し、ミッション後に得た成果を統合するリセットポイントの役割を果たします。

ヒューがフィールドで倒れた場合、彼はシェルターへと送り返されます。これにより直近のミッションの進行状況は失われますが、拠点で「プリント」された長期的なアップグレードはそのまま維持されます。これにより、失敗したランであっても全体的な強化に貢献するという確信を持って、危険な領域へとさらに踏み込むことを促す、魅力的なリスクとリターンのループが生まれています。

| 機能 | 説明 | ゲーム上の利点 |
| :--- | :--- | :--- |
| **武器プリント** | 設計図を使用して新しい銃器を作成する。 | 高ティアの装備への恒久的なアクセスを解放する。 |
| **収集品の配置** | 世界で見つけたアイテム（地球儀、おもちゃなど）を置く。 | ヒューとダイアナの絆を深める。 |
| **トラムシステム** | 「エスケープハッチ」を使用したファストトラベル網。 | 解放済みのゾーンへの素早いバックトラックを可能にする。 |
| **キャラクターカスタマイズ** | ヒューのスーツやダイアナの外見を変更する。 | 純粋に外見のみだが、一部の改造はステータスに影響する。 |

> **💡 ヒント：** シェルターではダイアナとの交流に時間を割きましょう。子供用の滑り台などの特定のおもちゃやアイテムを持ち帰ると、世界のロアを掘り下げるユニークなアニメーションや会話が発生します。

## 戦術的な戦闘と使い捨て武器システム

本作の戦闘は、射撃、回避、ハッキングを同時にこなす必要のある、めまぐるしいマルチタスクな戦いです。最近の**『プラグマタ』プレビュー**で明らかになった、最も賛否両論ありつつも興味深い機能の一つが、武器の耐久度システムです。ゲームを通して武器庫を持ち歩く従来のシューターとは異なり、ヒューのサブ武器（ショットガンやライフルなど）には厳格な弾数制限があります。

武器の弾倉が空になると、ヒューはそれを投げ捨てます。これにより、常に即興での対応が求められます。強力なショックウェーブガンで遭遇戦を始めたとしても、弾を使い切ればデフォルトのピストルに戻るか、新しい武器をプリントするための「ノード」をスカベンジ（物色）しなければなりません。ピストルは従来の弾薬ではなくクールダウン制で動作するため、完全に無防備になることはありませんが、巨大な脅威を阻止するほどの威力には欠けています。

### 主要な戦闘アビリティ

1.  **マルチハック：** ダイアナは複数の装甲兵を同時にターゲットにし、一撃でその防御を剥ぎ取ることができます。これはリソースを消費するため、「敵の群れを減らす」ために温存すべきです。
2.  **スタシスネット：** 素早く動くロボットを減速させるための戦術ツールで、ヒューが精密射撃を行う隙を作ります。
3.  **オーバードライブ：** 周囲のすべての敵を動けなくする究極能力で、回復やリロードのための決定的な時間を提供します。
4.  **スロー回避：** 完璧なタイミングの回避に報酬を与える高レベルのアップグレードで、他のアクションタイトルの「ウィッチタイム」のように、一時的に時間の流れを遅くします。

## ヒューとダイアナ：感情的な核

シューティング要素は、カルト的な人気を誇る『バイナリードメイン』と比較されるほどインパクトがありますが、本作を真に際立たせているのは二人の主人公の関係性です。ヒューは月面のAI反乱を生き延びようとするエンジニアであり、ダイアナは7歳の少女のような性格と好奇心を持つアンドロイドです。

ゲームはダイアナの純真さを使って、人間性というテーマを探求します。彼女は人類の歴史に関するデータが限られているため、地球文明の残滓を驚きを持って見つめます。これが、「なぜ人間は一緒に食事をするのか」や、子供用の滑り台が「なぜヒューには小さすぎるのか」とダイアナが問いかけるような、切ない瞬間を生み出します。これらの交流は単なる演出ではなく、ゲームプレイにも影響します。ダイアナはハッキングパズル（敵のシールドをバイパスしたりドアを開けたりするためにデジタル迷路を進むミニゲーム）を解くことで、戦闘をサポートしてくれます。

| 交流タイプ | ゲームプレイへの影響 | 物語上の意義 |
| :--- | :--- | :--- |
| **ハイタッチ／承認** | ダイアナの「シンクロ」レベルが上昇する。 | 彼女の成長と認められたいという欲求を示す。 |
| **プレゼント** | シェルターでの新しいインタラクションが解放される。 | 「普通の」子供時代を与えようとするヒューの試み。 |
| **ハッキングサポート** | 敵の装甲やシールドを無効化する。 | チームとしての互いへの依存を強調する。 |
| **環境警告** | ダイアナが隠されたアイテムや罠を見つける。 | 彼女の進化する状況認識能力を示す。 |

## 探索と月面環境

『プラグマタ』の世界は、月面基地の中に作られた「あべこべな」地球です。摩天楼やダイニングルームなど、ニューヨークのような景色が広がるエリアを探索することになりますが、そこには重力を無視したねじれや金属的な増殖物が存在します。探索は、『メタルギア ソリッド 3』の「ケロタン」のような、隠された収集品を見つけることで報われます。

カプコンは、ステージを厳密なオープンワールドではなく、階層的な構造として設計しました。チェックポイントやファストトラベルポイントとして機能する「エスケープハッチ」へと続く分岐路が存在します。これらを見つけることは、セミローグライクな構造の中で進行状況を維持するために不可欠です。

### 敵の種類と挑戦

「敵対AI」として知られる敵は、標準的な二足歩行ロボットから、真に奇怪な機械の怪物まで多岐にわたります。注目すべき遭遇戦の一つに、狂気じみた表情をした巨大な金属の「赤ん坊」が登場するものがあり、このデザインの選択はSF設定に心理的ホラーの層を加えています。

> **警告：** 2026年ビルドの難易度は大幅に調整されています。一撃受けるだけで戦略が崩れる可能性があり、距離を取るために奔走することになります。機動力こそが最大の資産です。

## FAQ

**Q: 『プラグマタ』は完全なローグライクゲームですか？**

A: いいえ、「セミローグライク」な要素を備えています。死亡するとシェルターに戻され、一部のミッション進行状況はリセットされますが、恒久的なアップグレード、武器の設計図、ストーリーの節目などは保持されます。

**Q: プレビュー版でダイアナを操作することはできますか？**

A: プレイヤーは主にヒューを操作しますが、常にダイアナと協力します。彼女はハッキングのミニゲームを担当し、戦闘中にサポートを提供するため、伝統的な意味での「プレイアブル」ではなくとも、ゲームプレイ体験において不可欠な存在です。

**Q: 武器の弾薬システムはどのように機能しますか？**

A: 基本のピストルは無限の弾薬を持ちますが、熱／クールダウン制で動作します。ショットガンやライフルなどの特殊武器は弾数に限りがあり、使い切ると破棄されます。再び使用するには、新しいものを拾うかプリントする必要があります。

**Q: プレビューで言及されている「エスケープハッチ」とは何ですか？**

A: レベルの各所で見つかるチェックポイントです。起動すると、トラムシステムを使用してシェルターへファストトラベルできるようになり、戦闘の合間にインベントリの管理やアップグレードを行いやすくなります。
