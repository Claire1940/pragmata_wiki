## PRAGMATA ストーリーレビュー：プロットと設定の分析

PRAGMATAの物語は、かつてはイノベーションの拠点として栄え、今はゴーストタウンと化した月面研究施設を中心に展開します。この施設の主な目的は、マイクロチップから巨大な構造物まで、複雑な物体をほぼ瞬時に3Dプリントすることを可能にするLunum鉱石由来の素材、**ルナフィラメント**の研究でした。物語は、地球が月とのすべての通信を失い、戦術チームがブラックアウトの原因を調査するために派遣されるところから始まります。

プレイヤーは、壊滅的な月震の後に分隊から孤立してしまった兵士、ヒュー・ウィリアムズを操作します。ヒューはいわゆる典型的なストイックなヒーローではありません。彼は、荒涼とした環境において必要なコントラストを与える、ひねくれた皮肉なユーモアのセンスによって特徴付けられています。彼の生存は、瓦礫の中から彼を救い出した「心」を持つ謎めいたアンドロイドの少女、ダイアナに完全にかかっています。彼らの目標はシンプルながらも困難です。施設のロボット軍団を掌握した暴走AI **IDIS** の魔の手を逃れながら、施設を脱出して地球に帰還することです。

### 主人公コンビ

いかなる**PRAGMATA ストーリーレビュー**においても、核心となるのは二人の主人公の相乗効果です。ヒューとダイアナは単なる戦闘のパートナーではなく、物語的に相互依存しています。

| 特徴 | ヒュー・ウィリアムズ | ダイアナ |
| :--- | :--- | :--- |
| **役割** | 人間の兵士 / 戦闘員 | アンドロイドの相棒 / ハッカー |
| **性格** | 皮肉屋、機転が利く、現実主義 | 好奇心旺盛、謎めいている、共感的 |
| **主なスキル** | 銃器と重火器 | ハッキングと装甲の無力化 |
| **動機** | 地球への帰還 / 生存 | 施設の外にある生命の発見 |

> **💡 ヒント:** 非戦闘時の会話に注目してください。ヒューとダイアナの進化する関係性は、施設の歴史やダイアナの神秘的な出自についてのヒントを与えてくれることがよくあります。

## 戦闘メカニクスと物語の融合

カプコンは、ヒューとダイアナの物語的な絆を反映するようにPRAGMATAの戦闘を設計しました。相棒が単なるバックアップとして機能する従来のアクションゲームとは異なり、本作では協力が必須条件となっています。敵対的なIDISによって制御されている敵ロボットは、ヒューの標準的な射撃ではほとんど歯が立たない高度な装甲を備えています。

成功するためには、プレイヤーはダイアナのハッキング能力を活用しなければなりません。これには、ダイアナが敵のシステムに侵入して防御を無力化するグリッドベースのミニゲームが含まれます。装甲が剥がれれば、ヒューは特定の弱点を狙って大ダメージを与えることができます。この「武装解除して破壊する」ループは、どちらのキャラクターも月面施設で一人では生き残れないという考えを補強しています。

### 戦術的兵装とアップグレード

ヒューの宇宙服は2026年のテクノロジーの結晶であり、交換可能な武器用の4つのスロットを備えています。これらの武器は、標準戦術、防御、重攻撃モジュールに分類されます。しかし、ゲームにはサバイバル要素も導入されており、武器には耐久度と弾薬の制限があります。

| システム | 機能 | 物語の背景 |
| :--- | :--- | :--- |
| **3Dプリンター** | 武器とノードのアップグレード | ルナフィラメント技術を利用 |
| **シェルター** | セーフゾーン / メンテナンス | IDISの監視が届かない隠れ家 |
| **トラム** | ファストトラベルシステム | 施設の各セクターを接続 |
| **キャビン** | サポートAI | 二人を支援する独立したロボット |

## 月面研究施設の探索

PRAGMATAの設定は、ヒューやダイアナと同じくらい重要なキャラクターです。施設は、ハイテク研究所や工業地帯が広がる巨大な迷宮です。探索はヒューのスラスターによって促進され、激しい遭遇戦の際の上昇移動や回避を可能にします。進行に合わせて、チェックポイントと**シェルター**へのアクセスポイントの両方の役割を果たす**脱出ハッチ**をアンロックしていきます。

シェルターは、ゲームのRPG要素が前面に押し出される場所です。ここでは、IDISから独立した友好的なロボット、**キャビン**に出会うことができます。キャビンは、ヒューのスーツやダイアナのハッキングノードをアップグレードするために必要なインフラを提供します。この成長システムは、施設を旅する中でスカベンジ（収集）した材料と直接結びついており、あらゆる「隅々」を調査する価値があります。

## FAQ

**Q: PRAGMATAはシングルプレイヤーですか、それともマルチプレイヤーですか？**

A: PRAGMATAは、二人のメインキャラクター、ヒューとダイアナの協力関係に焦点を当てたシングルプレイヤーのアクションアドベンチャーゲームです。2026年現在、協力型マルチプレイヤーモードの予定は確認されていません。

**Q: PRAGMATAのストーリーにおいて、Lunum（ルナム）鉱石にはどのような重要性がありますか？**

A: Lunumは、このゲームの世界における中心的な資源です。これは3Dプリンティングに革命をもたらしたルナフィラメントの生成を可能にしました。物語は、このテクノロジーがもたらした結末と、なぜ研究施設が音信不通になったのかを軸に展開します。

**Q: PRAGMATAのデモを旧世代のコンソールでプレイできますか？**

A: いいえ、「The Sketchbook」デモおよび本編は、PlayStation 5、Xbox Series X/S、PCを含む次世代プラットフォーム専用です。Nintendo Switch 2向けバージョンも発売されています。

**Q: 戦闘中のハッキングシステムはどのように機能しますか？**

A: ハッキングは、ダイアナによって行われるグリッドベースのミニゲームです。プレイヤーはカーソルをノードに誘導して敵の装甲を無力化し、ヒューがダメージを与えられるようにします。このミニゲームで良い成績を収めると、敵の装甲をより長い時間露出させたままにできます。
