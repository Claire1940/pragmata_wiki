### Optimizing VR Performance
When running the VR mod, many users report a "shimmering" effect or double images on the HUD. To fix these issues, follow these optimization steps:

1.  **Disable Ray Tracing:** In the options menu, turn Ray Tracing to "Off." This significantly reduces shimmering in the VR headset.
2.  **Turn Off Subsurface Scattering:** This setting can cause ghosting on character models in 3D space. Disabling it results in a much cleaner image.
3.  **Manage the HUD:** Because the current VR mod creates a double image for the heads-up display (HUD), many players prefer to turn the HUD off entirely in the game settings or use a specialized Toggle HUD mod.

## Step-by-Step Installation Guide

Most modifications for this title follow a standard installation path. Whether you are adding a new suit or a gameplay script, the process generally involves placing files into the game's root directory.

### Installing Engine-Level Mods (VR, Trainers)
1.  Download the mod file from a reputable source like Nexus Mods.
2.  Locate your game installation folder (usually found in `SteamLibrary\steamapps\common\PRAGMATA`).
3.  Unzip the mod files directly into the folder where the game's executable (.exe) is located.
4.  Launch the game. If using a VR mod, ensure your headset is connected via Virtual Desktop or Link before starting the application.

### Installing Reshade Presets
1.  Download and install the latest version of the ReShade framework.
2.  Select the *Pragmata* executable during the ReShade setup.
3.  Place the specific `.ini` preset file you downloaded (e.g., "Giorma's Realistic Reshade") into the game folder.
4.  Press the `Home` key in-game to select and activate the preset.

> **💡 Tip:** If you encounter a "double cursor" or UI glitches when using visual mods, check if the mod requires a specific version of the game. Most 2026 mods are built for the latest "Sketchbook" build.

## Troubleshooting Common Issues

Modding is rarely a perfect science. Due to the high-fidelity nature of Capcom's RE Engine, even small conflicts can result in crashes or visual artifacts.

### Troubleshooting Table
| Problem | Likely Cause | Solution |
| :--- | :--- | :--- |
| **Game Crashes on Launch** | Incompatible script version. | Remove recent `.dll` files and verify game files on Steam. |
| **HUD Double Image** | VR Mod incompatibility. | Disable the HUD in the main game options menu. |
| **Low Frame Rate** | Reshade/Texture overhead. | Lower "Shadow Quality" and "Volumetric Lighting" in settings. |

## FAQ

**Q: Will this pragmata mod guide work for the console versions?**

A: No. Currently, modding is only possible on the PC version of the game. Consoles like the PS5 and Xbox Series X/S do not support the file-level access required for these modifications.

**Q: Is it safe to use the Ultimate Trainer in online modes?**

A: *Pragmata* is primarily a single-player experience. However, if any future multiplayer components are added, using trainers or cheats could result in an account ban. Always use gameplay-altering mods in offline mode.

**Q: How do I remove a mod if it breaks my game?**

A: The easiest way is to delete the files you added to the game directory. If you are unsure which files were added, use the "Verify Integrity of Game Files" feature in Steam to restore the original game state.

**Q: Can mods improve the game's performance on older hardware?**

A: Yes, certain mods like "No-HUD-No-Effects" or specialized performance configuration files can disable demanding post-processing effects, potentially helping the game run smoother on mid-range PCs from earlier years.
