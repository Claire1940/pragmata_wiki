## Step-by-Step Installation Guide

To begin using **pragmata diana outfit mods**, you first need to locate your game's root folder. For most players, this is handled through Steam. Right-click on Pragmata in your library, select **Manage**, and then **Browse local files**. You should see the `pragmata.exe` file in this directory.

### 1. Installing REFramework
REFramework is the backbone of the modding community. Without it, most custom outfits will simply fail to load.

1. Download the latest `pragmata.zip` from the REFramework nightly builds.
2. Open the zip file and locate the `dinput8.dll` file.
3. Drag and drop `dinput8.dll` directly into your Pragmata root folder (where the .exe is located).
4. Launch the game once to initialize the framework. You should see a menu in the top-left corner.
5. In the REFramework menu, ensure that **Packed Directory Loading** and **Loose File Loader** are both checked on.

### 2. Setting Up Fluffy Mod Manager
Fluffy Mod Manager makes it incredibly simple to toggle different **pragmata diana outfit mods** without manually moving files every time you want a change.

1. Download the Fluffy Mod Manager and extract it to a folder (ideally within your game directory for easy access).
2. Run `ModManager.exe`.
3. From the "Choose Game" list, select **Pragmata**.
4. Define the game's install folder by pasting the directory path you found earlier.
5. You are now ready to drag and drop mod archives (usually .zip or .rar) directly into the Mod Manager window.

## Enhancing Visuals with ReShade and RenoDX

While **pragmata diana outfit mods** change the character's appearance, visual mods like ReShade and RenoDX change how the entire world looks. These are perfect for players who want a more cinematic or "crispy" aesthetic.

### Installing ReShade with Full Add-on Support
1. Download the ReShade installer and select `pragmata.exe`.
2. Choose **DirectX 10/11/12** as the rendering API.
3. When prompted for effects, it is often best to "Uncheck All" and then "Check All" to ensure you have every tool available for custom presets.
4. After installation, download a preset (like the popular `prtt.ini`) and place it in your game folder.
5. Use the **Home** key in-game to open the ReShade menu and select your preset from the dropdown.

### RenoDX for Pragmata
RenoDX is a specialized add-on that works alongside ReShade to fix HDR issues and improve the game's internal lighting engine. To install it, simply drag the `Addon64` file into your root folder. This is a standalone enhancement that significantly improves the depth of field and color accuracy in Pragmata’s 2026 version.

## Managing and Removing Mods

If you ever wish to return to the vanilla experience or want to clean up your mod list, Fluffy Mod Manager makes this process painless. 

*   **To disable a mod**: Simply toggle the switch next to the mod name in the manager.
*   **To delete a mod**: Highlight the mod and click the **Trash Icon**.
*   **To refresh**: If you manually add files to the `Games/Pragmata/Mods` folder, hit the **Refresh** button in the GUI to see them appear.

## FAQ

**Q: Can I use pragmata diana outfit mods on the console versions of the game?**

A: No, modding is currently restricted to the PC version of Pragmata. Console versions (PS5, Xbox Series X/S) do not support the file injection methods required for REFramework or custom meshes.

**Q: Will using mods get me banned from the game?**

A: Pragmata is primarily a single-player experience. While Capcom generally discourages file modification, there is no active anti-cheat for the single-player campaign that would result in a ban for using cosmetic mods. However, always use mods at your own risk and back up your save files.

**Q: Why is my game crashing after I installed a new outfit?**

A: Crashing is usually caused by a conflict between two mods or an outdated version of REFramework. Ensure you have the latest `dinput8.dll` and try disabling your mods one by one in Fluffy Mod Manager to identify the culprit.

**Q: Can I combine hair mods and outfit mods for Diana?**

A: Yes, as long as the mods modify different files. For example, the "Eight Hair" mod only changes the head mesh, while "Colorful Diana" changes the body textures. These can typically be used together without any issues.
