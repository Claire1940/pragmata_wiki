## Outils essentiels pour le modding de Pragmata

Avant de plonger dans le Workshop, vous avez besoin d'une suite d'outils pour gérer vos fichiers. Parce que le jeu utilise une version sophistiquée du RE Engine, le simple glisser-déposer de fichiers fonctionne rarement. Vous aurez besoin d'un gestionnaire de mods capable de "lire" les archives .pak du jeu et d'injecter vos modifications souhaitées en toute sécurité.

### 1. Fluffy Mod Manager (Édition 2026)
Cela reste la référence absolue pour les titres Capcom. Il vous permet d'activer ou de désactiver les mods sans altérer de façon permanente votre répertoire de jeu. C'est essentiel pour le dépannage si un mod spécifique provoque un plantage du jeu lors des transitions lunaires.

### 2. REFramework
Pour les utilisateurs avancés, REFramework fournit une puissante API de scripting. C'est nécessaire pour les mods qui modifient la logique du jeu, comme le carburant de jetpack infini ou des angles de caméra personnalisés. Il sert également de contournement pour certaines restrictions au niveau du moteur qui pourraient empêcher les échanges de modèles de personnages.

### 3. Optiscaler et Frame Generation
Comme discuté dans les récentes analyses de performance, de nombreux joueurs utilisent des outils comme Optiscaler pour injecter le FSR4 ou le XSS2 dans le jeu. C'est particulièrement utile pour les utilisateurs de Steam Deck qui souhaitent maintenir un 60 FPS stable tout en explorant les environnements haute densité de la cité lunaire.

> **💡 Conseil :** Sauvegardez toujours votre fichier `re_chunk_000.pak` avant de tenter des injections manuelles de mods. Bien que les gestionnaires de mods soient sûrs, une archive corrompue peut forcer un nouveau téléchargement de 60 Go.

## Guide d'installation étape par étape

Pour garantir que votre jeu reste stable, suivez ces étapes pour installer des mods via les deux méthodes principales disponibles en 2026.

### Méthode A : Steam Workshop (Officiel)
1. Accédez à la page de **Pragmata** dans votre bibliothèque Steam.
2. Cliquez sur l'onglet **Workshop**.
3. Parcourez les catégories (Succès, Personnages, Cartes, etc.).
4. Cliquez sur **S'abonner** sur le mod de votre choix.
5. Lancez le jeu ; le client Steam gérera automatiquement le placement des fichiers.

### Méthode B : Installation manuelle (Avancée)
L'installation manuelle est souvent requise pour les mods les plus populaires, comme ceux qui transforment votre "compagnon de sac à dos" en diverses icônes de la culture pop ou fournissent les échanges cosmétiques pour "Diana" mentionnés dans les discussions de la communauté.

| Étape | Action | Description |
| :--- | :--- | :--- |
| **1** | **Télécharger le Mod** | Obtenez le fichier .rar ou .7z à partir d'un dépôt de confiance. |
| **2** | **Ouvrir le Gestionnaire** | Lancez Fluffy Mod Manager et sélectionnez Pragmata dans la liste des jeux. |
| **3** | **Déposer les fichiers** | Faites glisser l'archive téléchargée dans la fenêtre du Mod Manager. |
| **4** | **Actualiser la liste** | Cliquez sur "Define Game Path" si c'est votre première fois, puis sur "Refresh Mod List". |
| **5** | **Activer** | Basculez le mod sur "On" et lancez le jeu via le gestionnaire. |

## Trouver les meilleurs mods cosmétiques

Une question fréquente au sein de la communauté concerne l'emplacement de mods de personnages spécifiques pour Diana. Alors que le Steam Workshop est excellent pour les ajustements environnementaux, les mods "cosmétiques" spécifiques aux personnages se trouvent souvent sur des sites "thématiques" externes.

Les membres de la communauté ont noté que certains échanges de modèles de haute qualité — similaires aux célèbres mods de Resident Evil 2 — sont hébergés sur des sites comme NexusMods ou des forums communautaires privés. Ces mods peuvent aller de simples recolorations de tenues à des remplacements complets de modèles, comme les échanges humoristiques "Danny DeVito en compagnon de sac à dos" ou "Homer Simpson" qui sont récemment devenus viraux sur les réseaux sociaux.

## L'éthique du modding en 2026

À mesure que le modding se professionnalise, le débat entre "mods payants" et "contenu communautaire gratuit" se poursuit. En 2026, de nombreux créateurs proposent un accès anticipé à leur travail via des plateformes comme Patreon, tandis que d'autres estiment que tous les mods devraient rester gratuits en tant que "fan art".

Le consensus parmi la communauté des moddeurs vétérans est que si les commissions sont acceptables pour un travail personnalisé, la sortie publique des mods devrait idéalement être gratuite pour assurer la longévité de la base de joueurs du jeu. Le modding est essentiel pour maintenir les titres en vie longtemps après que les développeurs sont passés à de nouveaux projets.
