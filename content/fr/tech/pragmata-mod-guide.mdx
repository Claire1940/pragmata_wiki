## Bien démarrer : Prérequis pour le modding

Avant de commencer à modifier vos fichiers de jeu, il est vital de comprendre qu'en 2026, le modding est principalement une entreprise centrée sur le PC. Alors que les joueurs sur console sont souvent limités aux mises à jour officielles, les utilisateurs de PC ont la liberté d'accéder aux ressources internes du jeu. Pour garantir une expérience fluide, vous devez préparer votre système avec quelques outils de base et des mesures de sécurité.

> **⚠️ Avertissement :** Des mods excessifs ou mal installés peuvent déstabiliser votre client de jeu et potentiellement corrompre vos fichiers de sauvegarde. Testez toujours les nouveaux mods de manière progressive et conservez une sauvegarde propre de votre dossier "Vanilla Saves".

### Liste de contrôle essentielle pour le modding
| Exigence | Description | Objectif |
| :--- | :--- | :--- |
| **Plateforme PC** | Un PC sous Windows avec accès administratif. | Requis pour la modification de fichiers et l'injection de scripts. |

### Optimiser les performances VR
Lors de l'utilisation du mod VR, de nombreux utilisateurs signalent un effet de "scintillement" ou des images doubles sur l'HUD. Pour corriger ces problèmes, suivez ces étapes d'optimisation :

1.  **Désactiver le Ray Tracing :** Dans le menu des options, réglez le Ray Tracing sur "Off". Cela réduit considérablement le scintillement dans le casque VR.
2.  **Désactiver la dispersion sous-surfacique (Subsurface Scattering) :** Ce réglage peut provoquer des images fantômes sur les modèles de personnages dans l'espace 3D. Le désactiver permet d'obtenir une image beaucoup plus nette.
3.  **Gérer l'HUD :** Comme le mod VR actuel crée une image double pour l'affichage tête haute (HUD), de nombreux joueurs préfèrent désactiver complètement l'HUD dans les paramètres du jeu ou utiliser un mod spécialisé "Toggle HUD".

## Guide d'installation étape par étape

La plupart des modifications pour ce titre suivent un chemin d'installation standard. Que vous ajoutiez une nouvelle combinaison ou un script de gameplay, le processus consiste généralement à placer des fichiers dans le répertoire racine du jeu.

### Installer des mods au niveau du moteur (VR, Trainers)
1.  Téléchargez le fichier du mod depuis une source fiable comme Nexus Mods.
2.  Localisez votre dossier d'installation du jeu (généralement situé dans `SteamLibrary\steamapps\common\PRAGMATA`).
3.  Décompressez les fichiers du mod directement dans le dossier où se trouve l'exécutable du jeu (.exe).
4.  Lancez le jeu. Si vous utilisez un mod VR, assurez-vous que votre casque est connecté via Virtual Desktop ou Link avant de démarrer l'application.

### Installer des préréglages ReShade
1.  Téléchargez et installez la dernière version du framework ReShade.
2.  Sélectionnez l'exécutable *Pragmata* lors de la configuration de ReShade.
3.  Placez le fichier de préréglage `.ini` spécifique que vous avez téléchargé (par exemple, "Giorma's Realistic Reshade") dans le dossier du jeu.
4.  Appuyez sur la touche `Origine` (Home) en jeu pour sélectionner et activer le préréglage.

> **💡 Astuce :** Si vous rencontrez un "double curseur" ou des bugs d'interface utilisateur lors de l'utilisation de mods visuels, vérifiez si le mod nécessite une version spécifique du jeu. La plupart des mods de 2026 sont conçus pour la dernière version "Sketchbook".

## Dépannage des problèmes courants

Le modding est rarement une science exacte. En raison de la nature haute fidélité du RE Engine de Capcom, même de petits conflits peuvent entraîner des plantages ou des artefacts visuels.

### Tableau de dépannage
| Problème | Cause probable | Solution |
| :--- | :--- | :--- |
| **Le jeu plante au lancement** | Version du script incompatible. | Supprimez les fichiers `.dll` récents et vérifiez l'intégrité des fichiers du jeu sur Steam. |
| **Double image de l'HUD** | Incompatibilité du mod VR. | Désactivez l'HUD dans le menu des options principales du jeu. |
| **Faible taux de rafraîchissement** | Surcharge due au Reshade/Textures. | Baissez la "Qualité des ombres" et l' "Éclairage volumétrique" dans les paramètres. |
| **Personnages en pose T (T-Posing)** | Erreurs de script en VR. | Redémarrez le point de passage ; c'est un bug connu des premières injections VR. |

## FAQ

**Q : Ce guide des mods Pragmata fonctionnera-t-il pour les versions consoles ?**

R : Non. Actuellement, le modding n'est possible que sur la version PC du jeu. Les consoles comme la PS5 et la Xbox Series X/S ne permettent pas l'accès aux fichiers requis pour ces modifications.

**Q : Est-il sûr d'utiliser l'Ultimate Trainer dans les modes en ligne ?**

R : *Pragmata* est principalement une expérience solo. Cependant, si des composants multijoueurs sont ajoutés à l'avenir, l'utilisation de trainers ou de cheats pourrait entraîner un bannissement de compte. Utilisez toujours des mods modifiant le gameplay en mode hors ligne.

**Q : Comment supprimer un mod s'il casse mon jeu ?**

R : Le moyen le plus simple est de supprimer les fichiers que vous avez ajoutés au répertoire du jeu. Si vous ne savez plus quels fichiers ont été ajoutés, utilisez la fonction "Vérifier l'intégrité des fichiers du jeu" dans Steam pour restaurer l'état original du jeu.

**Q : Les mods peuvent-ils améliorer les performances du jeu sur du matériel ancien ?**

R : Oui, certains mods comme "No-HUD-No-Effects" ou des fichiers de configuration de performance spécialisés peuvent désactiver des effets de post-traitement gourmands, aidant potentiellement le jeu à fonctionner plus fluidement sur des PC de milieu de gamme des années précédentes.
