## Comment changer les paramètres de langue

Si votre jeu n'a pas démarré automatiquement avec la **pragmata deutsche synchro** activée, ou si vous souhaitez expérimenter différentes pistes audio, suivez ces étapes pour ajuster vos paramètres :

1.  Accédez au **Menu Principal** ou ouvrez le **Menu Pause** pendant le jeu.
2.  Sélectionnez l'onglet **Options** (Optionen).
3.  Allez dans les **Paramètres de langue** (Spracheinstellungen).
4.  Sous **Langue des voix** (Sprachausgabe), faites défiler jusqu'à trouver **Allemand** (Deutsch).
5.  Assurez-vous que les **Sous-titres** (Untertitel) sont activés si vous souhaitez un support textuel.
6.  Sauvegardez vos modifications et revenez au jeu.

> **⚠️ Attention :** Changer la langue des voix pendant une mission active peut nécessiter que le jeu recharge le dernier point de contrôle pour synchroniser correctement les nouveaux fichiers audio.
