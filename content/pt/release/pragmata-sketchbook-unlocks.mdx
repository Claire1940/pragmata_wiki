## O Encontro Secreto no Elevador

O cerne do mistério atual gira em torno da sequência do elevador, onde um trio de sentinelas robóticas tenta interromper seu progresso. Embora a maioria dos jogadores limpe esta sala usando sua arma mais poderosa, fazer isso na verdade bloqueia o acesso ao conteúdo secreto. Para acionar o chefe oculto — que é um pré-requisito para um dos mais importantes **desbloqueios do sketchbook de Pragmata** — você deve demonstrar maestria sobre todo o seu arsenal.

O jogo rastreia o "golpe de misericórdia" desferido em cada um dos três robôs iniciais. Se você usar a mesma arma para matar os três, a sequência termina normalmente. No entanto, se você diversificar seus golpes fatais, o jogo detecta sua proficiência tática e faz surgir um inimigo oculto de alto nível assim que o elevador chega ao seu destino.

### Requisitos de Finalização de Arma

Para acionar com sucesso o surgimento do inimigo secreto, você deve aderir à "Regra de Três". Cada robô deve ser destruído por uma fonte de dano diferente. É importante notar que você pode enfraquecer os inimigos com qualquer arma que escolher; o jogo verifica apenas o golpe final que reduz a saúde deles a zero.

| Alvo Inimigo | Arma de Finalização Recomendada | Notas |
| :--- | :--- | :--- |
| **Robô A (Primeiro)** | **Pistola / Tiro Padrão** | Mais fácil de controlar para a primeira morte. |
| **Robô B (Segundo)** | **Arma de Onda de Choque / Artilharia Pesada** | Certifique-se de que o dano colateral não atinja acidentalmente o Robô C. |
| **Robô C (Terceiro)** | **Hack / Finalização Multi-Hack** | Usar a mecânica de hacking como finalizador é altamente confiável. |

> **⚠️ Aviso:** Seja extremamente cuidadoso ao usar armas de área de efeito (AoE). Se uma única explosão ou onda de choque matar dois robôs simultaneamente, contará como a mesma fonte de arma para ambos, e o desbloqueio secreto falhará.

## Guia Passo a Passo para Desbloquear o Segundo Desenho de Diana

Desbloquear o chefe oculto está diretamente ligado ao segundo desenho de Diana. No mundo de Pragmata, os esboços de Diana representam a percepção dela sobre as ameaças que vocês enfrentam juntos. Sem encontrar o inimigo secreto do elevador, ela nunca ganha a "inspiração" para desenhá-lo, deixando uma lacuna permanente em seu Sketchbook.

Siga estes passos durante a sequência do elevador para garantir o sucesso:

1.  **Isole os Alvos:** Assim que os três robôs caírem no elevador, mova-se para um canto para atrair a atenção deles.
2.  **Enfraquecimento:** Use seu rifle primário para reduzir a saúde de todos os três robôs para aproximadamente 10%. Não se comprometa demais com seus combos.
3.  **A Primeira Morte:** Mude para sua arma secundária (Pistola) e dê um único tiro preciso no primeiro robô.
4.  **A Segunda Morte:** Mude para a Arma de Onda de Choque. Espere o segundo robô se afastar do terceiro e dispare uma rajada para finalizá-lo.
5.  **A Morte Final:** Aproxime-se do último robô e inicie um Hack. Concluir a sequência de hack deve destruir o robô, satisfazendo o terceiro requisito de arma única.
6.  **A Revelação:** Quando o elevador parar, em vez da saída habitual, um aviso de "Intrusos Detectados" piscará em vermelho, e o chefe oculto aparecerá na arena.

| Fonte de Dano | Categoria de Desbloqueio | Eficácia para Finalizações |

| **Pistola** | Cinética | **Alta** - Muito fácil de controlar o dano de tiro único. |
| **Arma de Onda de Choque** | Energia/Atordoamento | **Média** - Risco de dano colateral a robôs próximos. |

O uso do Multi-Hack é um ponto de discórdia entre os jogadores. Embora o dano de um Multi-Hack não necessariamente "quebre" o requisito, se o último tique de dano de uma única ativação de Multi-Hack matar dois robôs diferentes, o jogo os registra como tendo a mesma fonte de morte. Para os **desbloqueios do sketchbook de Pragmata** mais consistentes, atenha-se a finalizadores de alvo único.

## Estratégias de Combate para Perfeccionistas

Ao tentar os **desbloqueios do sketchbook de Pragmata**, seu maior inimigo é frequentemente sua própria produção de dano. À medida que você aprimora seu equipamento na demo, torna-se cada vez mais fácil "matar com um golpe" acidentalmente os robôs no elevador, dificultando a troca de armas a tempo.

### Gerenciando Limiares de Dano

Para evitar mortes acidentais, considere desequipar modificadores de alto dano antes da luta no elevador. Você quer que os robôs sejam resistentes o suficiente para que você possa reduzir cuidadosamente a saúde deles. Se você estiver tendo dificuldades com o tempo, tente o seguinte:

*   **Toques de Pistola:** Use a pistola para a maior parte da luta. Ela tem o menor DPS (dano por segundo), mas a maior precisão.
*   **Sinais Visuais:** Observe os robôs começarem a soltar faíscas ou fumaça. Este é o sinal universal em Pragmata de que um inimigo está com menos de 15% de saúde.
*   **O Indicador de Hack:** Quando um robô estiver baixo o suficiente, um aviso para um "Hack Finalizador" aparecerá. Este é o seu sinal de que o robô está pronto para ser colhido para o desbloqueio.

> **💡 Dica:** Se você errar a rotação e matar dois robôs com a mesma arma, você pode reiniciar rapidamente a partir do último checkpoint. O checkpoint é acionado logo antes de você entrar no elevador, permitindo tentativas rápidas.

## FAQ

**P: O "Multi-Hack" conta como uma arma ou várias para os desbloqueios do sketchbook de Pragmata?**

R: O Multi-Hack é considerado uma única fonte de dano. Se você usar um Multi-Hack para matar dois robôs ao mesmo tempo, falhará no requisito para a finalização com arma única. No entanto, se você usá-lo para matar apenas um robô, ele conta como uma finalização única de "Hack".

**P: Posso usar a Arma de Onda de Choque para causar dano aos robôs, desde que não os mate com ela?**

R: Sim. O jogo rastreia apenas o "Golpe de Misericórdia". Você pode usar qualquer combinação de armas para diminuir a saúde de um inimigo. Desde que o golpe final que destrói o robô venha de uma arma diferente das outras duas mortes, o segredo será desbloqueado.

**P: O que acontece se eu desbloquear o chefe oculto, mas morrer durante a luta?**

R: Se você morrer após o chefe oculto ter surgido, geralmente terá que refazer a sequência do elevador para acionar o surgimento novamente. A entrada do Sketchbook para o chefe só é salva quando o inimigo é derrotado e Diana completa sua animação de desenho.

**P: Existe uma ordem específica em que as armas devem ser usadas?**

R: Não, a ordem não importa. Você pode finalizar com o Hack primeiro, a Pistola em segundo e a Arma de Onda de Choque em terceiro, ou qualquer outra variação. A única regra estrita é que todos os três golpes finais devem originar-se de tipos de armas diferentes.
