## Primeiros Passos: Pré-requisitos para Modding

Antes de começar a alterar os arquivos do seu jogo, é vital entender que o modding em 2026 é um empreendimento focado principalmente no PC. Enquanto os jogadores de console muitas vezes ficam limitados às atualizações oficiais, os usuários de PC têm a liberdade de acessar os ativos internos do jogo. ### Otimizando a Performance em VR
Ao executar o mod de VR, muitos usuários relatam um efeito de "cintilação" ou imagens duplicadas no HUD. Para corrigir esses problemas, siga estes passos de otimização:

1.  **Desative o Ray Tracing:** No menu de opções, mude o Ray Tracing para "Off" (Desligado). Isso reduz significativamente a cintilação no headset VR.
2.  **Desligue o Subsurface Scattering:** Essa configuração pode causar ghosting nos modelos de personagens no espaço 3D. Desativá-la resulta em uma imagem muito mais limpa.
3.  **Gerencie o HUD:** Como o mod de VR atual cria uma imagem dupla para a interface (HUD), muitos jogadores preferem desligar o HUD inteiramente nas configurações do jogo ou usar um mod especializado de Toggle HUD.

## Guia de Instalação Passo a Passo

A maioria das modificações para este título segue um caminho de instalação padrão. Quer você esteja adicionando um novo traje ou um script de gameplay, o processo geralmente envolve colocar arquivos no diretório raiz do jogo.

### Instalando Mods de Nível de Motor (VR, Trainers)
1.  Baixe o arquivo do mod de uma fonte confiável como o Nexus Mods.
2.  Localize a pasta de instalação do jogo (geralmente encontrada em `SteamLibrary\steamapps\common\PRAGMATA`).
3.  Descompacte os arquivos do mod diretamente na pasta onde o executável do jogo (.exe) está localizado.
4.  Inicie o jogo. Se estiver usando um mod de VR, certifique-se de que seu headset esteja conectado via Virtual Desktop ou Link antes de iniciar o aplicativo.

### Instalando Presets de Reshade
1.  Baixe e instale a versão mais recente do framework ReShade.
2.  Selecione o executável do *Pragmata* durante a configuração do ReShade.
3.  Coloque o arquivo de preset `.ini` específico que você baixou (ex: "Giorma's Realistic Reshade") na pasta do jogo.
4.  Pressione a tecla `Home` dentro do jogo para selecionar e ativar o preset.

> **💡 Dica:** Se você encontrar um "cursor duplo" ou falhas na interface ao usar mods visuais, verifique se o mod requer uma versão específica do jogo. A maioria dos mods de 2026 foi construída para a build mais recente do "Sketchbook".

## Solução de Problemas Comuns

O modding raramente é uma ciência perfeita. Devido à natureza de alta fidelidade do RE Engine da Capcom, até mesmo pequenos conflitos podem resultar em crashes ou artefatos visuais.

### Tabela de Solução de Problemas
| Problema | Causa Provável | Solução |
| :--- | :--- | :--- |
| **Jogo fecha ao iniciar** | Versão de script incompatível. | Remova os arquivos `.dll` recentes e verifique a integridade dos arquivos no Steam. |
| **Imagem Dupla no HUD** | Incompatibilidade com o Mod de VR. | Desative o HUD no menu de opções principal do jogo. |
| **Baixa Taxa de Quadros** | Sobrecarga de Reshade/Textura. | Diminua a "Qualidade das Sombras" e a "Iluminação Volumétrica" nas configurações. |
| **Personagens em Pose T** | Erros de script no VR. | Reinicie o checkpoint; este é um comportamento conhecido das primeiras injeções de VR. |

## FAQ

**P: Este guia de mods de pragmata funcionará para as versões de console?**

R: Não. Atualmente, o modding só é possível na versão para PC do jogo. Consoles como o PS5 e Xbox Series X/S não suportam o acesso ao nível de arquivo necessário para essas modificações.

**P: É seguro usar o Ultimate Trainer em modos online?**

R: *Pragmata* é primariamente uma experiência single-player. No entanto, se quaisquer componentes multiplayer futuros forem adicionados, o uso de trainers ou cheats pode resultar em um banimento da conta. Sempre use mods que alteram o gameplay no modo offline.

**P: Como removo um mod se ele quebrar meu jogo?**

R: A maneira mais fácil é deletar os arquivos que você adicionou ao diretório do jogo. Se não tiver certeza de quais arquivos foram adicionados, use o recurso "Verificar Integridade dos Arquivos do Jogo" no Steam para restaurar o estado original do jogo.

**P: Os mods podem melhorar o desempenho do jogo em hardware mais antigo?**

R: Sim, certos mods como "No-HUD-No-Effects" ou arquivos de configuração de desempenho especializados podem desativar efeitos exigentes de pós-processamento, ajudando potencialmente o jogo a rodar de forma mais suave em PCs de gama média de anos anteriores.
