## Como Instalar um プラグマタ mod

A maioria das modificações para *Pragmata* segue o procedimento padrão de modding da RE Engine. Como os arquivos do jogo estão compactados em contêineres `.pak`, você normalmente precisará de um gerenciador de mods para injetar seus arquivos personalizados com segurança.

1.  **Baixe o Fluffy Mod Manager:** Este é o padrão ouro para jogos da Capcom. Certifique-se de ter a versão mais recente de 2026.
2.  **Selecione Pragmata:** Abra o gerenciador e selecione *Pragmata* na lista de jogos. Se não estiver listado, você pode precisar definir o caminho do jogo manualmente.
3.  **Baixe seu Mod:** Visite o [Nexus Mods Pragmata](https://next.nexusmods.com/games/pragmata) para encontrar as criações mais recentes da comunidade.
4.  **Coloque os Arquivos:** Mova os arquivos de mod baixados (geralmente `.zip` ou `.rar`) para a pasta `Games/Pragmata/Mods` dentro do diretório do seu Fluffy Mod Manager.
5.  **Ative e Inicie:** Atualize a lista de mods no gerenciador, mude o **プラグマタ mod** desejado para "On" e inicie o jogo diretamente pelo gerenciador.

> **💡 Dica:** Se um mod exigir o "ShaderToggler", certifique-se de instalar o software base ReShade primeiro, pois muitos mods de alternância de UI dependem dessa estrutura para identificar endereços de shader específicos.

## FAQ

**P: Usar um プラグマタ mod pode me banir?**

R: *Pragmata* é primariamente uma experiência de um jogador. A Capcom geralmente não bane jogadores por usar mods em modos single-player. No entanto, se o jogo apresentar tabelas de classificação online ou elementos cooperativos, é melhor desativar os mods antes de se conectar aos servidores oficiais para evitar sinalizações automáticas.

**P: Por que meu mod não está aparecendo no jogo?**

R: Isso geralmente ocorre devido a uma incompatibilidade de versão. Muitos mods foram criados para a demo *Pragmata Sketchbook* e podem exigir uma atualização para funcionar com o lançamento completo de 2026. Sempre verifique a página de descrição do mod para notas de compatibilidade e garanta que seu Gerenciador de Mods esteja apontando para o diretório de instalação correto.

**P: Posso usar vários Reshades ao mesmo tempo?**

R: Você só pode ter uma predefinição de Reshade ativa por vez através do menu padrão do ReShade. No entanto, você pode combinar manualmente as configurações de diferentes predefinições se estiver familiarizado com a edição de arquivos `.ini`. Para a maioria dos usuários, é melhor escolher um **プラグマタ mod** visual de alta qualidade e mantê-lo.

**P: Onde posso encontrar os melhores mods para o jogo?**

R: A comunidade mais ativa está atualmente hospedada no Nexus Mods. Você pode encontrar de tudo, desde trocas simples de textura até reformulações complexas de jogabilidade. Sempre leia os comentários dos usuários nas páginas dos mods para ver se outros jogadores estão enfrentando bugs com os patches mais recentes do jogo.
